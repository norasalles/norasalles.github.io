Set Up Berkshire Hathaway Travel Protection

Once you obtained a Berkshire Hathaway Travel Protection login and password, take the following steps to set up ClientBase for Live Connect:

Step 1) Create a new ClientBase Live Connect Provider for Berkshire Hathaway Travel Protection by going to Tools|Settings|Live Connect Providers and clicking Add.

Fill in the following fields:

Provider Name: Enter Berkshire Hathaway Travel Protection

URL: Type https://agents.bhtp.com/trams

Travel Category: Select Insurance from the drop-down.

Import XML from: Select HTML Source.

Click Save.

Step 2) Each user can put in his own login information under Tools|My Login|Live Connect.  Click Add and enter the following information:

Live Connect Provider: Select Berkshire HathawayTravel Protection from the drop-down list.

User Name/ Password :  Enter the User Name and Password based upon the login information provided to you when you contacted Berkshire Hathaway Travel Protection.

Account Number /Agency Code : Leave blank.


 Step 3) Create a vendor profile in ClientBase for Berkshire Hathaway Travel Protection if one does not exist. From the Profile Manager, retrieve the vendor profile. Check Insurance as the Travel Category in Additional Information area. From the lick Live Connect Providers area, click Add, then select Berkshire Hathaway Travel Protection from the drop-down list. The vendor code is BHTP. Save.

Step 4)  Make Berkshire Hathaway Travel Protection a Trusted Site.  Do this by going in Internet Explorer:

  • Open Internet Explorer and navigate to https://agents.bhtp.com/trams.
  • Open the Internet Explorer Options dialogue by pressing ALT-T, then choosing Internet Options from the list.
  • Click on Security, then the green checkmark Trusted Sites, then the button labeled Sites.
  • Click Add to include https://agents.bhtp.com/trams in the list of trusted sites, then click Close.
  • Select Low as the Security Level and click OK to save.

Create New Bershire Hathaway Travel Protection Reservation in Live Connect

Step 1) Retrieve the desired client profile and create a new Res Card.  After entering basic Res Card info, launch Live Connect by clicking Live Connect on the menu bar. (You can also launch Live Connect from the Profile action gear.)

Step 2) Complete the Live Connect settings by identifying the vendor, the Live Connect Provider and appropriate Login information. (If you have set up your logins in Tools|MyLogin|Live Connect, login information will auto-populate.)  Click Create New Reservation.

Step 3 ) Select any Profile, Passenger, Branch and Res Card information you want to use to book the reservation and click Connect to launch the on-line booking engine.

Step 4 ) The system prompts with the reservation records included in this Res Card. Click each reservation to insure. Again, this step is designed to save time during the insurance booking process, as less fields need to be completed manually. Any missing or incomplete data can be updated on the website confirming the insurance booking. Click Connect.

Step 5) Although still in ClientBase, you are now on-line at the booking site. Complete the reservations and when the reservation is confirmed, click Continue to TRAMS Import.

Next, click Import Reservation and all reservation data booked on-line is now located in the reservation fields in ClientBase.
