Signature HotelConnection 2.0

This feature is only available to agencies who belong to Signature Travel Network as the Signature HotelConnection 2.0 booking engine is exclusive to this group. Signature members can log into the Signature Intranet, www.signaturetravelnetwork.com, and find setup instructions under Training, in the Hotels and Resorts section. Just scroll down to Reference Guides and “The NEW Hotel Connection 2.0 Advisor Hotel Booking Engine Launch Begins” to access the setup document. For more information on Signature HotelConnection 2.0 or Signature Travel Network, please contact Signature Travel Network directly at techsupport@signaturetravelnetwork.com.

Set Up Signature HotelConnection 2.0

Step 1) Create a new or check that a Live Connect Provider for Signature HotelConnection 2.0 exists by going to Utilities|Live Connect Providers, and clicking Add. (Make sure the URL is correct if Provider is present.)

Provider Name: Enter Signature HotelConnection 2.0.

URL: Enter https://www.signaturetravelnetwork.com/SigNet/index.cfm/LiveConnect/hotels

Travel Category: Select Hotel from the drop-down.

Import XML from: Select HTML Source.

Step 2) Each agent can set up a login for HotelConnection 2.0 by going to Utilities|My Login and clicking on the Live Connect tab. Click Add and enter the following information:

Live Connect Provider: Select Signature HotelConnection 2.0 from the drop-down list.

User Name, Password and Account Number: Enter your Signature Intranet User Name and Password. The account number is the agency’s key. Check with your owner or manager for the Signature Agency Key.

Step 3) When importing, Signature HotelConnection 2.0 will be passing over the Hotel Property name as the Vendor Name. You will be able to search for, and create if necessary, the Vendor Profile upon import.

Signature HotelConnection 2.0 Features

Signature HotelConnection 2.0 support the following Live Connect Features if checked:

Create New Signature HotelConnection 2.0 Reservation

Step 1) Use the ClientBase Res Card to launch Signature HotelConnection 2.0.

Once the setup steps have been completed, retrieve the desired client profile and start or retrieve a Res Card and launch Live Connect by clicking Live Connect on the menu bar. (You can also launch Live Connect from the Profile action gear.)

Step 2) Verify the default Login information and select "Create New Reservation".

Leave the Vendor box blank, and select Signature HotelConnection 2.0 from the Connect To drop down. Confirm that the stored login information appears and is correct. If not, you can follow the setup instructions to store a login or simply enter it now. Use the Create New Reservations button to book a new reservation.

Step 3) Select the desired client information to pass to HotelConnection 2.0. Prior to connecting you to the Signature HotelConnection 2.0 site, a client selection screen appears, allowing you to select the desired client data you would like to automatically pass to Signature HotelConnection 2.0. Passing this data helps you save time during the booking process, as fewer fields need to be completed by hand.

Step 4) Use Hotel Finder to find the "perfect hotel." Use HotelConnection 2.0 to finish confirming your Reservation and upon receiving a confirmation number click Import Reservation to import the details into your ClientBase database for managing further. When importing, the system will have you search for the Hotel Vendor profile. If one does not exist, click Create Vendor to add a new Vendor profile for the property.

 

 

This reservation information is now available for you to invoice, to create Reminders to track the administrative tasks involved in managing the trip and to report on for future marketing purposes.

Import Existing Reservation Made Directly in SigNet

Begin a new Live Connect following the same steps as above for Creating a New Reservation in steps 1 and 2, except click Import an Existing Reservation. Do not enter a confirmation number, but when in the Signature HotelConnection 2.0 booking site, locate the retrieve reservation area. Once the reservation details are showing, click Import Reservationand the details of the reservation are imported into theClientBase reservation record.

Retrieve an Existing Reservation for Editing or Cancelling

To edit or cancel an existing reservation using Live Connect, follow these steps:

Step 1) Launch Live Connect from inside the existing reservation to edit or cancel in ClientBase.

Step 2) A dialog box appears. Click Retrieve Reservation, then Connect.

Step 3) Click on Search and then click on the GDS Look up Number box in the lower right hand corner of the results. Once the booking is displayed, click on Import Reservation.
