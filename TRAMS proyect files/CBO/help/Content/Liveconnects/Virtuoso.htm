Virtuoso

If you are not currently registered as a Virtuoso.com advisor, go to www.virtuoso.com to do so.  Upon obtaining a Virtuoso.com advisor login, take the following steps to set up Virtuoso as a ClientBase Live Connect Provider.

Set Up Virtuoso

Step 1) Once you have obtained accurate login information, you need to add this Live Connect Provider if it does not already exist into ClientBase by going to Tools|Settings|Live Connect Providers and clicking Add.

Fill in the following fields:

Name of the Provider: In blank field provided, type Virtuoso.

Status: Leave the default of Active. If you ever want to make it Inactive, go back in this area to turn off.

URL: Enter http://www.virtuoso.com/LiveConnect?product=liveconnect

Travel Category: Select Cruise.

Import XML from: Select HTML Source.

Include Authentication: Leave blank.

Click Save.

Step 2) You do not have to set up agent logins in ClientBase for use with Virtuoso, but you will have to use your Virtuoso.com advisor login when you launch the site from ClientBase.  You do need; however, to create a vendor profile in ClientBase if one does not exist. From the ClientBase Profile Manager, retrieve the Virtuoso vendor profile (or create if not in database). From the General Info area, click Live Connect Providers. From here click Add, then select Virtuoso from the drop-down list. The vendor code is blank. Save.  Make sure the Travel Category of Cruise is checked off in the Virtuoso vendor profile.

Note:  This Virtuoso Live Connect only works with HOTEL, CRUISE AND TOUR product quotes. When the product comes into ClientBase, you will be changing the Vendor name from Virtuoso, AND the Travel Category to the correct one from the drop-down.

Virtuoso Live Connect Features

Create New Virtuoso Reservation in Live Connect

Step 1) Retrieve the desired client profile, create a new Res Card and after entering basic Res Card info, launch Live Connect by clicking Live Connect on the menu bar. (You can also launch Live Connect from the Profile action gear.)

Step 2) Select Virtuoso as the Vendor and Virtuoso from the drop-down menu by Connect To. (You do not need to enter any login information as this is entered at the Virtuoso Web site where you will use your Virtuoso.com advisor login.

Step 3) Click the Create New Reservation button.

Step 4) You do not have to select any Profile, Traveler, Branch and Res Card information as this data is NOT passed to Virtuoso.  Click Connect to launch the on-line booking engine.

Step 5) Although you are still in ClientBase, you are now on-line at the Virtuoso site and can browse the site for the product quote to import into ClientBase.  When you bring up a specific product, click the Import Reservation button to transfer the information to the Res Card. It will land in the Reservation Service Provider|Itinerary area.

You may change the vendor information to reflect the actual vendor you are using.  This will use the actual supplier information as well as the product information you selected to import from virutoso.com.   You must also change the Travel Category to either HOTEL, TOUR OR CRUISE.  This Live Connect only brings these types of products in for an import.

Note:  Cruise Itineraries may have blank times if the itinerary shows as 12 a.m. for all days.  Greater detail imports for Virtuoso Voyager Club.
