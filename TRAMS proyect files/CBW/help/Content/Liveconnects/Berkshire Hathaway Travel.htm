Set Up Berkshire Hathaway Travel Protection

Once you obtained a Berkshire Hathaway Travel Protection login and password, take the following steps to setup ClientBase for Live Connect:

Step 1) Create a new (if none exists) ClientBase Live Connect Provider for Berkshire Hathaway Travel Protection by going to Utilities|Live Connect Providers, and clicking Add. (Make sure the URL is correct if Provider is present.)


Provider Name: Enter Berkshire Hathaway (the full name does not fit in field)

URL: Type https://agents.bhtp.com/trams

Travel Category: Select Insurance from the drop-down.

Import XML from: Select HTML Source.

Step 2) Setup for Live Connect agent logins can be done individually. Agents can set up their own logins by going to Utilities|My Login|Live Connect and clicking Add.


Live Connect Provider: Select Berkshire Hathaway from the drop-down list.

User Name/Password:  Enter the User Name and Password based upon the login information provided to you when you contacted Berkshire Hathaway Travel Protection.

Account Number/Agency Code: Leave blank.


Step 3) Create/Edit a vendor profile in ClientBase for Berkshire Hathaway Travel Protection. From the Profile Manager, retrieve the Berkshire Hathaway Travel Protection vendor profile (or create one). From the General Info Tab, click Live Connect Providers. From here, click Add, then select the desired booking engine (Berkshire Hathaway) from the drop-down list. The vendor code is BHTP. Select the Travel Type, Insurance.  Click OK to save.

Step 4)  Make Berkshire Hathaway Travel Protection a Trusted Site.  Do this by going in Internet Explorer:

  • Open Internet Explorer and navigate to https://agents.bhtp.com/trams.
  • Open the Internet Explorer Options dialogue by pressing ALT-T, then choosing Internet Options from the list.
  • Click on Security, then the green checkmark Trusted Sites, then the button labeled Sites.
  • Click Add to include https://agents.bhtp.com/trams in the list of trusted sites, then click Close.
  • Select Low as the Security Level and click OK to save.

 

Berkshire Hathaway Travel Protection Features

Berkshire Hathaway Travel Protection supports the following Live Connect Features if checked:

Create New Berkshire Hathaway Travel Protection  Reservation

Step 1) Enter the Res Card information needed and click the Reservation Tab and Add in the Res Card. Enter Vendor and Travel Category and then click Live Connect if not automatically launched.

Step 2) Select Berkshire Hathaway Travel Protection from the drop-down menu by Connect To (if not defaulted), and then click Create New Reservations.

Step 3) Select any customer information for use to book the Berkshire Hathaway Travel Protection reservation. Click Next to continue.

This data is automatically passed to Berkshire Hathaway Travel Protection and saves time during the booking process, as less fields need to be completed manually. Click Connect.

Step 4) The system prompts with the reservation records included in this Res Card. Highlight each reservation you want to insure. To highlight and select multiple reservations to insure, use the CTRL key in combination with clicking, and the system completes the summary trip information. Again, this step is designed to save time during the insurance booking process. Any missing or incomplete data can be updated on the Bershire Hathaway Travel Protection website confirming the insurance booking. Click Connect.

Step 5) Although still in ClientBase, you are now on-line at the booking site. Complete the reservations and when the reservation is confirmed, click Continue to TRAMS Import.

Next, click Import Reservation and all reservation data booked on-line is now located in the reservation fields in ClientBase.
