Virtuoso

If you are not currently registered as a Virtuoso.com advisor, go to www.virtuoso.com to do so.  Upon obtaining a Virtuoso.com advisor login, take the following steps to set up Virtuoso as a ClientBase Live Connect Provider.

Set Up Virtuoso

Step 1) Once you have obtained accurate login information, you need to add this Live Connect Provider if it does not already exist into ClientBase by going to Utilities|Live Connect Providers and clicking Add.

Fill in the following fields:

Provider Name: Enter Virtuoso.

Status: Leave the default of Active. If you ever want to make it Inactive, go back in this area to turn off.

URL: Enter http://www.virtuoso.com/LiveConnect?product=liveconnect

Travel Category: Select Cruise.

Import XML from: Select HTML Source.

Include Authentication: Leave blank.

Click Save.

Step 2) You do not have to set up agent logins in ClientBase for use with Virtuoso, because you will use your Virtuoso.com advisor login when you launch the site from ClientBase.  You do need; however, to create a vendor profile in ClientBase if one does not exist.

From the ClientBase Profile Manager, retrieve the Virtuoso vendor profile (or create if not in database). From the General Info area, click Live Connect Providers. Click Add, then select Virtuoso from the drop-down list. The vendor code is blank. Save.  Make sure the Travel Category of Cruise is checked off in the Virtuoso vendor profile.

Note:  This Virtuoso Live Connect only works with HOTEL, CRUISE AND TOUR quotes. When the product quote comes into ClientBase, you will be changing the Vendor name from Virtuoso, AND the Travel Category to the correct one from the drop-down.

Create New Virtuoso Reservation in Live Connect

Step 1) Create a Res Card in ClientBase, enter the Res Card Cycle, Trip Name, and Region/Destination and add a reservation (best practice). Select Virtuoso as the vendor, then click the Live Connect icon.

Step 2) Virtuoso should show as the Connect To vendor. (You do not need to enter any login information as this is entered at the Virtuoso Web site where you use your Virtuoso.com advisor login.

Step 3) Click the Create New Reservation button.

Step 4) You do not have to select any Profile, Traveler, Branch and Res Card information as this data is NOT passed to Virtuoso.  Click Connect to launch the on-line booking engine.

Step 5) Although you are still in ClientBase, you are now on-line at the Virtuoso site and can browse the site for the product quote to import into ClientBase.  When you bring up a specific product, click the Import Reservation button to transfer the information to the Res Card. It will land in the Reservation Service Provider|Itinerary area.

Once data is imported into the ClientBase Res Card, you may add the rest of the product information (confirmation numbers, dates, rates, etc.) by going into the Reservation area and editing the imported data.

You may change the vendor information to reflect the actual vendor you are using.  This will use the actual supplier information as well as the product information you selected to import from virutoso.com.   You must also change the Travel Category to either HOTEL, TOUR OR CRUISE.  This Live Connect only brings these types of products in for an import. 

Note:  Cruise Itineraries may have blank times if the itinerary shows as 12 a.m. for all days.  Greater detail imports for Virtuoso Voyager Club.
