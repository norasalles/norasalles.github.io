Set Up Ensemble Navigate

Upon receiving login information for each agent in your office, take the following steps to setup ClientBase for Live Connect.

Step 1) Create a Live Connect Provider for Ensemble Navigate if none exists by going to Utilities|Live Connect Providers, and clicking Add.

Provider Name: Enter Ensemble Navigate

URL: Enter https://navigate.ensembletravel.com/app/0/poweragent_sso_v1/0/clientbase_sso_login.html?clear=all

Travel Category: Enter Cruise from the drop-down menu

Import XML from: Enter HTML Source

Step 2) Set up for Live Connect agent logins can be done individually. When Live Connect is launched, the system first looks to see if there is login information at the User Login level, and if not, then asks the user to login at the provider website. Users can set up their own logins by going to Utilities|My Login|Live Connect and clicking Add.

Live Connect Provider: Select Ensemble Navigate from the drop-down list.

User Name, Password: Enter your User Name and Password. Account Number and Agency Code are left blank.

Step 3) From the Profile Manager, retrieve the desired Vendor from the list below or create a new Vendor profile if one does not already exist. From the General Info tab, click Live Connect Providers. Click Add, then select Ensemble Navigate from the drop-down list. Enter the corresponding Vendor Code from the table below.

Live Connect Features Supported by Ensemble Navigate

Ensemble Navigate supports the following Live Connect features if checked:

Creating a New Reservation

Step 1) When ready to book on-line, create a client Res Card and fill in the Agent, Status, Reservation Cycle, Trip Name, Region/Destination fields and select the travelers. Use the Add Reservation button to start a new reservation. Fill in the Vendor and Travel Category fields and then click Live Connect.

Step 2) Select Ensemble Navigate from the drop-down menu by Connect To (if not defaulted), and then click Create New Reservation.

Step 3) Select any customer information to use to book the reservation. This data is automatically passed to Ensemble Navigate and saves time during the booking process, as less fields need to be completed manually. Click Connect.

Step 4) Although still in ClientBase, you are now on-line at the booking site. Complete the booking and when done, scroll to the bottom and click on View Your Order.

 

Step 5) You’ll be taken to a summary of the booking and can then click Import Reservation.

All reservation data booked on-line is now located in the reservation fields in ClientBase.

When connecting to Ensemble Navigate multiple times in a session of ClientBase, the site will give you a prompt of whether or not you want to continue with the last client. As long as you’re connecting with a new client, choose No on this prompt for it to continue with the new client details being sent from ClientBase.

Importing an Existing Reservation

Step 1) Create a new reservation in ClientBase and after entering vendor information, click Live Connect and Import an Existing Reservation. Enter a confirmation number or leave blank, but when in booking site, locate the retrieve reservation area.

Step 2) When the reservation is confirmed on the Thank You Page, click Import Reservation.

All reservation data booked on-line is now located in the reservation fields in ClientBase.

Retrieve an Existing Reservation for Editing or Cancelling

To edit or cancel an existing reservation using Live Connect, follow these steps:

Step 1) Launch Live Connect from inside the existing reservation to edit or cancel in ClientBase.

Step 2) A dialog box appears. Click Retrieve Reservation.

Step 3) Live Connect launches the booking engine site. Edit or cancel the reservation using instructions supplied by the booking engine, and click Import Reservation. All reservation data edited or cancelled online is now located in the ClientBase reservation record.
