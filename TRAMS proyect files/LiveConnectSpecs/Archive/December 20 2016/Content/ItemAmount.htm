Specification Table

Element Name Data Type (Length) Example Required Travel Categories Notes
TRAVELCATEGORY Integer 3 (Cruise) N All See RESTYPE for valid values. Note Markups or Discounts will not
PASSENGER Char(40) Jones/Tom N All Must be included if you use ITEMAMOUNT
QTY Integer 1 N All  
DESCRIPTION Char(100) Fare, Port Charges, SurCharges(descriptions for customer to see) & Discounts N All Summarization option
PLUSMINUS Char(1) +/- N All  
ITEMBASE Integer 500000(5000.00) N All  
TAX Integer 45000(450.00) N All  
TAXPERCENT Integer   N All  
COMMISSION Integer 50000(500.00) N All  
COMMPERCENT Integer   N All  
REMARKS Char(500) Comprehensive internal detail descriptions ie NCF,Fuel Sur Charges, and Markups N All No Summarization.
