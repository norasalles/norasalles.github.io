Specification Table

Element Name Data Type (Length) Example Required Travel Categories Notes
PROVIDERTYPE Integer 0 = Air 1 = Hotel 2 = Car 3 = Cruise 4 = Insurance 5 = Miscellaneous 6 = Rail 7 = Tour 8=Service Fee 9= Transportation Y All Service Provider type.
SERVICEPROVIDERNAME Char(80) American Airlines Y All Name of Service Provider (free flow) Cruise Ship, Hotel Name, Description of Service.
SERVICEPROVIDERCODE Char(8) AA N 0, 6  
STARTDATE Date MMDDYYYY N All Date when this segment of the trip starts.
ENDDATE Date MMDDYYYY N All Date when this segment of the trip ends.
STARTTIME Time HHMM N All Check-in time, pickup time, or flight departure time.
ENDTIME Time HHMM N All Check-out time, drop off time, or flight arrival time.
SPBASE Integer 15000 (150.00) Implied decimal point location: 2 places to the right N Recommend do not use. Base amount of this segment of the trip.Note: this is optional information that the system does not use in the calculations.It just helps explain the totals where more than one service provider are within the reservation.
TAX Integer 1500 (15.00) N All Tax amount of this segment of the trip.
TAXPERCENT Integer 1000 (10.00) N All Tax percent relative to the SPBASE amount.
COMMISSION Integer 1500 (15.00) N All Commission amount of this segment of the trip.
COMMPERCENT Integer 1000 (10.00) N All Commission percent relative to the SPBASE amount.
ADDRLINE1 Char(50)   N 1,2,9 Hotel, Car Rental or Transportation Address.
ADDRLINE2 Char(50)   N 1,2,9  
ADDRCITY Char(30)   N 1,2,9  
ADDRSTATE Char(15)   N 1,2,9  
ADDRZIP Char(12)   N 1,2,9  
ADDRCOUNTRY Char(20)   N 1,2,9  
ADDRPHONE Char(40)   N 1,2,9 Hotel, Car Rental or Transportation Phone number.
ADDRFAX Char(40)   N 1,2,9 Hotel, Car Rental or Transportation Fax number.
CATEGORY Char(40)   N 1,2,3,6,7,8,9 Used in the following Travel Categories: Hotel: Room Type; Car: Car Category; Cruise: Category; Rail: Category; Tour: Tour Name Transportation: Type (Bus, Van, Limo, etc.).
DESCRIPTION Char(256)   N 1,2,3,6,7,8,9 Used in the following Travel Categories: Hotel: Room Description; Transportation Type: Air, Car, Rail, Cruise, Insurance, Tour; Miscellaneous: Descriptions.
DEDUCTIBLES Integer   N 4 Insurance: Deductible.
MAXCOVERAGE Integer   N 4 Insurance: Maximum Coverage.
SHIPDECK Char(20) ie Promenade N 3 Cruise: Name of the Deck.
CABINROOM Char(20)   N 3 Cruise: Number of Cabin.
DINING Char(30) Large Tble-Confirmed N 3 Cruise: Dining option.
SMOKING Char(1) T for Smoking F for Non Smoking N 1,3  
ITINREMARKS Char(4000)   N 1,2,3,6,7,8 Use “|” to separate remark lines.
AGENTREMARKS Char(4000)   N   Remarks in this section do not print in invoice. Use “|” to separate remark lines.
FLIGHTNO Char(10)   N 0, 6 Air/Rail: Flight or Train number/Car number.
DEPARTCITYCODE Char(3) LAX N 0,1,2,6,9 Car Pick Up City Code, Hotel Location.
DEPARTCITYNAME Char(80) Los Angeles N 0,2,6,9 Car Pick Up City.
ARRIVECITYCODE Char(3) MIA N 0,2,6,9 Car Drop Off City Code.
ARRIVECITYNAME Char(80) Miami N 0,2,6,9 Car Drop Off City.
CONNECTINGFLIGHT Char(1) Y/N N 0,6 Set to Y if this is a connecting flight segment.
AIRCRAFTTYPE Char(15) 747 N 0,6  
MEALSERVICE Char(20) Dinner N 0,6  
FLIGHTLENGTH Char(5) HH:MM N 0,6 Length of the flight.
CHECKINTIME Time HHMM N 0,6 Check in time.
DEPARTTERMINAL Char(20) 6 N 0,6 Airport/ rail station terminal.
ARRIVETERMINAL Char(20) E N 0,6 Airport/ rail station terminal.
SEGMENTMILEAGE Integer   N 0,6  
AIRCRAFTLOCATOR Char(10)   N 0,6  
NOOFSTOPS Integer   N 0,6  
MISCELLANEOUS Char(128)   N 0,6 Other remarks for this segment of the trip.
