CHARGEDAS

Data Type (Length): Integer

Required: N

Notes: This is dependent on the pricing fields above (TOTALBASE, TOTALTAX, TOTALCOMMISSION). If the prices are the total for the whole reservation, then assign 0 on the CHARGEAS field. If the prices refer to per passenger, then this will be 3. 1 if it's per night/day or 2 if its per policy etc. In the case of Tour reservations, this is usually set to 0.
