Specification Table – ServiceProvider – Cruise
Element Name Data Type (Length) Sample/Valid Values Format Required - Y, N or D (Desired) Default Used with Travel Category Description
PROVIDERTYPE Integer 0 = Air 1 = Hotel 2 = Car 3 = Cruise 4 = Insurance 5 = Miscellaneous 6 = Rail 7 = Tour 8 = Service Fee 9 = Transportation Y   All Service Provider type.
SERVICEPROVIDERNAME Char(80) American Airlines Y   All Name of Service Provider (free flow) Cruise Ship, Hotel Name, Description of Service
STARTDATE Date MMDDYYYY Y   All Date when this segment of the trip starts. Field is required, but of omitted, will be populated by Reservation level Start Date.
ENDDATE Date MMDDYYYY Y   All Date when this segment of the trip ends. Field is required, but of omitted, will be populated by Reservation level Start Date.
CATEGORY Char(40)   D   1,2,3,7,9 Used in the following Travel Categories: Hotel: Room Type Car: Car Category Cruise: Cabin Category Tour: Tour Name Transportation : Type (Bus, Van, Limo, etc)
DESCRIPTION Char(256)   D   All Used in the following Travel Categories: Hotel: Room Description, Car: Type, Transportation: Type, Cruise: Cabin Description, Insurance: Travel Policy Name, Tour: Description of package, Air: Misc Description, Rail: Misc Description, Miscellaneous: Description
SHIPDECK Char(20) ie Promenade, Deck # D   3 Cruise: Name of the Deck.
CABINROOM Char(20)   D   3 Cruise: Number of Cabin.
DINING Char(30) Freestyle, My Time, Early Dining, Late Dining D   3 Cruise: Dining option selected.
SMOKING Char(1) T for Smoking F for Non Smoking N F 1,3  
ITINREMARKS Char(4000) * Mandatory USD18.00 plus tax, per day per room payable direct to hotel Includes:- * Blu-ray, DVD and PlayStation 3 game rental * Daily coffee & tea * High-speed Internet access & Wi-Fi * Daily newspaper * In-room safe usage * Local & Toll Free Calls Conditions:- * Subject to change after time of booking D   All Use “|” to separate remark lines
AGENTREMARKS Char(4000)   N   All Remarks in this section do not print in invoice. Use “|” to separate remark lines
