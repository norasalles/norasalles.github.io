PASSENGER

RESCARD > PASSENGERLIST > PASSENGER

 

 

Application Screen – Passenger

 

Specification Table – Passenger

Element Name

Data Type (Length)

Sample/Valid Values
Format

Req
Y, N, or D (Desired)

Default

Used with Travel Cat

Description

LASTNAME Char(30)   Y   All Passenger last name.
FIRSTNAME Char(30)   Y   All Passenger first name.
MIDDLENAME Char(30)   D   All Passenger middle name.
PASSENGERTYPE Char(1) A = Adult C = Child I = Infant Y   0, 6 Type of passenger. Lap child are considered infant. Defaults to Adult.
PRIMARYPASSENGER Char(1) Y/N N N   If no node or not set, the first passenger will be set to the primary
