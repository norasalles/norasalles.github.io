Before you begin

You must have the keyword ACCESS and duty code 9 (type SI9 to activate) in your Employee Profile Record (EPR) to update the Travel Journal Record (TJR).

Add branch access to a single PCC:

W/TAA(your PCC)¥B(branch PCC)

W/TAAB4T0¥BKK41

Note: To have reciprocal branch access, each PCC must make this entry.

System Response:

BRNCH ACC UPDATE OK

 

 

Add branch access to multiple PCCs:

W/TAA(your PCC)¥B(branch PCC)/(branch PCC)

W/TAAB4T0¥BKK41/KJ41

Note: You can grant access to up to five PCCs in one entry.

Delete branch access from a single PCC:

W/TAD(your PCC)¥B(branch PCC)

W/TADB4T0¥BKK41

System Response:

BRNCH ACC UPDATE OK

 

Delete branch access from multiple PCCs:

W/TAD(your PCC)¥B(branch PCC)/(branch PCC)

W/TADB4T0¥BKK41/KJ41

Note: You can delete access to up to five PCCs in one entry.
