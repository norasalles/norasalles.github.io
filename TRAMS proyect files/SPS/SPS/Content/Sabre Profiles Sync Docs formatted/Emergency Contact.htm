Emergency Contact

  • If an emergency contact does not exist in ClientBase that does exist in Sabre Profiles, the emergency contact will be added to the ClientBase Traveler.
  • If the same emergency contact exists in both travelers (First Name and Last Name match), then if the Sabre Profile record is more recently updated, the ClientBase traveler will be updated. If the ClientBase traveler is more recently updated, then the Sabre Profiles traveler will be updated.
  • If the ClientBase emergency contact does not have both a first and last name PLUS a phone number and/or email address, it will not sync to Sabre Profiles, as it will not meet the Sabre Profiles requirements for an emergency contact.
