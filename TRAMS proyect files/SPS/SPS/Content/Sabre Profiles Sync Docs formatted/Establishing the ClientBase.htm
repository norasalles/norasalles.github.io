Establishing the ClientBase Profile’s Primary Address

  • If there is already a Primary address in ClientBase, that one remains marked as Primary.
  • If there is no Primary address in ClientBase, or if the Primary address is BLANK, the first Sabre Billing Address will be marked as the Primary in ClientBase.
  • If the Sabre Profiles profile does not have any Billing addresses, then the first Delivery Address will be marked as the Primary.
  • If the Sabre Profiles profile does not have any Delivery addresses, the first remaining Address will be marked as Primary
