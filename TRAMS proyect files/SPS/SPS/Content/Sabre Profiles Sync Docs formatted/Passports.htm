Passports

  • If a passport does not exist in ClientBase that does exist in Sabre Profiles, the passport will be added to the ClientBase Traveler.
  • If a passport does not exist in Sabre Profiles that does exist in the ClientBase traveler, the passport will be added to the Sabre Profiles traveler.
  • If the same passport exists in both travelers (Passport number + Issuing Country match), then if the Sabre Profiles Traveler record is more recently updated, the ClientBase traveler will be updated.
  • If the ClientBase traveler is more recently updated, then the Sabre Profiles traveler will be updated.
