Corporate and Group Profiles Matching Logic

Section 8

SP Travelers with an AN/DK (Goal: Preserve that AN/DK.)

 

If SP Profile type is Traveler and has an AN or DK of its own...

First look for a profile having a Traveler with a matching PASSENGER.SPUNIQUEID.

 If one match found...

Update that Traveler with the SP Traveler data.

 If more than one match found...

Log an error. Do not attempt any further action.
