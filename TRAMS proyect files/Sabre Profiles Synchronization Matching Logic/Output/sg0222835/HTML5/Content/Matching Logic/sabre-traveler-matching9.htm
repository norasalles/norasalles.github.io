Corporate and Group Profiles Matching Logic

Section 9

If no match found by PASSENGER.SPUNIQUE ID:

 

If they are associated with any SP Profiles...

Look for a CB Corporate or Leisure Profile that matches any of the associated SP Profiles by PROFILE.SPUNIQUEID.

If multiple matches found by PROFILE.SPUNIQUEID...

Log an error. Do not attempt to do any further actions.

If one match is found by PROFILE.SPUNIQUEID...

Check whether interface ID matches one of the SP Traveler's ANs/DKs.

If one CB Profile interface ID matches one of the SP Traveler's AN or DK...

Look at the Travelers within that Profile.

  • Look for a match by Known Traveler Number.
If one match is found by KTN...

Update the CB Traveler with the SP Traveler data and Link the Traveler to any associated profiles found.

If no match is found by KTN, or if there are multiple matches by KTN...

Look for a match by Passport Number + Issuing Country.

  • Both fields must match in order for this to be considered a match.
  • The matching passport may be in Passport #1 or Passport #2.
If one match is found by Passport number + Issuing Country...

Update the CB Traveler with the SP Traveler data and Link the Traveler to any associated profiles found.

If no match is found by Passport number + Issuing Country, or if there are multiple matches by Passport number + Issuing Country...

Look for a match by First Name + Middle Name + Last Name + Birth Date.

  • All four fields must match in order for this to be considered a match.
  • Middle name can be considered a match if CB, SP, or both have a blank field.
  • Birth Date can be considered a match if both sides are blank; if there is a value present both sides must have the same value.
If a match is found by First Name + Middle Name + Last Name + Birth Date...

Update the CB Traveler with the SP Traveler data.

If no match is found by First Name + Middle Name + Last Name + Birth Date, or if there are multiple matches by First Name + Middle Name + Last Name + Birth Date...

Insert as a new Traveler within the Profile.
