Logout

Logout ends the active session linked to the SessionID token.

Usage: <ServerURL>/logout

If the SessionID token is not passed as part of the Authentication header, then it needs to be specified in the request content:

 

  • Request Type: POST
  • Request Format: JSON
  • Request Content: {“SessionID”:<SessiondID token>}

 

Response: A resultcode of 6 indicates the session was not found.
