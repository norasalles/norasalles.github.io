Version Method

Version returns the current API version. SessionID token is not required.

Usage: <ServerURL>/version

Response: <version number>, i.e. 4.06.00.032

All other methods need to be a POST with the parameters specified in JSON format in the request content and return, at a minimum, a JSON response in the following format:

 

{
    "apiVersion": "4.06",
    "result": {
        "resultcode": <numeric result code>,
        "resulttext": "<text response>"
    }
}

 

A resultcode of zero indicates the operation was successful. Non-zero result codes indicate an error or other condition that needs to be handled.
