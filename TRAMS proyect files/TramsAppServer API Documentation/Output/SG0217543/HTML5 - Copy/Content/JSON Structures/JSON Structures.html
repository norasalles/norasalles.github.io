JSON Structures

Element

Element

Element

Format

Params

 

 

 

{“Params”:[Param, …]}

 

Param

 

 

{“Name”:<ParamName>, “Type”:DataType, “Value”:<ParamValue>}

 

 

DataType

 

“Integer”, “LargeInt”, “Boolean”, “DateTime”, “Date”, “Double”, “Blob”, “String”

 

 

 

 

 

Dataset

 

 

 

{“Dataset”:{MetaData, Data}}

 

MetaData

 

 

“MetaData”:{<DatasetName>:MetaDataRow}

 

 

MetaDataRow

 

{MetaDataColumn, …}

 

 

 

MetaDataColumn

“Name”:<ColumnName>:”Type”:DataType

 

Data

 

 

“Data”:{<DatasetName>:[DataRow, …]}

 

 

DataRow

 

{DataColumn, …}

 

 

 

DataColumn

<ColumnName>:<Value>

<Value> can be a nested Dataset

 

 

 

 

 

DeltaDataset

 

 

 

{“DeltaDataset”:DeltaData}

 

DeltaData

 

 

{<DatasetName>:[DeltaDataRow, …]}

 

 

 

DeltaDataRow

 

{“UpdateAction”:<DeltaAction>, DeltaDataColumn, …}

 

DeltaAction: None, Insert (New), Update (Old/New), Delete (Old)

 

 

 

DeltaDataColumn

<ColumnName>:{“OldValue”:<Value>, “NewValue”:<Value>} (Insert/Update/Delete)

 

<ColumnName>:<Value> (UpdateAction=None or Nested Dataset)

 

 

 

 

 

Result

 

 

 

{"result": {"resultcode": <resultcode>,"resulttext": <resulttext>, <return-param-name>:<return-param-value>}}
