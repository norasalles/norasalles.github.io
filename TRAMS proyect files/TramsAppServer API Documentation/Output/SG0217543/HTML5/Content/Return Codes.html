Return Codes

Return Codes

Code Meaning Description
200 Successful Response

The request succeeded. Information returned with the response depends on the method used in the request.

Get: The entity corresponding to the requested resource is fetched and transmitted.

Post: The entity describing or containing the result of the action is transmitted.
400 Bad Request The API does not support, or is unable to recognize, the request made to it.
404 Not Found The resource is invalid (unknown).
500 Internal System Error The server encountered an unexpected condition that prevented it from fulfilling the request.
