An In-Depth Look at Live Connect

This chapter provides a detailed overview of using the Live Connect feature in ClientBase.

Introduction

One of the most innovative features in ClientBase is Live Connect which is found in the Res Card. Save time and improve efficiency by transmitting selected client information to a booking engine on-line. Then electronically transmit back reservation details and automatically complete the reservation form with what’s been booked to generate invoices.

Note to ALL ClientBase Users: New Live Connects Providers are added between ClientBase releases all the time. View the newest list on the Trams website here:

http://static.trams.com/tramslibrary/liveconnect/index.html

Note to Trams Back Office and ClientBase Windows Users: (As ClientBase shares a database with Trams Back Office, the invoices generated from the Res Card automatically become part of the invoices shared with Trams Back Office.) ClientBase continues to add more on-line booking engines to connect to via Live Connect. All Live Connect Provider settings, including login information and vendor profile setup can be done in ClientBase Windows, or if a user has been granted administrative permission can be done in ClientBase under Tools|Settings (only in a Primary-Sync database (ClientBase Windows) or a Non-Sync activated database in ClientBase).

Live Connect Comparison Shopping

Want to shop on multiple Live Connect sites for a client?  In order to make comparison shopping easier, Live Connect now allows up to five simultaneous sessions of Live Connect, when launched from the Res Card level and using the Create New Reservations option. All Live Connect providers launched in a single session must belong to the same Travel Category. After starting the first Live Connect session, users can click on the plus (+) sign to start another session in a new tab. Each new Live Connect session will pass the same checked profile items as the first one.

 

Step 1) Create a client Res Card and fill in the Agent, Status, Reservation Cycle, Trip Name, and Region/Destination fields and select the travelers.

Step 2) From the Res Card level, click Live Connect, select the Live Connect Provider from the Connect To: drop-down and click Create new reservations.

Step 3) Select customer data to book the reservation (Traveler, Credit Card, Phone and Billing Address, etc.) and click Connect. Select customer data to book the reservation (Traveler, Credit Card, Phone and Billing Address, etc.) and click Connect.

Step 3) Although still in ClientBase, you are now on-line at the first Live Connect booking site you want to view.

Step 4) Click on the plus (+) sign to start another session in a different tab with a new Live Connect Provider. All Live Connect providers launched in a single session must belong to the same Travel Category. You may select up to five. Each new Live Connect session will pass the same checked profile items as the first one.


Step 5) When a reservation is booked and confirmed in a Live Connect provider booking site, click Import Reservation. All reservation data booked on-line is now located in ClientBase.
