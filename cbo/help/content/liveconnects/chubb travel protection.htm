Set Up Chubb Travel Protection

To partner with Chubb Travel Protection, call +1 844 772 3493 or email ChubbTravelSales@chubb.com.

Upon receiving login information, take the following steps to setup ClientBase for Live Connect.

Step 1) Create a Live Connect Provider for Chubb Travel Protection if none exists by going to Tools|Live Connect Providers and clicking Add.

Check if the URL is correct, and fill in the following fields:

Provider Name: 

Chubb Travel Protection

Status:

Active
- If you ever want to make it Inactive, go back in this area to turn off.

URL:

https://ctiusa.chubbtravelinsurance.com/en/client-base/?postQuote

Travel Category:

Insurance

Import XML from: 

HTML Source

Include Authentication: 

Leave blank

Step 2) Set up for Live Connect agent logins can be done individually. When Live Connect is launched, the system first looks to see if there is login information at the User Login level, and if not, then asks the user to login at the provider website. Users can set up their own logins by going to Tools|My Login|Live Connect. Click Add and enter the following information:

Live Connect Provider: Select Chubb Travel Protection from the drop-down list.

Enter your User Name, your password, and Account Number.

Step 3) Retrieve the Chubb Travel Protection vendor profile (or create if not in database). From the General Info link, click the Live Connect Providers button. Click Add, then select Chub Travel Protection from the drop-down list. The vendor code is CB. In the Travel Category profile field, select Tour from the drop-down menu. Click OK.

 

Live Connect Features Supported by Chubb Travel Protection

Chubb Travel Protection supports the following Live Connect features if checked (please see further below for information on importing and retrieving an imported Quote):

 

Create New Chubb Travel Protection Reservation

Step 1) Retrieve the desired client profile, create a new Res Card and after entering basic Res Card info, launch Live Connect by clicking Live Connect on the menu bar. (You can also launch Live Connect from the Profile action gear.)

Step 2) Complete the Live Connect settings by identifying the vendor, the Live Connect Provider and appropriate Login information.

Step 3) Click the Create New Reservation button.

Step 4) Select any Profile, Passenger, Branch and Res Card information you want to use to book the reservation and click Connect to launch the on-line booking engine.

Step 5) Although you are still in ClientBase, you are now on-line at the Live Connect Provider’s booking site running the on-line booking engine through a ClientBase window. Complete the reservation and when the reservation is confirmed, click Import Reservation from the confirmation page. All reservation data booked on-line is now located in the reservation fields in ClientBase.

 

New Feature: Import a Quote

Chubb Travel Protection introduced a new feature November 2020, allowing agents to Save a Quote, and once Saved, you’ll receive the prompt seen below and can click on Import Reservation to import the quote into your ResCard after clicking “OK”.

 

You can then later use the Retrieve Reservation feature to reconnect, continue with the booking process, and import the completed booking. In ClientBase, this will overwrite the Quoted status Reservation with the new Confirmed Reservation details.
