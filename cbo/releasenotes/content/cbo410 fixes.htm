ClientBase Online 4.10.00 | New Features and Enhancements
Release date: 18 Mar, 2021

 

ClientBase Online 4.10.00 Fixes

Changes made between ClientBase Online 4.09.00.108 and 4.10.00.84

Chrome Support Related

  • Profiles Manager

    • Fixed an issue where selecting a Saved Search from the Profile Manager dropdown menu did not result in querying for the records. 

    • Fixed an issue where the application was freezing when saving a Loyalty Program or Credit Card.

    • Fixed an issue where editing any fields on the General Info page resulted in the Additional Info section displaying incorrectly.

    • Fixed an issue where the Create Res Card page failed to open when selecting the Res Card option from the Create Res Card dropdown menu on the Res Card page in Profiles. 

    • Fixed an issue on the Profile Advanced Search page under the Res Card tab where clicking on the Select Vendor dropdown opened a new web browser tab instead of expanding the list with available Vendors.

    • Fixed an issue where a user could not clear the highlight of records selected after running a query in the Profile Manager.

       

  • Activity Manager

    • Fixed an issue where selecting a Saved Search from the Activity Manager dropdown menu did not result in querying for the records.

    • Fixed an issue where dropdowns (Time, Duration, Login name, Reminder Type, Subject) on the Reminder Create page were not updating entries.

    • Fixed an issue where dropdowns (Time, Login Name, Subject) on the Note Create page were not updating entries.

    • Fixed an issue where dropdowns (Time, Login Name, Mailer Type, Subject) on the Mailer Create page were not updating entries.

    • Fixed an issue where the GOTO button would not grey out when no data was entered in the Link To Details field on the Mailer Create page.

    • Fixed an issue on the Activities Advanced Search page under the Inventory tab where clicking on the Select Vendor dropdown opened a new web browser tab instead of expanding the list with available Vendors.

    • Fixed an issue where a user could not clear the highlight of records selected after running a query in the Activity Manager.

       

  • Res Cards Manager

    • Fixed an issue where selecting a Saved Search from the Res Card Manager dropdown menu did not result in querying for the records. 

    • Fixed an issue where the Create Activity page failed to open when selecting an Activity Type from the Create Activity dropdown menu on the Activity page in Res Cards.

    • Fixed an issue where clicking the Add button on the Attachments page in Res Cards caused the header to display incorrectly.

    • Fixed an issue where the Marketing Source, Group, and Region selections did not save when creating a Res Card.

    • Fixed an issue where selecting the Service Provider option from the Create New Profile dropdown did not open the Create Service Provider page.

    • Fixed an issue where the yellow info box was misplaced when changing the Charge As option for Hotel reservations.

    • Fixed an issue where a user could not clear the highlight of records selected after running a query in the Res Card Manager.

       

  • Merge to PNR

    • Fixed an issue when editing Merge To PNR Rules or Merge to PNR Settings where the mouse cursor reverted to the beginning of the entry after each Merge Field insert action.

       

  • PNR Import

    • Fixed an issue on the Vendor Not Found page where clicking on the Select Vendor dropdown opened a new web browser tab instead of expanding the list with available Vendors.

       

  • Templates

    • Fixed an issue where adding a Merge Field to a Document Template caused the previous Merge Field to be highlighted and replaced.

    • Fixed an issue when editing Document Templates where the mouse cursor reverted to the beginning of the entry after each Merge Field insert action.

       

  • Agency Settings

    • Fixed an issue under Insurance Prompting where clicking on the Select Vendor dropdown next to the Insurance Vendor name field opened a new web browser tab instead of expanding the list with available Vendors.

       

  • Login & Diagnostics

    • Fixed an issue where the Sabre PCC field on the Login page was not fully visible when entered using Caps Lock.

    • The ClientBase Online Diagnostics Check, which ran upon logging in, has been removed since Google Chrome does not support Microsoft Active X Controls.

       

  • Invoice Related

    • Fixed an issue where it was impossible to change the Reservation Tax or Commission amounts when tracking the Res Card Trip Cost in Foreign Currency.

    • Fixed an issue where the Refund table was misaligned on the Generate Invoice page. 

    • Fixed an issue on the Generate Invoice page where the GST/HST label on the Sales table was misplaced, and the table lines were missing.

    • Fixed an issue on the Generate Invoice page where a warning message could not be closed when entering a higher amount than what was available to be invoiced (applies to fields: Base, Tax, Comm, GST / HST).

    • Fixed an issue on the Generate Invoice page where the Agent section header was not in line with the table border.

    • Fixed an issue on the Generate Invoice page where the Agent table borders were not complete when an Agent had the Disable the ability to view agent commission when invoicing setting turned on.

       

  • Dashboard

    • Fixed an issue where the Advisor Dashboard was not displaying in full-screen mode.

    • Fixed a problem where the Recent Profile label and the + icon for creating new Profiles on the Advisor Dashboard were not displaying correctly.

 

Profile Related

  • Fixed an issue where the Profile Header links: Interface ID and Created were not opening the correlated areas on the General Info page in edit mode.  
  • Fixed an issue where the Travel History in Detail View Mode was running off the viewable screen.
  • Fixed an issue where the Travel History sort by column feature was not working correctly.
  • Fixed an issue where the Referred By field did not allow a free flow entry for users with the Disable change Profile "Create Date" and Profile "Create User"' settings turned on.

 

Res Card Related

  • Fixed an issue where Advanced Search on the Res Card Manager returned an incorrect number of records when there were more than approx. 200 results.
  • Fixed an issue where the Vendor name was unable to be changed when Traveler Pricing existed on a reservation.

 

Invoice Related

  • Fixed an issue where creating a Refund invoice for only the tax amount caused the invoice to show as a Sale invoice.
  • Fixed an issue where after importing a PNR from a GDS and clicking on the Generate Invoice button, the page froze.

 

Saved Search Related

  • Fixed an issue where a user couldn't cancel the creation of a Saved Search when leaving the Name field blank.

 

Agency Settings Related

  • Fixed an issue in the My Login settings on the PNR Import tab where the Delete button on a Worldspan PNR Import record was not working correctly.
  • Fixed an issue where users were able to edit the Primary Agent on a Profile, Activity, or Res Card to an agent who did not have permissions for the branch selected.
  • Fixed an issue where updates to the Travel Category defaults were not saving in the Default Field Values for New Invoices settings.

 

Forms

  • To avoid saving errors when users paste values other than the permitted numerical values, we added validation to identify invalid values on the following pages:

    • Agency Settings > Membership ID Fields tab > Max Length field

    • Create Activity page > Profile No. field

    • Create Activity page > Res Card No. field

    • Res Card Manager > Simple Search > Res Card Record Number field

    • Inventory Search > Inventory Detail Record # field

    • Inventory Details Search > Inventory Detail Record # field

    • Inventory Flight Info Search > Segment Mileage fields > Number of Stops field
