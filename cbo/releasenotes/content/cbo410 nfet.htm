ClientBase Online 4.10.11 New Features and Enhancements

Enhancements made between ClientBase Online 4.09.00.108 and 4.10.00.84

Google Chrome Related

  • Added core functionality support of ClientBase Online with Google Chrome for Windows and Google Chrome for Mac. We have introduced several fixes and changes so that the use of the application in key areas is fully possible.  Those key areas include all functionality within the Dashboard, Profile, Activity, and Res Card Managers as well as Invoicing, Templates, Configuration settings, Security Group settings, and Agent Rights permissions.

 

Merge to PNR Related: Apollo & Galileo

  • In preparation for a Smartpoint-only version of Galileo Desktop, we have updated how the communication works for our Merge to PNR feature. When you Merge to PNR for Apollo or Galileo, ClientBase Online now detects if the Travelport Smartpoint application is running and sends the selected data to the active Area in that application. If Smartpoint is not running, then ClientBase Online will send the data to the active Area in Galileo Desktop, as it previously did. ClientBase Online supports this functionality in both single and multiple PCC environments.

    Please note when using the CTRL-F12 shortcut to toggle back and forth between Galileo Desktop and Smartpoint, the hidden application is no longer available to ClientBase Online. In this case, ClientBase Online sends the Merge to PNR data to the currently active application. Alternatively, minimizing Smartpoint (with the minimize button in the top right corner of the application) keeps the application available. In this case, ClientBase Online will still send the Merge to PNR data to the Smartpoint application.

 

Invoice Related 

  • For increased intuitiveness and usability in the invoicing process of both Sale and Refund invoices on the Generate Invoice page, we changed the validation mechanism in cases where the user is trying to enter incorrect characters (other than digits) or attempts to invoice a higher amount than what is available to invoice (applies to fields: Base, Tax, Comm, GST / HST). In such a case, ClientBase Online will prevent invalid entries and present a message about the incorrect value. The descriptive messages will display directly above the field to which they relate. When leaving an invoice field containing a greater amount than what is available to invoice, the dollar value will automatically reset to the last correct amount entered by the user.
