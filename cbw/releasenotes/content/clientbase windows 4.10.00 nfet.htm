ClientBase Windows 4.10.00 New Features and Enhancements

Enhancements made between ClientBase Windows 4.09.00.090 and 4.10.00.003

Merge to PNR Related: Apollo & Galileo

  • In preparation for a Smartpoint-only version of Galileo Desktop, we have updated how the communication works for our Merge to PNR feature. When you Merge to PNR for Apollo or Galileo from a Profile, Res Card, or Invoice, ClientBase will now detect if the Travelport Smartpoint application is running and send the selected data to the active area in that application. If Smartpoint is not running, then ClientBase will send the data to the active area in Galileo Desktop, as it previously did. This functionality is supported in both single and multiple PCC environments.

    Please note that if you are using the CTRL-F12 shortcut to toggle back and forth between Galileo Desktop and Smartpoint, the hidden application is no longer available to ClientBase. In this case, ClientBase will send Merge to PNR data to the application that is currently active. Alternatively, minimizing Smartpoint (with the minimize button in the top right corner of the application) keeps the application available. In this case, the Merge to PNR data will still be sent to the Smartpoint application.
