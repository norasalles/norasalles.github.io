Air Configuration Beta

Path: Air > Air Configuration Beta Page

  1. To enable air searches on the new Home page, select the Enable air search check box.

  2. To enable air multiple destination searches on the new Home page, select the Enable multiple destination air search check box.

    Notes   

    • The air multiple destination search can be enabled without air search enabled, but Air Multi Destination does not display if Air Search is not enabled.
    • These settings do not change the existing Begin Search page.
  3. Select whether to Enable Branded Fares. This allows users to shop branded fare air content on sites that have Sabre configured as the primary GDS. If selected, users can then access to the branded fare content, view what is bundled as part of the offering, and book the fare that aligns with the travel policy and their requirement. Users can also modify and put on hold bookings with branded fares.
  4. Select whether to For Apollo and Galileo GDS, only enforce the search time window configured for all shopping requests. This makes the Extend the search before and after the time the user selected settings on the Air Configuration page work for Apollo and Galileo on the Search by price path and the Low Fare Options page just as they currently do for other GDS. If you don't select this check box, additional flight options in an 8 hour window are returned to the user.
  5. Click Save.
