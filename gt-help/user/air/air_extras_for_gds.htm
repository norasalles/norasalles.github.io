Air Extras for GDS

Path: Air > Air Extras for GDS

If Sabre is the primary GDS for the site, you can use the Air Extras for GDS page to configure Air Extras for the site.

Notes   

  • Air Extras should not be enabled for sites that allow multiple passengers. For more information, see Select the maximum number of travelers per itinerary on the Air Configuration page.
  • If a carrier does not return a fee amount in their seat map response, the system allows those carriers’ paid seats to be selected even if Air Extras is not configured.

To configure Air Extras

  1. To enable Air Extras for the site, select the Enable Air Extras check box.
  2. Select whether to show that Air Extras are reimbursable as follows:
    • To show a message on the Seat Map page that prepaid seat fees are reimbursable and on the Trip review and checkout page that Air Extras are reimbursable, select the Display a message that fees for Air Extras are reimbursable check box.
    • To show a message on the Seat Map page that prepaid seat fees are not reimbursable and on the Trip review and checkout page that Air Extras are not reimbursable, clear the Display a message that fees for Air Extras are reimbursable check box.

  3. Select whether to Allow travelers to book prepaid seats.

    You cannot clear this check box when you have selected the Enable Air Extras for GDS check box.

  4. Use the Add and Remove buttons to move carriers to the Enabled Carriers list for which you want travelers to be able to book prepaid seats.

    Note  At least one carrier must be enabled when Air Extras and book prepaid seats are enabled.

  5. Click Save.
