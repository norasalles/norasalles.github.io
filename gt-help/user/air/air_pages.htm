Air Pages

Path: Air > Air Pages

You can use the Air Pages page to configure the screen layout and defaults displayed on the Flight Search (Home) page and Flight Availability page, and similar options under the Flight Preferences page of the user's profile. This page also enables the display of custom messages for preferred airlines.

To configure air pages

  1. Configure items in the Flight Search and Air Preferences section as follows:
    1. In the Advance Flight Search Page list, select whether the Round-trip, One-way, or Multi-destination (Multi-City) option button is selected by default on the Flight Search (Home) page.
    2. Ignore the sort options as they are no longer supported.

    3. Select What list of airlines should be displayed in the Preferred Airline lists on the Flight Search and User Profile pages.

      You can choose the default which is all carriers in the list below, create a custom list from below, or choose none. (Choosing none removes the preferred airline section from the Begin Search page and the profile.)

      • If a customer’s strategy is to promote preferred, you may want to suggest disabling the list.
      • If the customer requests that a list be displayed, you may want to suggest only listing the company’s preferred carriers. 
      • If the you want to use the default list, do the following on the Air & Car Setup page:
        1. In the What kind of request for airline availability should the system send to the host GDS list, click Request all airlines from the GDS.
        2. Select the Group preferred airlines and display as the first options on the airline availability page check box.  The preferred vendors appear at the top of the page and the non-preferred vendors appear at the bottom of the page.

      Some concerns to consider when list is displayed:

      • If the traveler selected preferred carrier is the same as a company preferred carrier for that market, only that carrier flight option(s) displays.
      • Showing the list to Travelers may lead them to think they can bias the availability display to their selected preferred carrier.  This may not happen if a company preferred carrier has been noted in Market Based Preferences for that market.  Company preferred carriers override Traveler preferred carriers.

      If you click Custom, in the list of airlines, click the airlines you want as preferred. To select multiple items, hold down the Ctrl key while you click other items that you want to select.

    4. Select whether to Display a custom message above the Preferred Airline List. If you select this check box, do the following:
      1. In the next box, type a label (header) for the custom message.
      2. In the next box, type a message.

      Note  Use of HTML is allowed but make sure tags are written correctly.

    5. To display the cabin class option for the departing and returning flights, select the Enable users to select a different service class for return flights on round trips check box.

      Note  This check box should not be selected for Apollo, Galileo, and Amadeus sites as mixed class of service is not supported.  If this is enabled for these GDS, the pricing format brings the itinerary down to the lower class and does not price them in mixed classes of service.

    6. To displays the cabin class option for each flight on the multi-destination Home page, select the Enable users to select a different service class for each flight on multiple destination trips check box.

      Note  This check box should not be selected for Apollo, Galileo, and Amadeus sites as mixed class of service is not supported.  If this is enabled for these GDS, the pricing format brings the itinerary down to the lower class and does not price them in mixed classes of service.

  2. Configure the service classes that are to be displayed and the names for each service class as follows:
    1. For each service class, select one of the following from the list:
      • Display the service class using the default text.
      • Display the service class using custom text. If you select this option, enter the custom text in the box to the right. To prevent the booking tool page from expanding, type no more than 45 characters.
      • Do not display this service class.
    2. Select the default service class.
  3. Configure the pricing options that are to be displayed and the names for each pricing option as follows:
    1. For each pricing option, select one of the following from the list:
      • Display the pricing option using the default text.
      • Display the pricing option using custom text. If you select this option, enter the custom text in the box to the right. To prevent the booking tool page from expanding, type no more than 45 characters.
      • Do not display this pricing option.
    2. Select the default pricing option.
  4. Configure options In the Flight Availability Pages section, as follows:
    1. Ignore the Display the negotiated airfare icon check box because it is no longer valid as it works with Air Contract fares, and these types of fares are no longer supported in GetThere. 
    2. Select whether to Display the negotiated airfare icon. If you select this check box, the system displays of a single gold star image and message for any flight option that has an airfare using your negotiated prices.
    3. If you select the previous check box, you can Enter your custom discounted message.
    4. Select whether to Display codeshare carrier information on the Flight/Equipment column on the Flight Availability page.
  5. Click Save.
