Custom Airline Notes

Path: Air > Custom Airline Notes

A custom airline note (AKA configurable pricing display) is a tool that creates a hyperlink (and optionally, shows an icon) that takes the user to detailed information about specific flights on the Availability page. The hyperlink is applied to all flights that match the criteria specified and can include a custom message about a discount, negotiated amenity, or other promotion. You can also use a custom airline note to convey reasons to avoid certain flights. For example, you can configure a custom airline note to warn travelers to avoid a specific airline that has repeated safety violations.

Note  A custom airline note does not create discount fares; it only applies a hyperlink to flights that match the specified criteria.

For more information, click one of the following:

Create a Custom Airline Note

  1. Click Add.
  2. In the Note Name box, type a name to identify the custom airline note. This name is not displayed to users.

  3. In the Sequence number box, type the number that determines the order of the notes in the database.

    In cases where more than one note can be applied to a flight, the system uses the one with the lowest sequence number. For this reason, it is usually good policy to leave a gap of 10 to 20 numbers between sequence numbers, so you can create new notes with lower sequence numbers that take precedence over the older ones.

  4. In the Airline list, click the airline for which the note applies. You must associate each note with an airline and you can only associate one airline with a note.

  5. Ignore the Icon list because only one icon now appears as follows:

    On the card in the upper-right corner, the following appears:

    When you click the card, the following appears in the drawer that drops down:

  6. In the Start Date box, enter the first date to apply the custom airline note to matching flights.
  7. In the End Datebox, enter the date to stop applying the custom airline note to matching flights.
  8. If you want to select only certain days of the week on which to apply the custom airline note in the specified date range, in the Effective Day(s) of Week list, click the days you want. By default all days of the week are selected.

  9. If you want the note to only appear for certain classes of service, in the Class of Service Code(s) box, type the single letter code for the classes of service, such as Y, C, or F. Type the code(s) in uppercase, and separate each code by a comma. Contact your travel agency for assistance with determining the class of service codes.

  10. If you want the note to only appear for flights using a specific fare basis for the airfare, in the Fare Basis Code(s) box, type the fare basis codes. Separate multiple codes with a comma and use uppercase.

    Notes   

    • The fare basis trigger for the custom airline note only applies when the traveler uses the Search by price path.
    • If you want to display the note by Fare Basis Code, you must select Yes in the Display prices on the availability pages for each flight choice list on the Air > Air Configuration page in the Fare Options section.

    You can also use wild cards in the Fare Basis Code(s) box. For more information, click here.

    You can use the following operators:

    Operator

    Description

    ^

    Starts with

    .+

    The following are examples of using wild cards:

    Note  “regex” should always be in lowercase and is used as an identifier for the rest of the expression inside the “()” as the code pattern.

    1. To find any fare basis code that ends in the number "9" (applies to any number 0-9), enter the following format in the Fare Basis Code(s) box:

       

      regex(.+9$)

       

    2. To find any fare basis code that ends in the letter "R" (apply to any letter in the alphabet), enter the following format in the Fare Basis Code(s) box:

       

      regex(.+R$)

       

    3. To find any fare basis code that ends in the special character "@" (apply to any special character), enter the following format in the Fare Basis Code(s) box:

       

      regex(.+@$)

       

    4. To find any fare basis code that begins with number, letter or special character, enter the following format in the Fare Basis Code(s) box:

       

      regex(^[9|R|@].+)

       

      This statement means FB starts with ‘9’ or ‘R’ or ‘@’

      Custom Note would apply to a FB that was a R21NR

      Custom Note would NOT apply to a FB that was F9R@

       

    5. To find any fare basis code that BOTH begins with the number, letter or special character specified AND ends with the number, letter or special character specified, enter the following format in the Fare Basis Code(s) box:

       

      regex(^[9|R|@].+[9|R|@]$)

       

      This statement means FB starts with ‘9’ or ‘R’ or ‘@’ AND ends with ‘9’ or ‘R’ or ‘@. The “.+” in the middle of statement means there can be any characters in between.

      Custom Note would apply to a FB that was a R21NR

      Custom Note would NOT apply to a FB that was F9R@

       

    6. To find any fare basis code that includes the number, letter or special character specified, enter the following format in the Fare Basis Code(s) box:

       

      regex(.+[9|R|@].+)

       

      This statement means FB starts with ‘9’ or ‘R’ or ‘@’ AND ends with ‘9’ or ‘R’ or ‘@. The “.+” in the middle of statement means there can be any characters in between.

       

  11. If you want the note to only appear for specific flight numbers or a range of flight numbers, in the Include Flight Number(s) box, type the flight numbers. Use a dash to create a flight range. Separate each flight number or range of flight numbers by a comma. For example: 245, 3000-4567.
  12. If you want the note to only appear for all flights except for specific flight numbers or a range of flight numbers, in the Exclude Flight Number(s) box, type the flight numbers. Use a dash to create a flight range. Separate each flight number or range of flight numbers by a comma. For example: 245, 3000-4567.

  13. If the note applies to flights that depart after or before a specific time, do the following:

    Note  If using time to apply your note, select both an earliest and latest departure time, or the time will default to 00:00.

    To show flights that depart

    Select the time in the

    On or after a specific time

    On or before a specific time

    Latest Departure Time list.
  14. In the Hypertext Link Text box, type the text you want to display on the flight availability pages when a flight matches your note criteria. This text will appear as a hyperlink and you can enter HTML in this field to format the text. The maximum number of characters including HTML is 30.
  15. In the Note Text box, type the text you want to display in a pop-up window when users click on the note hyperlink. You can enter HTML in this field to format the text. The maximum number of characters including HTML is 512.
  16. Click Save.

Edit a Custom Airline Note

  1. In the Name column, select the custom airline note that you want to edit.
  2. Edit the options as described in the Create a Custom Airline Note section above.
  3. Click Save.

Delete a Custom Airline Note

  1. Select the check boxes for the custom airline notes you want to delete.
  2. Click Delete.
  3. To confirm that you want to delete the selected custom airline notes, click OK.

Delete all Custom Airline Notes

  1. Select the check box in the header row.

  2. Click Delete.

    A confirmation dialog box appears.

  3. Click OK.
