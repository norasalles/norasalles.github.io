Pricing & Stored Fare Options

Path: Air > Air Configuration > Pricing & Stored Fare Options

If Amadeus is your GDS, you can use this page to enable pricing and stored fare options at the site level.

Notes   

  • This page only appears if you select Amadeus as the GDS on the GDS page (Setup > Super User > GDS).
  • The site must have Amadeus Web Services activated for the validating carrier, selling point override, and ticketing point override to fully work.

To enter pricing and stored fare options

  1. If you want the validating carrier of the selected fare included in the format when the fare is stored, select the Store the carrier who validated the fare in the PNR. check box.
  2. In the Selling Point Override box, type an airport or city code to be used as the selling point in the entries to price and store fares.
  3. In the Ticketing Point Override box, type an airport or city code to be used as the ticketing point in the entries to price and store fares.
  4. Click Save.
