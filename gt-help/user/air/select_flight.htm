Select Flight

Path: Air > Select Flight

You can use the Select Flight page to filter out availability options with a maximum number of connections and sold out options and to enable Traveler Filtering.

To select flight configuration options

  1. In the Maximum number of stops for display list, click the maximum number of flight connections to display on the availability page.

  2. Select whether to Display sold out flightsof the requested class of service on the availability page. If another class of service is available, this is indicated and may be available to book based on air configuration settings.

    Notes   

    • Do not select this check box to prevent sold out flights from displaying.
    • Sold out flights display on the last page of flight options that come back in availability.
  3. Click Save.
