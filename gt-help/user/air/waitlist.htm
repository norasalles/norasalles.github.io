Waitlist

Path: Air > Waitlist

If a flight segment is not available in the desired cabin at the time booking, the waitlist feature allows travelers to place their names on a list of people desiring passage on a full flight. The airline processes the waitlist when they receive cancellations on the desired flight. These site administration settings control the itinerary options allowed in a booking that contains a waitlisted segment, the actions the booking takes after receiving a waitlist clearance from the airline via the GDS, and additional messaging.

Note  This feature is available on Sabre and Abacus sites and is enabled by a GetThere account manager.

To configure waitlist options

  1. Select whether you want to Allow travelers to add a car and/or hotel booking if they have waitlisted flight segments.
    • If you select the check box, select whether you want to allow travelers to add a Car booking, a Hotel booking, or both a Car and Hotel booking in an itinerary containing a waitlisted segment.
  2. Select whether you want to Send an e-mail when a flight clears the waitlist. If you select this check box, continue as follows:
    1. Select whether you want to send the e-mail to the Traveler, the Traveler and booking travel arranger, or the Traveler and all assigned travel arrangers.
    2. If you want to display custom text at the top of the e-mail notification, type the custom text in the first Custom Text box.
    3. If you want to display custom text at the bottom of the e-mail notification, type the custom text in the next data entry box.

    4. Optional: Enter text you want displayed at the bottom of the e-mail notification.

      Notes   

      • If you choose Traveler and all assigned travel arrangers, the system sends an e-mail to the traveler and up to 10 randomly selected Traveler Arrangers.
      • HTML is not supported in the top or bottom e-mail notification message.
      • The default top message for the e-mail notification is: WAITLIST CLEARED, please confirm flight(s) to avoid any cancellations. There is no default text for the bottom message.

  3. If you want to display custom advisory text in a popup box when travelers select their first waitlist option during the booking process, type the custom advisory text in the second.Custom Text box.

    Notes   

    • The default text is: Please note that the airline controls whether your request can be fulfilled.
    • The default text or your custom advisory message is appended to the following message, which cannot be customized: "You have selected a waitlist option for your itinerary. You may select a combined total of 20 flights for your entire itinerary, including both waitlist options and confirmed flights."
  4. Type the dedicated Queue number for processing waitlist cleared e-mail notification.
  5. Type the Temporary queue number for cleared waitlist.
  6. Type the Cleared waitlist work queue number.

  7. If you want to advise travelers about credit card charges and waitlisted segments, select the Display an alert message when travelers click the Submit Trip button check box.

    If you want to enter custom text you want displayed to travelers when they have a waitlisted segment and they click the Submit Trip button, type the custom text in the Custom Alert Message box.

    Note  The default alert message is: Your credit card may not be charged until your flights are confirmed and there are no waitlisted segments in your itinerary. Click OK to confirm your transaction or click Cancel.

  8. Click Save.
