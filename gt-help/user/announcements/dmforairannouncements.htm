Dynamic Messaging for Air Announcements

The GetThere-certified JavaScript Messaging API supports the ability to add air-related announcements, popup messages, and banners (that are displayed to travelers) based on the criteria outlined below. For more detailed information, please refer to JS Messaging API and Class UIglobals. Please note that you must have strong working knowledge of JavaScript to create dynamic messages.
