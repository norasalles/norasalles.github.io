You can display the following custom messaging on the Home page:

Type

Description

Travel Arranger Announcement

No longer supported.

Company Announcement

Series of up to five custom messages that display one at a time below the menu bar. If the messages are too long to display completely, you can click a link in the message to see the entire message. You can pause to view a message. You can also display a custom message at the bottom of the page.

Note  This only appears if you have activated the new Home page.

Multi-destination

Custom message that appears above the From and To boxes when a user clicks the Multi-destination option button.

Note  The types of messages available on your system depends on the features you have enabled. By default these message areas are blank. There are no default messages. You can customize the messages for each language you have enabled.

To enter messages

  1. If applicable, click on the tab for appropriate language.
  2. Do one of the following:

    If you are using the

    Then

    New Home page

    1. In the Title box, type a title for your message.
    2. In the Post Date box, type the date you want to show for the message. This does not determine the date this appears in the system.

    3. In the Summary Message box, use the text editor to enter or edit a summary message that you want to show in the Message Carousel. You can include HTML and are limited to 300 characters.

      Note  If you do not create a summary, nothing will appear in the Message Carousel, and users will be unable to access the detailed text of your message.

    4. In the Detail Message box, use the text editor to enter or edit the complete message. This is optional. If you enter a more detailed message, the system shows a View Full Message link in the summary message.

    Old Home page

    Use the text editor to enter or edit your message.

  3. Click Save at the bottom of the text editor to save your messages in Site Administration. You do not have to click Save each time you enter a message in a different language.
