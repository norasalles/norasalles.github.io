JS Messaging API

You can use the JavaScript Messaging Application Program Interface in conjunction with the NUI (New User Interface) to display dynamic messages, banners, and popup messages based on defined data available on the following pages within the booking tool:

  • Home
  • Flight Availability
  • Air Pricing
  • Reservation Complete
  • Air Review & Modify Trip
  • Trip Details
  • Hotel Search
  • Hotel Availability
  • Hotel Details
  • Car Search
  • Car Availability
  • Rail Availability
  • Please Wait Messages

Note: To add dynamic messages, you must have a strong working knowledge of JavaScript.

The JS Messaging API uses the UIGlobals class and supports the following elements and methods:
