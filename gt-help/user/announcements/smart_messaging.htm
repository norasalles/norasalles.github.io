Smart Messaging

Path: Messaging > Smart Messaging

Smart Messaging is rules-based messaging that allows customers to sway purchasing behaviors using triggers found in the user’s profile and inputs from the Home page. Customers create messages based on what a user inputs and selects on the Home page, as well as data in the user’s profile.

The process starts in the Policy Administration application where you Add rules with notifications from the Shopping Request tab. Then in Site Administration you attach messages to your notifications.

For more information, click one of the following:

Rule Creation Tab

You can use the Rule Creation tab to see an overview of the steps to configure smart messaging.

Notification Configuration Tab

You can use the Notification Configuration tab to add messages to notifications and to make notifications active or inactive.

For more information, click one of the following:

  • Add a Message to a Notification

    Note  You can add up to 5 messages to a notification.

    1. Click the Add message link for the notification for which you want to add a message.
    2. Do one of the following:

      To

      Do

      Add an existing message

      1. Click the Add message link.

      2. Click Add.

        The message appears at the bottom of the list of messages for the notification.

      Create a new message

      1. For the notification to which you want to add the message, click the Add message link.

        The Add a Message dialog box appears.

      2. Click the Create a new message link.

        The Smart Messaging - Add a New Field page appears.

      3. In the Field Name box, type a unique field name. You can use up to 64 alphabetical and numerical characters and underscores.
      4. In the Message box, type the message you want to appear. You can use up to 2,000 characters. You cannot use a backslash (\) or equal sign (=).

      5. Click Save.

        The new message is created and is automatically added to the notification from step 1. The Notification Configuration tab appears.

    3. Click Save.
  • Change the Order of Messages Triggered by a Notification
    1. Drag the white bar for a message above or below an existing message in a notification.
    2. Click Save.
  • Remove a Message from a Notification

    1. Click the Remove link for the message you want to remove.

      A confirmation dialog box appears.

    2. Click OK.

      Note  This does not delete the message from the database. You can add the message again to this notification or another notification.

  • Make a Notification Active/Inactive

    Do one of the following

    To

    Do

    Make a notification active

    In the Inactive Notifications area, drag the icon for the message to the Active Notifications area.

    Make a notification inactive

    In the Active Notifications area, drag the icon for the message to the Inactive Notifications area.

Message Management Tab

You can use the Message Management tab to add, edit, or delete messages.

For more information, click one of the following:

  • Add a Message

    1. Click Add.

      The Smart Messaging - Add a New Field page appears.

    2. In the Field Name box, type a unique field name. You can use up to 64 alphabetical and numerical characters and underscores.
    3. In the Message box, type the message you want to appear. You can use up to 2,000 characters. You cannot use a backslash (\) or equal sign (=).
    4. Click Save.
  • Edit a Message
    1. In the Name column, click the name of the message you want to edit.
    2. Change the message information as needed.
    3. Click Save.
  • Delete a Message
    1. In the References Points table, select the check boxes for the messages you want to delete.

    2. Click Delete.

      A confirmation dialog box appears.

    3. Click OK.
  • Delete All Messages
    1. Select the check box in the header row.

    2. Click Delete.

      A confirmation dialog box appears.

    3. Click OK.
