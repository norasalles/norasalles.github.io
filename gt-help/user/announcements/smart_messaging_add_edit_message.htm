Smart Messaging - Add New Field/Edit Field

Path: Setup > Site Preferences > Smart Messaging > Add message

Path: Setup > Site Preferences > Smart Messaging > Edit message

You can use the Smart Messaging - Add New Field page to edit an existing message used for smart messaging.

You can use the Smart Messaging - Edit Field page to edit an existing message used for smart messaging.

For more information, see Smart Messaging.

To add a new message

  1. In the Smart Messaging page, do one of the following:

    From the

    Do

    Notification Configuration tab
    1. For the notification to which you want to add the message, click the Add message link.

    2. Click the Create a new message link.

      The Smart Messaging - Add a New Field page appears.

    Message Management tab

    Click Add.

    The Smart Messaging - Add a New Field page appears.
  2. In the Field Name box, type a unique field name. You can use up to 64 alphabetical and numerical characters and underscores.
  3. In the Message box, type the message you want to appear. You can use up to 2,000 characters. You cannot use a backslash (\) or equal sign (=).
  4. For each configured language, add the translated version of the field label and instructional text. Any language for which you do not add translated text appears in the site's default language.
  5. Click Save.

To edit a message

  1. In the Smart Messaging page, click the Message Management tab.

  2. In the Field Name column, click the name of the message you want to edit.

    The Smart Messaging - Edit Field page appears.

  3. In the Field Name box, type a unique field name. You can use up to 64 alphabetical and numerical characters and underscores.
  4. In the Message box, type the message you want to appear. You can use up to 2,000 characters. You cannot use a backslash (\) or equal sign (=).
  5. For each configured language, add the translated version of the field label and instructional text. Any language for which you do not add translated text appears in the site's default language.
  6. Click Save.
