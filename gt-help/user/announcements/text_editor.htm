Text Editor

You can use the text editor to create and format custom messages and announcements that appear on the booking tool. The text editor displays the background color of the page where the message is displayed. This editor uses tools similar to those used in MicrosoftÂ® Word and also uses the correct HTML formats. You can view the HTML code by clicking on the HTML button: .

From the Text Editor window, you can:

  • Add or remove text formatting
  • Edit text
  • Insert images, links, special characters, horizontal lines, and HTML

Remember to click Save on the bottom of the text editor to save your messages in Site Administration.

 

Note  Extra space now appears after each paragraph. If you want to break lines without showing extra space between each line, press SHIFT+ENTER at the end of each line.

Insert/edit link

Add a hyperlink:

  1. Select the text or image that will be the hotspot for the link.
  2. Click (Insert/Edit Link). The Insert/edit link dialog box appears.
  3. In the Link URL box, type the link address (URL).
  4. In the Target list, select whether to open the link in the same window (the booking tool) or a new window.
  5. If you want to give the page a title, in the Title box, type the title.
  6. To insert the hyperlink into the message box, click Insert.

Edit a hyperlink:

  1. Click on the text or image that contains the hyperlink.
  2. Click (Insert/Edit Link).
  3. Click Update.

Remove hyperlink
  1. Click on the text or image that contains the hyperlink.
  2. Click (Unlink).

Inserts a horizontal line
  1. Place the cursor where you want to insert a horizontal line.
  2. Click (Insert Horizontal Line).

Insert a custom character (symbol)
  1. Place the cursor where you want to insert a symbol.
  2. Click (Insert Special Character).
  3. From the dialog box, click the character you want to insert.

Insert HTML
  1. If necessary, copy existing HTML code from an external source.

  2. Place the cursor in the text editor, and click (Edit HTML Source). The HTML Source Editor window appears. Any existing messages appear as HTML source code.

  3. Paste the copied HTML code, or enter new HTML code.
  4. Click Update. The formatted text appears in the text editor.

Insert/edit an image

Add an image:

If you are adding an image to the summary of a message for the new Home page, the image can be no larger than 277 pixels  x 150 pixels. If you want text with the image, it is recommended that the image be 50 pixels  x 50 pixels.

  1. Place the cursor where you want to insert an image.
  2. Click (Insert/Edit Image).
  3. In the Image URL box, type the image address (URL or path name).
  4. Optional:
    1. If you want to show an image description when a person mouses over the image, in the Image Description box, type the image description.
    2. In the Alignment list, click one of the following to determine how you want to align the image within the text:
      • Left - The text wraps around and to the right of the image.
      • Right - Text wraps text around and to the left of the image.
    3. If you want the image dimensions to show differently in the message than they do in the original image, type the new length in pixels in the first Dimensions box, and type the new height in the second Dimensions box. If you want to use the actual size of the image, leave the Dimensions boxes blank.
    4. If you want to add a border to the image, in the Border box, type the number of pixels for the thickness of the borders you want.
    5. If you want to add space (padding) above and below the image between the image and text, in the Vertical Space box, type the number of pixels of space. Padding is useful when you are wrapping text around images.
    6. If you want to add space (padding) left and right of the image between the image and text, in the Horizontal Space box, type the number of pixels of space. Padding is useful when you are wrapping text around images.
  5. Click Insert to insert the image into the message box.

Edit an image:

  1. Select the image by clicking on the image in the message box.
  2. Click (Insert /Edit Image).
  3. Update the characteristics of the image as needed.
  4. Click Update.
