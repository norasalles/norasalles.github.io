+--UIGlobals

class UIGlobals

You do not have to create a new instance of the UIGlobals object. A global UIGlobals object is available in the booking process with the name of globals

Example:

globals.hotelVendorExist('HI')

globals.ondIndex()

boolean addButtonAlertMsg(<String> elementId, <String> message)

Add a JavaScript alert message to a "BUTTON" object that exists on a page.

This method works in the footer messages on the Air and Hotel availability pages.

Air Availability
  AvailabilityButton
  DirectGovernment:
    YCAAvailableButton
     CAAvailableButton
Lowfare Options Page
  AvailabilityButton
Hotel Availability and Room Rates Pages
  AvailabilityButton

Number button IDs from 1 to n, e.g., AvailabilityButton1, AvailabilityButton2, AvailabilityButton3, etc. (only enter the maximum number per page; the system resets the numbers on subsequent pages. e.g., use 1-25 if you allow 25 options per page)

Parameters:

elementId - The id of the specific button on the page.

message - The message you would like to display to the user.

boolean airVendorExist(<String> vendor)

String dropOffCity()

Get the car drop-off city the user is searching for.

Example:

globals.dropOffCity() == 'DFW'

Note: Use a three-letter airport code.

String getDepart(<int> which)

Get a specific depart airport or rail station code throughout the booking process.

One Way and Round Trip
  DFW to SFO
  DFW = depart(0)

Multi Destination
  DFW to JFK
  LGA to MCI
  DFW = depart(0)
  LGA = depart(1)

Example:

globals.getDepart(0)

Parameters:

which - The index of depart you would like to access (from 0 to 7).

String getDestination(<int> which)

Get a specific destination airport or rail station code throughout the booking process.

One Way and Round Trip
  DFW to SFO
  SFO = destination(0)

Multi Destination
  DFW to JFK
  LGA to MCI
  JFK = destination(0)
  MCI = destination(1)

Example:

globals.getDestination(0)

Parameters:

which - The index of destination you would like to access (from 0 to 7).
