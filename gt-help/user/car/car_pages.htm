Car Pages

Path: Car > Car Pages

You can use the Care Pages page to control screen configuration and default settings on the Car Search pages and in the user's profile.

Set up the Car Pages

  1. Select whether to Permit the user to select a car category (two or four door model) on the Car Search page.

    Best Practice: Disabled is the recommended and the default setting. (Important for international travelers.)

  2. Select whether to Permit the user to select a fuel type on the Car Search page. If selected, the user can request fuel preference for their car search. The options that are offered include: hybrid/green, diesel or Unspecified.

    Note  The default is Unspecified.

  3. Select whether to Permit the user to select from a list of special requests on the Car Search page. The list contains special items available with a rental car, such as handicap controls or a luggage rack. Most of these items are available for an additional fee.

    Best Practice:Enabled is the recommended and the default setting.

    Before disabling this option, carefully consider your users and whether you need to accommodate users with disabilities.

  4. Select whether to Permit the user to select 4-wheel drive/all-wheel drive cars on the Car Search page. If selected, the user can request drive/train preference for their car search. The options offered include: Yes/No for 4WD and AWD.
  5. Select whether to Permit users to book cars at off-airport locations.
  6. Click Save.
