Car Setup

Path: Car > Car Setup

These settings control how users perform a car search.

To set up the Car pages

  1. Select whether to Enable users to search for and reserve cars.

    If you disable this feature, be sure to turn off the setting that requires users to book cars.

  2. In the Car Search Options section, do the following:
    1. Select whether to Enable Express Booking.
    2. If you selected to Enable Express Booking, click one of the following in the Selection Criteria list:
      • Most preferred contracted vendor
      • Lowest price option
      • User's preference
      • Most preferred vendor by market preference

      Best Practice:Enable is the recommended and the default setting. (Important for international travelers.)

    3. If you enabled Mine Spec vehicles, select whether to Default the car search options for Mine Spec vehicles.
  3. Select how users can select a car type in the Car Types section as follows:

    If you want

    Then

    Users to select from a list of car types when searching for cars

    1. Click Allow users to select car size.
    2. Select the car types you want the system to show users. Users can select up to five of the car types shown to include in their search.

    The system to automatically search preselected car types

    Note  Users will not be able to select preferred car types in their search.

    1. Click Use a pre-defined list of car types for searches.
    2. Select up to five car types you want the system to search for all users on the site.
  4. In the Car Availability Page section, do the following:

    1. Select whether to Display Make/Model examples for each car rental option in the rental car search results. This information helps users by providing a vehicle example for a particular rental car class (e.g. Economy). However, users are not guaranteed that the displayed car make/model will be available from the car rental company.

      Best Practice:Enabled is the recommended and the default setting.

    2. Select whether to Display elite rental car options. If enabled, the system displays elite car options when available. Elite car models are versions of the same car size which have a bigger engine, higher trim, and some special features.

      Note  Elite models are only returned if the regular model of the same requested size is not available.

    3. Select whether to Preference cars with Air Conditioning when available. Locations outside the USA offer both air-conditioned and non-air-conditioned cars as standard. If you select this setting, the system makes a request that if cars with air conditioning are available, only cars with air conditioning are returned in the search results. If there are no cars with air conditioning, the system returns available cars without air conditioning. If you do not select this option, the system does not preference cars with air conditioning. Generally, only cars with air conditioning are shown. The system may experience problems with international itineraries if this field is enabled and the destination city does not have cars with air conditioning.

      Best Practice: Enabled is the recommended setting. The default setting is: disabled. Don't select this option if most rentals will be outside the United States.

    4. In the Limit the display of car choices list, click one of the following to determine what car vendors appear on the Car Availability pages:

      If you want to

      Then

      Only display contracted car vendors

      Note  Users will not be able to select preferred car types in their search.

      1. Click Yes, only display contracted car vendors.

      2. Select whether to use the direct sell GDS format for car vendors with at least one contract in the booking system. The options are: No, Yes, and One Way.

        Select Yes to use the direct sell GDS format for car vendors with at least one contract in the booking system. If there is a discount number in the contract, the system uses the corporate discount number entered for that contract to automatically obtain the contracted rental rates and display them to the user. If there is no corporate discount number direct sell still works, it just does not use a negotiated discount. This option requires the entry of a car rental contract for each vendor you want to display.

        If you select No, then the system displays the rates manually entered in the car contract to the user and only uses rates from the Host GDS when the user completes the booking. Direct Sell is usually more accurate than information stored in the Host GDS because Direct Sell goes directly to the information stored in the computer reservation systems of the car vendors themselves. Car rental information in the Host GDS is not always accurate or up to date.

        Best Practice: Recommended setting: Yes, if you have entered car contracts for each car vendor you want to display. The default setting is No.

      Display car vendors with contracted rates and all other available car vendors

      1. Click No, display all car rental vendors.
      2. Select whether to display two car rental options lists with contract vendors as the top list and non-contract vendors in the other?

        Best Practice: Recommended setting: Enabled. The default setting is: Disabled. If you select Disabled, contracted and non-contracted cars appear together in one list. 

      Display all contracted car vendors and any less expensive noncontracted car vendor options

      1. Click Yes, display contracted vendors & vendors with lower rates.

      2. Select whether to use the direct sell GDS format for car vendors with at least one contract in the booking system. The options are: No, Yes, and One Way.

        Select Yes to use the direct sell GDS format for car vendors with at least one contract in the booking system. If there is a discount number in the contract, the system uses the corporate discount number entered for that contract to automatically obtain the contracted rental rates and display them to the user. If there is no corporate discount number direct sell still works, it just does not use a negotiated discount. This option requires the entry of a car rental contract for each vendor you want to display.

        If you select No, then the system displays the rates manually entered in the car contract to the user and only uses rates from the Host GDS when the user completes the booking. Direct Sell is usually more accurate than information stored in the Host GDS because Direct Sell goes directly to the information stored in the computer reservation systems of the car vendors themselves. Car rental information in the Host GDS is not always accurate or up to date.

        Best Practice: Recommended setting: Yes, if you have entered car contracts for each car vendor you want to display. The default setting is No.

      Best Practice: Recommended setting: No, display all car rental vendors. The default setting is: Yes, only display contracted car vendors.

      If users cannot see all possible options, they may get frustrated and call the support desk. If they select out-of-policy options, the system notifies them when they try to make the booking. Also, if users cannot see all options, they may try to make reservations with the Host Travel Agency directly.

    5. Determine What rate information should be displayed on the Car Availability page: Both Daily/Weekly/Monthly and Total Rate or Total Rate only. This setting controls what total price is displayed on the Car Availability page only. The Weekly and Monthly Rates are always provided by the supplier system or GDS. If the Daily Rate is not available from supplier systems with a direct connect, it is calculated from the Total Rate divided by the number of days in the rental period.

      If you select Daily/Weekly/Monthly and Total Rate, both rate types are displayed (Daily, Weekly or Monthly, and Total Rate). If you select Total Rate only, only this rate is displayed.

      Notes   

      • We have added Total Rate as a display option for the Car Booking process. Total Rate is the estimated total cost for the length of the rental (including base rental rate, tax and surcharges). Items such as optional coverage, child safety seats, and gas are not included.
      • We only support rental agencies that have a direct connection with the booking tool, including Avis and Budget. Support for GDS rental agencies will be added in a subsequent release.
  5. Click Save.
