Setup Hotel Details/Rates

Path: Hotel > Hotel Details/Rates | Setup tab

You can use this page to setup the types of rates to display as well as to include mapping and directions.

To setup the Hotel Details/Rates page

  1. If necessary, click the Setup tab.

  2. In the Select room rates to display list, click the type of rates to display to the traveler.

    Note  This setting is only applicable to Apollo, Sabre and Worldspan.

  3. In the Enter keywords to reference and identify negotiated rates box, type keywords for a specific room rate that you want the system to designate as a company negotiated room rate. Based on the keyword matches to the rate description returned from the GDS, the system can automatically identify the negotiated rate and assign the rate label.

    Note  You can enter multiple keywords separated by commas. Keywords are not case sensitive. You can enter up to 500 characters.

  4. In the Specify the text for the negotiated rate label box, type a custom label for negotiated rates. Use text only, no HTML, and a maximum of 25 characters.
  5. To control whether specific room rates are displayed to the traveler, enter Room Rate Filters as follows:
    1. In the Enter keywords for automatic display box, type keywords the system uses to control whether specific room rates are displayed to the traveler. Based on keyword matches to the rate description returned from the GDS, they system can automatically display room rates that include the keywords listed.
    2. In the Enter keywords for automatic exclusion box, type keywords the system uses to control whether specific room rates are not displayed to the traveler. Based on keyword matches to the rate description returned from the GDS, they system can automatically exclude room rates that include keywords listed.

    Note  The automatic display and automatic exclusion lists work together. For example, if you choose to automatically display rates including the keyword "deluxe" but automatically exclude rates including the keyword "AAA" the following room rate types will be displayed to the user:

    • Any rate including the keyword "deluxe" but not the keyword "AAA." For example, deluxe room rates but not deluxe AAA room rates.
    • Any rate including any keyword other than "AAA." For example, standard room rates but not standard AAA room rates.
  6. Click Save.

Discounts

Path: Hotel > Hotel Details/Rates | Discounts tab

You can use this page to enter negotiated chain discounts.

To setup hotel discounts

  1. Click the Discounts tab.
  2. Enter corporate hotel discount reference codes, separated by a number sign (#), comma (,), hyphen (-), or a space ( ). If your company has access to discounted hotel pricing via the GDS, enter the special 3-character alpha/numeric codes used to reference these rates. You can enter up to four negotiated rate codes. Insert the proper separator for each GDS between each code. For example,

    Amadeus/Apollo/Galileo

    AMX#THR#MOT

    Worldspan

    AMX-THR-MOT

    Sabre (use a number sign, comma, or space)

    XYZ#MOT#AMX#THR

  3. Select Permit rate retries on single property queries, if the corporate hotel discount reference code is not valid for the hotel when requesting rates and you want the system to retry the request without the discount code.
  4. Enter up to 25 Negotiated Hotel Chain Discounts: Enter the Hotel Chain’s two-letter hotel code. Enter the Corporate Discount # (CD)associated with this hotel chain.
  5. Click Save.

Messaging

Path: Hotel > Hotel Details/Rates | Messaging tab

You can use this page to display a custom message on the top and bottom of the following hotel pages:

Type

Appears

Hotel Search

On the hotel search page

Select Hotel

On the page where you select a hotel

Hotel Details/Rates

On the page where you select a hotel rate

Note  The types of messages available on your system depends on the features you have enabled. By default these message areas are blank. There are no default messages. You can customize the messages for each language you have enabled.

To enter messages

  1. Select the Messaging tab.
  2. If applicable, click on the tab for appropriate language.
  3. Do one of the following:

    To add a message

    Do

    That is dependent on one or more conditions and/or requires JavaScript

    Enter information in the Top Dynamic Message box or Bottom Dynamic Message box. For more information, see the Dynamic Text Editor topic or the Velocity Student Guide.

    That always appear regardless of travel choices and does not require JavaScript

    Enter information in the Top Message box or Bottom Message box. For more information, see the Text Editor topic.

  4. To save your messages in Site Administration, at the bottom of the text editor, click Save. (You do not have to click Save each time you enter a message in a different language.)
