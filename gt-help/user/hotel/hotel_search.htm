Hotel Setup

Path: Hotel > Hotel Search | Setup tab

You can use this page to enable hotel searches, define search options, and display a preferred hotel chain list.

To enable hotel searches, set up options, and display chain list

  1. If necessary, select the Setup tab.

  2. To enable users to search for hotels, select the Enable hotel search check box.

    Notes   

    • If you do not select this option (i.e., you disable this feature), the Hotel check box is removed from the Home page.
    • If you want to disable this feature, please ensure your Travel Policy does not require a user to book hotels.
    • If you have Hold Trips enabled and the user already has a trip on hold that contains a hotel segment – and you disable the hotel search feature – when the user elects to purchase the trip, the hotel segment is purchased.

  3. To have the Hotel Search page default to the Negotiated Properties search and the results page has the Preferred Hotel Properties listed at the bottom of the page, select the Make Preferred Hotel Properties the default search type check box.

    If you clear this check box, the default hotel search is Near an Airport.

    Note  The Negotiated Properties default search only applies to cities where a preferred hotel has been loaded via a preferred hotel database.

  4. To make it easier for a traveler to stay at a downtown hotel one night and at an airport hotel the next night, for example, select the Allow users to book multiple hotels in a city check box. When selected, the user can add another hotel booking from the Trip review and checkout page.
  5. To add an option to the Hotel Search page to select the number of rooms to book, select the Allow users to book multiple hotel rooms in a hotel check box. The options are 1, 2, or 3 rooms. All rooms are held in the user’s name and the user’s credit card is used to guarantee or make any required deposits to the hotel. If you Allow users to request multiple hotel room types in a hotel, the system asks the user to select a room type for each room from the Hotel Details page.
  6. The Allow users to search for hotel properties by distance check box is no longer applicable.
  7. To allow the user to choose up to three common special hotel requests (such as non-smoking room, king bed, etc.) when searching for a hotel, select the Allow users to select special hotel requests check box.
  8. In the Distance used for Preferred Hotel Property searches list, click the maximum distance you want to use for searching preferred hotel properties. The system displays a note next to the Distance box on the Hotel Search page to advise users of the distance used for the display of preferred hotel properties.
  9. In the Distance used for Non-Preferred Hotel Property searches (includes Preferred Chains) list, click the maximum distance you want to use for searching non-preferred hotel properties and preferred chains. The system displays a note next to the Distance box on the Hotel Search page to advise users of the distance used for the display of preferred hotel properties.
  10. Select whether to Display a preferred hotel chain list on the Hotel Search and Profile pages. If you select this check box, do the following:
    1. Select whether to Use default hotel chain list or Use custom hotel chain list.
    2. If you selected the Use custom hotel chain list option button, hold down the Ctrl key while you click other items that you want to select.

    Note  This setup has been temporarily discontinued. However, user will still find the hotel chain list in the user profile > hotel preferences, if configured but user selection of hotel chain preferences in the user profile will not affect the hotel availability. Hence, it is recommended not to configure this feature.

  11. Select whether to Display a custom message above the preferred hotel chain list. If you select this option, enter the Title Text and the Message Text.
  12. Enter the custom message for the Hotel Availability page. This message is appended to the top message on the Select a Hotel page.
  13. Click Save.

Messaging

Path: Hotel > Hotel Search | Messaging tab

You can display a custom message on the top and bottom of the following hotel pages:

Type

Appears

Hotel Search

On the hotel search page

Select Hotel

On the page where you select a hotel

Hotel Details/Rates

On the page where you select a hotel rate

Note  The types of messages available on your system depends on the features you have enabled. By default these message areas are blank. There are no default messages. You can customize the messages for each language you have enabled.

To enter messages

  1. Select the Messaging tab.
  2. If applicable, click on the tab for appropriate language.
  3. Do one of the following:

    To add a message

    Do

    That is dependent on one or more conditions and/or requires JavaScript

    Enter information in the Top Dynamic Message box or Bottom Dynamic Message box. For more information, see the Dynamic Text Editor topic or the Velocity Student Guide.

    That always appear regardless of travel choices and does not require JavaScript

    Enter information in the Top Message box or Bottom Message box. For more information, see the Text Editor topic.

  4. To save your messages in Site Administration, at the bottom of the text editor, click Save. (You do not have to click Save each time you enter a message in a different language.)
