Reference Points

 Path: Hotel > Reference Points

Users have the option to search hotels using Reference Points (company locations). Users can click the Find your company locations link to show a list of company locations and select one for a search point. You can also show reference points on the Hotel Search page in map view using the Show company locations check box.

For more information, click one of the following:

Set the Reference Point Search Distance

  1. Select the Reference Point radius (relative to the search origin), which is used as the radius when searching for a hotel by a Reference Point.

    Note  We recommend you set this option to "within 50 miles (or 80 KM)" to ensure the maximum number of hotels display.

  2. Click Save.

Add a Reference Point

  1. Click Add.

    The Add Reference Point dialog box appears.

  2. In the Name box, type a name for the reference point. The maximum number of characters is 100.

    Note  The Reference Point name can only begin with an alpha or numeric character. You cannot use special characters or spaces as the first character in the name field.

  3. In the Country list, click the country in which the reference point is located.
  4. In the Street box, type the street on which the reference point is located. The maximum number of characters is 50.
  5. In the City box, type the name of the city in which the reference point is located. The maximum number of characters is 30.
  6. Optional: In the State/Province list, click the state/province in which the reference point is located.

  7. Optional: In the Zip/Postal Code box, type the zip code/postal code in which the reference point is located.The maximum number of characters is 10.

    Note  A maximum of 50 reference points are shown per page.

  8. Click Save.

Edit a Reference Point

  1. In the References Points table, in the Name column, click the name of the reference point you want to edit.
  2. Change the reference point information as needed.
  3. Click Save.

Delete a Reference Point

  1. In the References Points table, select the check boxes for the reference points you want to delete.

  2. Click Delete.

    A confirmation dialog box appears.

  3. Click OK.

Delete All Reference Points

  1. Select the check box in the header row.

  2. Click Delete.

    A confirmation dialog box appears.

  3. Click OK.

Sort and Filter Reference Points

You can sort Reference Points by name (default sort), city, state, or country. The secondary sorts are as follows:
