Select Hotel

Path: Hotel > Select Hotel

You can use the Select Hotel page to configure the following:

  • Hotel Setup
  • Hotel Messaging

For more information, click one of the following:

Setup

Path: Hotel > Select Hotel | Setup tab

You can use this tab to control the display of hotels on the Select a Hotel page.

To set up the Select a Hotel page

  1. Select whether to Enable map view, mapping and directions. This displays the Map View button on the Hotel Search and Select a Hotel page. It also displays the Maps & Directions link on the Select a Hotel, Hotel Details, Review/Modify Trip, and Reservation Complete pages.

    If you enable the above setting, choose whether to Make map view the default display on the Hotel Search page. If disabled, List View is the default display.

  2. Select whether to Apply user’s chain preferences to Preferred Hotel Properties. This filters the Preferred Hotel Properties list according to the user's preferred chains.  The page lists only the Preferred Hotel Properties that meet the user’s requirement.

  3. Select whether to Display approximate rates for Hotel Properties.  If enabled, then select which properties the guide rates are to display.  The options are Non-Preferred Hotel Properties, Preferred Hotel Properties and Both.

    Note  If your site uses Sabre, the system displays guide rates for the first 40 hotels. If you are using Amadeus, Apollo, Galileo, or Worldspan, you have the option to select the Maximum number of properties to display. You may choose from 3, 6, 9, 12, 15, 18, 21, and 24. .Sites using Amadeus, Apollo, Galileo, or Worldspan may have an impact to response time as the number of properties for which guide rates are displayed increases.

  4. If you select Preferred Hotel Properties or Both in step 3, select whether to Display rates for Preferred Hotel Properties from the Preferred Properties database or select whether to Display approximate (guide) rates for Preferred Hotel Properties from the GDS.
  5. Enter descriptive text for the Preferred Hotel Property graphic. When you enter text, the system adds an additional box, which further defines the Negotiated Property graphic.
  6. If you have a Government site, you can Display only fire-safe hotels approved by the government. The system adds a note to indicate that all hotels are approved by the U.S. Government as fire-safe properties. This setting is most useful for government employees who are required to stay at government approved fire-safe hotels when available.

  7. If you have a Government site, you can Display government per diems. The system adds the United States government per diem information on the Select a Hotel and the Hotel Details pages.

    Note  A per diem is a daily allowance for hotel expenses.  The United States government published per diems for cities and counties across the United States.

Messaging

Path: Hotel > Select Hotel | Messaging tab

You can display a custom message on the top and bottom of the Select Hotel page.

Note  The types of messages available on your system depends on the features you have enabled. By default these message areas are blank. There are no default messages. You can customize the messages for each language you have enabled.

To enter messages

  1. If applicable, click on the tab for appropriate language.
  2. Do one of the following:

    To add a message

    Do

    That is dependent on one or more conditions and/or requires JavaScript

    Enter information in the Top Dynamic Message box or Bottom Dynamic Message box. For more information, see the Dynamic Text Editor topic or the Velocity Student Guide.

    That always appear regardless of travel choices and does not require JavaScript

    Enter information in the Top Message box or Bottom Message box. For more information, see the Text Editor topic.

  3. To save your messages in Site Administration, at the bottom of the text editor, click Save. (You do not have to click Save each time you enter a message in a different language.)
