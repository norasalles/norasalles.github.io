Traveler Reviews

Path: Hotel > Traveler Reviews

You can use the Traveler Reviews page to enable and configure traveler reviews for the site. If you do not want to see public or leisure hotel reviews, GetThere’s Corporate Traveler Reviews give customers the option to capture, display, and action hotel reviews submitted only by their travelers. You can see reviews from all GetThere users or just users of your subsite or supersite.

You can configure the system to send post-trip e-mails to solicit review submissions. You can also use third-party tools to create scheduled reports and e-mail the reports to the Travel Managers for all submitted reviews.

For more information, click one of the following:

Configure Traveler Reviews settings

  1. Click the Setup tab.
  2. If you are a supersite administrator and want to allow subsite administrators to view, configure, and enable Traveler Reviews for their subsite, select the Allow Site Administrators to enable Hotel Traveler Reviews check box.
  3. To enable Traveler Reviews for users of this site, select the Enable Hotel Traveler Reviews check box.
  4. In the Review Types area, select which reviews to show travelers as follows:

    To show users reviews from

    Do

    All GetThere users

    Click Display all reviews.

    Only users in my supersite and all the subsites that are part of my supersite

    Click Display reviews from my supersite.

    Only users in this subsite

    Click Display reviews from my subsite only.

  5. Complete the following steps in the Post-trip Hotel Review E-mail Message area to set up e-mail to be sent to the traveler two days after a hotel stay has been completed:
    1. In the From box, type the name or e-mail you want to show as the sender of the e-mail.
    2. If you want to use a different e-mail address for replies than what you typed in the From box, type that e-mail address in the Reply to box.
    3. Choose the e-mail message text as follows:

      To

      Do

      Use the default e-mail message

      Click Use default message.

      Use a custom e-mail message

      1. Click Use custom message.
      2. In the Subject box, type the subject for the e-mail.
      3. In the Enter the message you want to send box, type the text for your e-mail message. Ensure the following requirements are met:
        • You must include NAME, HOTEL, and CITY so the system can dynamically replace them. They must be in English, even if the entire message is in a different language.
        • To create a line break, you must use <br> (with the angular brackets).
        • You must include [add link here] to indicate where the link to write the review will appear in the message.
  6. After you make all necessary changes, click Save.

Create a Traveler Reviews campaign

You can create a Traveler Review campaign to provide a base of hotel reviews. You can select the sites to include and the date range of past hotel stays for which to send e-mails requesting review information.

Note  After you create a Traveler Review campaign, you cannot change or delete the campaign.

  1. Click the Reviews Campaigns tab.
  2. Click Create New Campaign. The New Reviews Campaign dialog box appears.
  3. In the Select the subsite(s) list, click the subsites on which you want to request traveler reviews.

  4. In the Start Date and End Date boxes, enter the date range for hotel stays for which you want to request traveler reviews. All hotel stays with a checkout date in the specified date range are included in the campaign.

    Note  The start and end dates should be any time in the past up to the current day.

  5. Complete the following steps to set up e-mail to be sent to the traveler two days after a hotel stay has been completed:
    1. In the From box, type from whom the e-mail is coming.
    2. If you want to use a different e-mail address for replies than what you typed in the From box, type that e-mail address in the Reply to box.
    3. Choose the e-mail message text as follows:

      To

      Do

      Use the default e-mail message

      Click Use default message.

      Use a custom e-mail message

      1. Click Use custom message.
      2. In the Subject box, type the subject for the e-mail.
      3. In the Enter the message you want to send box, type the text for your e-mail message. Ensure the following requirements are met:
        • You must include NAME, HOTEL, and CITY so the system can dynamically replace them. They must be in English even if the entire message is in a different language.
        • To create a line break, you must use <br> (with the angular brackets).
        • You must include the [add link here] to indicate where the link to write the review will appear in the message.
  6. After you make all necessary changes, click Save.

View the status of a Traveler Review campaign

  1. Click the Reviews Campaigns tab.
  2. Review the following information for each Traveler Review campaign:
    • Campaign - Name of the campaign. You can click the link to see detailed information for the campaign.
    • Created By - E-mail address entered in the From box when creating the campaign.
    • Subsites - Number of subsites included in the campaign. You can click the link to see which sites were included and other information regarding the campaign.
    • Date Range - Date range of the hotel stays targeted by the campaign. All hotel stays with a checkout date in the specified date range are included in the campaign.
    • Status - Status and start date of the campaign. Status can include the following:
      • New - The campaign is in progress.
      • Failed - An error occurred during the campaign attempt. E-mails were not sent.
      • Number of e-mails sent in a successful campaign and the date they were sent.
