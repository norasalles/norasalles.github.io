No Reason Codes

Path: Policy > Corporate Travel Policy > No Reason Codes

If the Travel Policy system behavior is configured to require that users select a reason for non-compliant travel, you can use the No Reason Codes page to setup those reasons. No Reason codes are required when no car or hotel is booked.

Each reason can have an associated code that can be captured and formatted with a PNR editor or Modify PNR Editor string and documented in the PNR. For more information, see PNR Editor or Modify PNR Editor.

Note  You will define the reason codes. These codes can contain a maximum of three characters.

You can set up the following reason codes:

  • No Car Reason Codes
  • No Hotel Reason Codes
