Reason Codes

Path: Policy > Corporate Travel Policy > Reason Codes

The system's behavior can be configured to display a list of pre-defined reasons to the user when they create a non-policy compliant itinerary. The traveler is required to select from the list as a method of explaining why they are not following your guidelines.

Each reason can have an associated code that can be captured and formatted with a PNR editor or Modify PNR Editor string and documented in the PNR. For more information, see PNR Editor or Modify PNR Editor.

Note  You will define the reason codes and descriptions. These codes can contain a maximum of three characters. The reason descriptions can contain a maximum of 65 characters.

You can set up the following reason codes:

  • Air reason codes
  • Rail reason codes
  • Car reason codes
  • Hotel reason codes
