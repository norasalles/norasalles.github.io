Air & Car Setup

Path: Policy > Corporate Vendors Settings > Air & Car Setup

If your site has not previously used Market Based Preferences, you need to turn on this option within the Air & Car Setup page in step 1 below.

Air Availability Display

  1. Select What kind of request for airline availability should the system send to the host GDS as follows:

    • Disable Market Based Preferences – Disable the display of Market Based Preferences.
    • Only request preferred airlines from the GDS – Only display preferred airlines unless preferred airlines do not exist.

    • Request all airlines from the GDS – Display all airlines.

      Note  You must use this option if you enable Air Connect.

  2. Select the Maximum Air Market Based Preferences from the list.  The default is 500, the maximum is 2,000, and the minimum is 0.

    Notes   

    • This toggle limits how many MBPs can be created. The more MBPs you have, the greater the impact to response time.
    • When loading MBPs through the upload tool, if this is set to 200 and you are loading 201 MBPs then you may receive an error.
  3. To place preferred airlines at the top of the availability display for the user, select the Group preferred airlines and display as the first options on the airline availability page check box. For government sites, you must select this check box to show contracted fares (CPP) above commercial fairs. If you do not select this check box, the availability follows the user’s sorting preference.

  4. To enable if flight options that contain a mixture of preferred and non-preferred airlines to display as preferred for all legs, select the Display the entire flight option as Preferred if only one segment uses a preferred airline check box.

    If you do not select this check box or the next check box, the system does one of the following:

    • When determining preferred based on market preferences, by default the system considers the first flight in the flight option.
    • When determining preferred based on star ranking, by default the system considers the best flight in a flight option when determining preferred or non-preferred.

  5. To enable if flight options that contain a mixture of preferred and non-preferred airlines to display as demoted for all legs, select the Display the entire flight option as Demoted if only one segment uses a demoted airlinecheck box.

    If you do not select this check box or the next check box, the system does one of the following:

    • When determining preferred based on market preferences, by default the system considers the first flight in the flight option.
    • When determining preferred based on star ranking, by default the system considers the best flight in a flight option when determining preferred or non-preferred.

    Note  If you select the Display the entire flight option as Preferred if only one segment uses a preferred airline check box and the Display the entire flight option as Demoted if only one segment uses a demoted airlinecheck box, the option to show as Demoted takes precedence.

  6. To place a header above preferred airlines, select the Should preferred airlines be identified uniquely check box

    If not selected, no header displays.

  7. Select the type of Preferred Airline Header Message that is displayed at the top of the airline availability page as follows:

    • Default – A default Header is displayed at the top of the air availability page showing the following:

      Company Preferred airlines are indicated with the following icon:

       

    • Custom – You can customize the header message displayed to the user, enter your custom message.
  8. Select If excluded/demoted airlines are displayed to users, should they be demoted on the schedule search type, the price search type, or both. When you setup your MBPs for air, you have the choice to display or not display your demoted airlines.  If you choose to display your demoted carriers you can choose from the following:
    • Price and time – Demote airlines when searching by price or by time.
    • Price only – Demote airlines only when searching by price.
    • Time only – Demote airlines only when searching by time.

    Demoting pushes the demoted airlines to the bottom of the list of options, the bottom of the matrix, or the bottom of the price carrier list, depending on your selections in steps 8, 9, and 10.

  9. Select Should airlines be demoted in the matrix only, the list of flights only, or both. If you choose to demote airlines for the price search type in step 7, then airlines in the matrix are placed in a separate tab called Other Airlines.  Airlines in the list would be pushed to the bottom of the airline list below the matrix.  Choose from the following:
    • Both – Demote airlines in the matrix and the list of flights.
    • Matrix only – Demote airlines in the matrix only.
    • List only – Demote airlines in the list of flights only.
  10. If you want prices to show in the matrix for your demoted airlines, select the Should prices for demoted airlines be displayed to users in the matrix check box. Prices do show once the user chooses the specific carrier.
  11. If you want prices to show in the list of airlines displayed below the matrix, select the Should prices for demoted airlines be displayed to users in the list of flights check box. Prices do show once the user clicks the Select button beside the airline.
  12. Select whether When demoting in the list of flights, should demoted options be moved to the end of the entire list or to the end of the sort category as follows:
    • End of the entire list – The system moves demoted airlines to the bottom of the entire list of carriers regardless of price, time, etc.
    • End of the sort category – The system moves demoted airlines to the bottom of each sort category.

Blocked Airlines

  1. Type two-letters airline codes to block airlines.

    You can block up to 20 airlines from displaying anytime in the booking process, including global markets and even if your site is using SuperPNR functionality.  Blocked Airlines takes precedence over market based preferences.

    Note  Using the fields for the 20 Blocked Airlines does not affect the call to the GDS.  The blocked airlines are filtered out from the results and are not displayed.

Sort Order for Shop by Price and Low Fare Only

  1. Select whether to Sort flights based on Flight Type, Preferred Carriers, Code Share, and/or Price on the Search by Price path with Low Fare Options.

    If you select this check box, select up to four sort criteria and priorities as follows:

    Note  It is not mandatory for all drop-downs to be selected. However, if select this check box, you must select at least one sort option.

    To sort by

    Do

    Flight Type – Whether the flight option is non-stop, direct, 1-connection or any other

    Click one of the following:

    • Flight Type - Non-stop > Other – Where all non-stop flights will be pushed to the top and then other flight options will display
    • Flight Type - Non-stop > Direct > 1-connection > Other – Where all non-stop flights are pushed to the top, then direct flights display, then 1-connections display and then other flight options display
    • Flight Type - Non-stop > Direct > Other – Where all non-stop flights are pushed to the top, then direct flights display and then other flight options display
    • Flight Type - Non-stop & Direct > 1-connection > Other – Where all non-stop and direct flights together display at the top (no preference given to either between them), then 1-connections display and then other flight options display
    • Flight Type - Non-stop & Direct > Other – Where all non-stop and direct options together display at the top (no preference given to either between them) and then other flight options display
    • Flight Type - Non-stop > Direct & 1-connection > Other – Where all non-stop flights are pushed to the stop, then direct flights and 1-connections display (no preference given to either between them) and then other flight options display.

    Preferred Airline – Whether the flight option is of an airline which has been set as preferred in the market of search at Policy > Corporate Vendor Markets > Air Preferences. When you select this option:

    • This puts the flight options from preferred airlines above flight options from non-preferred airlines.
    • This arranges all flights from preferred airlines from most preferred airline to least preferred airline.

    Click one of the following:

    • Preferred Airline - Preferred (Based on Market Preferences) > Non-preferred – Where all the flight options from preferred airlines are put above flight options from non-preferred airlines. In addition, the system arranges all flight options from the preferred airlines in the order from the most preferred airline to the least preferred airline.
    • Preferred Airline - Preferred (Based on Star Ranking) > Non-preferred – Where all the flight options from preferred airlines are put above flight options from non-preferred airlines. In addition, the system arranges all preferred flight options indicated to display as 3 stars above those indicated to display 2 stars, and the 2 stars above those indicated to display 1 star. There is no ranking done based on the order of preference (most preferred to least preferred airline).

    Note  When determining preferred based on market preferences, by default the system considers the first flight in the flight option. When determining preferred based on star ranking, by default the system considers the best flight in a flight option when determining preferred or non-preferred. These options can be overridden by the Display the entire flight option as Preferred if only one segment uses a preferred airline check box and the Display the entire flight option as Demoted if only one segment uses a demoted airlinecheck box. If you select both of these check boxes, the option to show as Demoted takes precedence.

    Codeshare – Put non-codeshare flight options above code-share flight options

    Click Codeshare - Non-codeshare > Codeshare.

    Price – Sort flight options in the order of price, from the cheapest to more expensive

    Click Price - Lowest > Highest.

Car Rental Availability Display

  1. Select What kind of request for car rental availability should the system send to the host GDS from the list as follows:
    • Disable Market Based Preferences – Disable the display of Market Based Preferences.
    • Only request preferred airlines from the GDS – Only display preferred car companies unless preferred car companies do not exist.

    • Request all airlines from the GDS – Display all car companies.

  2. Select the Maximum Car Market Based Prefs from the list. The default is 500, the maximum is 2000, and the minimum is 0.

    Notes   

    • This toggle limits how many MBPs can be created. The more MBPs you have, the greater the impact to response time.
    • When loading MBPs through the upload tool, if this is set to 200 and you are loading 201 MBPs then you may receive an error.
  3. To place preferred car rentals at the top of the availability display for the user, select Group preferred car rental companies and display as the first options on the car rental availability page.  If you do not select this check box, the availability follows the user’s sorting preference.

  4. If you want to uniquely identify preferred car rental companies, select the Should preferred car rental companies be identified uniquely check box. A default Header is displayed at the top of the air availability page showing the following:

    Company Preferred are indicated with the following icons:

     

     

    If not selected, no header appears.

Preferred Airline and Car Rental Identification

  1. Select the preferred icon to be used for each preferred vendor, both air and car availability.  You can select an icon for up to ten preferred airlines and three preferred car rental companies.

    Note  At this moment, preferred carriers from 4 to 10 do not go through travel policy.

  2. Click Save.
