Hotel Setup

Path: Policy > Corporate Vendor Settings > Hotel Setup

You can use the Hotel Setup page to control the display of hotels.

Configure the Hotel Availability List

  1. Select one of the following options for displaying the hotel availability list.
    • Display all hotel - This displays the combinations of preferred properties, preferred chains and non preferred hotel.  
      • Choose one of the Display Order options.
    • Only display preferred properties and chains/channels – This only allows the display of preferred properties and preferred chains/channels. The non-preferred properties are removed.
      1. Choose one of the Display Order options.
      2. If preferred properties or chains/channels are not available, then select whether to Display non-preferred hotels.
    • Only display preferred properties – This only allows the system to display preferred properties.
      • Choose one of the options for what to show if no preferred properties exist.

    Note  The examples listed to the right of the screen shows the display orders of the above options.

  2. In the Secondary Display Order area, click one of the following:
    • Sort by Distance.
    • Sort by Price.
    • Sort by Preferred Chains.

    Secondary Display Order only applies to the first set of results returned in hotel availability. Any additional hotels viewed after the initial display are shown in system default order or in the order you selected in the Sort by list on the Select a hotel page.

    Secondary sort options only apply if you have preferred hotels or have set market preferences.

  3. Identify Promoted Hotels as follows:
    1. If you want preferred properties and hotel chains to be identified uniquely, select the Uniquely identify promoted hotels check box.
    2. Select whether to Display a header message for uniquely identified promoted hotels in the flight availability list.
      • If you select this check box, use the text editor to enter or edit the message to display above the hotel availability list.
    3. For Hotel Properties and Hotel Chain/Channels, do one of the following:
      • If you want the system to display a label for some or all promotion groups, type a label. You can use a maximum of 25 characters.
      • If you want the system to show the check mark icon () for a promotion group, select the Show icon check box.
  4. Click Save.
