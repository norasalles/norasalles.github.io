Preferred Hotel Upload Tool

Path: Policy > Corporate Vendors Settings > Preferred Hotel Upload Tool

You can use the Preferred Hotel Upload Tool to quickly load all your preferred hotels in the database.

When you have negotiated with specific hotels in different cities, you want to drive your Travelers towards these specific hotels to meet any quotas that may be part of the negotiations. Uploading the hotels through the Upload Tool places the preferred hotels on the Hotel Search page so they are the first hotels the Traveler sees (This needs to be setup in site administration for the search page to default to preferred hotels). It also marks preferred properties with preference indicators (icons) on the availability pages, if so configured on the Hotel Setup page. Depending on how restrictive you want to be with your Travelers, you can configure the site in a way that only preferred properties can be booked and nothing else (where available) or use travel policy to call them out of policy.

Caution  If your site utilizes GetThere’s SuperSite functionality or uses a different GDS on different SubSites, you need to upload a separate file for each GDS.

For more information, click one of the following:

Upload preferred hotels

  1. Download a template into which you can add your preferred hotel information as follows: 
    1. If necessary, click the Hotel Upload tab.
    2. Click the Hotel Upload Template link, and save the spreadsheet.
  2. Complete the spreadsheet as follows:

    1. Open the Excel spreadsheet.  At the bottom of the Excel spreadsheet there are a number of tabs.  The first tab is the Revision History.  Any changes to the upload process, additions or deletions of field names, is added to the revision history so you are informed of changes to the template.

      Note  It is recommended that you download an updated template monthly to ensure you have the latest database fields.

    2. Click the Key tab.  The Key is used to tell you what the name of the database fields are that you can update for hotels; and what characters, length of characters and specific data can be entered in each field.

    3. Click the Template tab.  The template gives you an example to follow.  You can use the template as shown or delete/add any fields you need or do not need.  You can also create a new template of your own without using this template.

    4. Complete the information on the excel spreadsheet for each hotel to be loaded.  Remember the Key to aid you in knowing field labels for each field you want to load.

      Notes   

      • To avoid errors, remove any columns with data field names that you are not using.  This also increases the speed of the upload.
      • Your file must include certain fields for certain types of properties.  Hotels that are available for booking in a GDS have different required fields than those that cannot be booked in the GDS.  Also, fields may require different entries depending on your GDS.  Remember to refer to the Key for specifications for each database field.
      • If you need to find a value for the supplier_property_id field in the spreadsheet, click the Hotel Connect Property Code link.
    5. Save the file as follows:
      1. Click File, and then click Save As.
      2. In the Save as type list, click CSV (Comma delimited) (*.csv).
      3. In the File name box, type an appropriate name, considering the following:
        • Spaces are not permitted in the upload or file name as this causes the upload to fail.
        • While not required, it is best practice to include the file type, the function you are uploading, the site name, the SubSite name (if applicable), and the date in the file name.  For example, hotel_add_abccorp_europe_011207.

      4. Click Save.

        A confirmation dialog box appears.

      5. Click OK.

        A confirmation dialog box appears.

      6. Click Yes.
  3. Select your csv file as follows: 
    1. Click Browse.
    2. Select your csv file, and click Open.

  4. In the SubSite(s) list, click the subsites for which you want to load the new preferred properties. To select multiple items, hold down the Ctrl key while you click other items that you want to select.

    Caution  This list of preferred properties replaces the existing list. Be sure your new csv file has the existing preferred properties you want to keep.

  5. If you want to load this list of preferred properties on your supersite, select the Load to Home Site check box.
  6. Click Save.

View file upload history

You can view information about the files you have uploaded or those waiting to be uploaded.

  1. Click the File Upload History tab.
  2. View the following information as needed:
    • File Name - You can view the name of the file or click the link to open and/or save a copy of the file.
    • Scheduled - The date the file was scheduled to upload.
    • Status - Status of the file upload as follows:
      • In queue - The file pending processing during off-peak hours. A status e-mail will be sent to the Administrator e-mail address when complete.
      • Completed - The file processed and loaded to sites.
      • Started - The file is being processed and may not be deleted. A status e-mail will be sent to the Administrator e-mail address when complete.
      • Failed - The file could not be processed. Please resubmit for processing.
      • Purged - The file is greater than 13 months old and has been removed from our system. If this is the most current file, hotel properties display to the user.
    • Loaded By - E-mail address of the user who loaded the file.
    • Multi-Site - Yes or No to indicate if the file was uploaded to multiple subsites. If Yes, you can click the link to see the sites to which the files were uploaded.
    • Log - You can click the link to see the log of information regarding the upload.

Remove a file from the upload queue

If a file you set to upload has not started uploading, you can remove it from the upload queue to prevent it from uploading.

  1. Click the File Upload History tab.

    Check boxes appear in the first column for all files that have not started uploading.

  2. In the first column of the table, click the check boxes for the pending files that you want to remove from the queue.

  3. Click Delete.

    A confirmation dialog box appears.

  4. Click OK.
