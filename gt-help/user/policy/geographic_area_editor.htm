Geographic Area Editor

Path: Policy > Geographic Area Editor

You can use the Geographic Area Editor to group countries into sets that can be labeled and referenced by the Travel Policy or Market Based Preference Editors. You can add, edit, or delete a geographic area.

For more information, click one of the following:

Create a geographic area

  1. Click Add.
  2. In the Name box, type the name you want to use for the geographic area.
  3. In the Countries list, click the countries that you want to be included in the geographic area. You must assign at least one country. To select multiple items, hold down the Ctrl key while you click other items that you want to select.
  4. Click Save.

Edit a geographic area

  1. In the table, click the link in the Geographic Area Name column for the geographic area you want to modify.
  2. Change the geographic area information as needed.
  3. Click Save.

Delete a geographic area

  1. Select the check boxes for the geographic areas you want to delete.

  2. Click Delete.

    A confirmation dialog box appears.

  3. Click OK.

Delete all geographic areas

  1. Select the check box in the header row.

  2. Click Delete.

    A confirmation dialog box appears.

  3. Click OK.
