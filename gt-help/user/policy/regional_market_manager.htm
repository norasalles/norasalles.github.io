Regional Market Manager

Path: Policy > Regional Market Manager

You can use the Regional Market Manager to create, modify, or delete regional markets so you can apply Agency Vendor Preferences.

Note  The regions created here are applicable to the agency preferences level and are independent from the regions at the corporate level. If you want to apply agency vendor preferences to an existing corporate level region, you need to create that exact same region at the agency level using the Regional Market Manager.

Create a Regional Market

  1. Click Add.
  2. In the Name box, type the name you want to use for the regional market.
  3. In the Countries list, click the countries that you want to be included in the regional market. You must assign at least one country to the market. To select multiple items, hold down the Ctrl key while you click other items that you want to select.
  4. Click Save.

Edit a Regional Market

  1. In the table, click the link in the Name column for the regional market you want to modify.
  2. Change the regional market information as needed.
  3. Click Save.

Delete a Regional Market

  1. Select the check boxes for the regional markets you want to delete.

  2. Click Delete.

    A confirmation dialog box appears.

  3. Click OK.

Delete all Regional Markets

  1. Select the check box in the header row.

  2. Click Delete.

    A confirmation dialog box appears.

  3. Click OK.
