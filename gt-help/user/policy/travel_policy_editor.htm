Travel Policy Editor

Path: Policy > Travel Policy Editor

You can use the Travel Policy Editor page to specify the policy to be used on the site. The name must meet the following requirements:

  • You must add a travel policy name that exactly matches the name of the policy created in the Policy Administration Application.
  • The name can be no longer than 60 characters. It may include alphabetical or numeric characters or an ampersand (&).
  • Each policy name must be unique.

  • The first travel policy will be used as the system default.

To add a travel policy

  1. Click Add.

    The Add Travel Policy dialog box appears.

  2. In the Enter a unique name for this travel policy box, add the name of the policy you created in the Policy Administration Application.
  3. Ignore all other options as they are no longer supported.
  4. Click Save.

Delete a travel policy

  1. Select the check boxes for the travel policies you want to delete.

  2. Click Delete.

    A confirmation dialog box appears.

  3. Click OK.
