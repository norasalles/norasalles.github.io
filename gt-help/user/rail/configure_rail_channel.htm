Configure Rail Channel

Path: Rail > GetThere Rail > Configure

GetThere Rail Allows you to only book rail travel and allow the agency and/or traveler to ticket and pay for it. The agency is notified by e-mail of the booking so they can ticket and/or pay for the rail travel. Based on the Rail supplier's e-mail policy, this e-mail address along with the travelers e-mail address is notified of various events, such as e-ticket generation and refunds on cancellation.

The rail segment will be canceled by Amtrak if the payment is not done by the ticketing time limit

If you enable book only functionality, charge card and delivery type fields on the Trip Review and Checkout page are not required and are hidden.

Note  The Book Only option has to be enabled at the rail channel level and cannot be applied at a site level across rail channels.

To enable book only functionality

  1. On the GetThere Rail page, click the Setup tab.

  2. In the Transaction Configuration column, click the Configure link for the channel for which you want to enable Book Only functionality.

    The Configure page appears.

  3. Select the Enable Book Only check box.
  4. In the Agency E-mail box, type the e-mail to which you want notifications sent for the bookings. The Agency can then use the information to ticket and/or make payment for the booking.
  5. Click Save.
