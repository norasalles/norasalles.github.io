Rail Setup

Path: Rail > Rail Setup

You can use this page to configure the settings common to Amtrak SNCF.

Amtrak is only offered through Sabre.

SNCF (Société Nationale des Chemins de Fer Français) is an integrated rail product that provides booking capabilities on the SNCF Host System. By integrating with this system, we offer the ability to make rail bookings on the SNCF, Trains à Grande Vitesse (TGV), Eurostar, Thalys, and Lyria rail lines.

If you want access to rail options, please contact your support representative.

To configure rail setup

  1. The system no longer supports comparing flights and train options.If you want the booking tool to display flight and rail search results together on the same page, select Allow users to compare flight and train options.

    Note  This option applies only when shopping by schedule.

  2. If you allow users to compare flight and train options, select whether you want to Exclude train options if the travel time exceeds a particular duration, then select the hours and minutes for the Maximum Travel Time. For example, do not show train options if the travel time is more than 5 hours, 30 minutes.

    Note  Travel time includes connections and stops.

  3. Select whether you want to Make direct route the default search type. If selected, only direct route trains display for SNCF or SNCB.  Non-direct trains do not display.

    Note  This toggle does not apply to Amtrak or RailConnect.  Only direct trains display for Amtrak.  If the Amtrak route does not offer direct trains, no availability displays.

  4. Select whether you want to Display the Business/Second class search option. If desired, enter a custom label for Business/Second. Select whether you want to Display the First class search option. If desired, enter a custom label for First.

    Note  You must display at least one service class.

  5. If you chose to display both classes of service, select the Default Class of Service: First or Business.
  6. Select one of the following Display Options the system uses for rail vendors that offer more than one class of service:

    Click the

    All available classes of service

    Display rail options for all available classes of service option button.

    Only classes of service allowed for the site based on settings you selected above on this page

    The lowest class of service that is available from the rail vendor and allowed based on settings you selected above on this page

    Display rail options only for the lowest class of service available and allowed for this site option button.

  7. Select the Minimum Interval between Booking and Departure Times if you want to set a limit on the number of hours or days between making a rail reservation and the departure time of the train.

  8. Select the Maximum Number of Travelers per Bookingin a rail itinerary.

    Notes   

    • When comparing flight and train search results, the most restrictive setting between the maximum number of travelers allowed on an air or rail itinerary applies. For Amtrak-only sites, the maximum number of travelers is 8. For SNCF-only sites, the maximum number of travelers is 6. If you have both Amtrak and SNCF enabled, the maximum number of travelers is 6.
    • If you set this field to 1, the booking tool does not display the “Passengers” option on the Home page.

  9. Click Save.

    Notes   

    • When you enable either Amtrak or SNCF, the system creates a section in the Profile called Rail Preferences. You can add an announcement in this section by going to User > Profile Pages > Travel Preferences. Users can select their home station in this section. SNCF users can also enter their seating and meal preferences, European rail discount information, and Pass Entreprise Number.
    • You can set the maximum number of rail travel program membership numbers a traveler can enter in the Frequent Traveler section by setting the number in User > Profile Pages > Travel Preferences > Frequent Traveler.
    • If you enable Amtrak and you offer Arrival and Departure Status to users (Setup > Accessories > Arrival & Departure Status), you can Include Train Status.
