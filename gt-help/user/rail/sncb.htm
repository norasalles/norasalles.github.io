SNCB Setup

Path: Rail > SNCB > Setup

You can use the SNCB Setup page to configure the settings for Société Nationale des Chemins de Fer Belges (SNCB).

Configure SNCB Settings

  1. If this option is available, select the Enable SNCB check box. If this option is not available, SNCB is enabled, but you cannot disable it. Contact your support representative for more information.
  2. In the Negotiated Fares section, do the following:
    1. In the Thalys Corporate Program (B2B) box, type the corporate number for Eurostar.

    2. In the Eurostar Corporate Program (ECP) box, type the corporate number for Eurostar.

      Note  This does not conflict with Eurostar being booked via the GDS.

    3. In the ICE Corporate Program (ICP) box, type the corporate number for Inter-City Express (ICE).
  3. In the Fare Options section, do the following:

    1. In the Maximum number of public fares to display list, click the number of public fares to display on the availability page.

      Note  The default value is 3 and the maximum number of fares allowed to display is 5.

    2. To display only negotiated fares, select the When available, display only negotiated fares check box.

      Note  If you select this check box but the requested route is not offered by any carrier supporting negotiated fares, the system offers public fares and disregards this setting.

  4. Click Save.
