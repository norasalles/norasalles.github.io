Eurostar Setup

Path: Rail > SNCF > Eurostar

You can use the Eurostar Setup page to enter Eurostar negotiated fares.

Add Eurostar Negotiated Fares

  1. Click Add.
  2. In the From box, type the station code to determine the originating station to which this fare applies.
  3. In the To box, type the station code to determine the arrival station to which this fare applies.
  4. In the Fare Basis Code box, type the Eurostar negotiated fare for the city pair entered above.
  5. To indicate the rate applies when travel is to and from the specified cities, select the Negotiated fare for this market is bi-directional check box.
  6. Click Save.

Edit a Eurostar Negotiated Fare

  1. In the table, click the link in the Market column for the Eurostar negotiated fare you want to modify.
  2. Change the Eurostar negotiated fare information as needed.
  3. Click Save.

Delete a Eurostar Negotiated Fare

  1. Select the check boxes for the Eurostar negotiated fares you want to delete.

  2. Click Delete.

    A confirmation dialog box appears.

  3. Click OK.

Delete all Eurostar Negotiated Fares

  1. Select the check box in the header row.

  2. Click Delete.

    A confirmation dialog box appears.

  3. Click OK.
