SNCF Setup

Path: Rail > SNCF > Setup

You can use the SNCF Setup page to configure the settings for SNCF. SNCF (Société Nationale des Chemins de Fer Français) is an integrated rail product that provides booking capabilities on the SNCF Host System. By integrating with this system, we offer the ability to make rail bookings on the SNCF, Trains à Grande Vitesse (TGV), Eurostar, Thalys, and Lyria rail lines.

Configure SNCF Settings

  1. If this option is available, select the Enable SNCF check box. If this option is not available, SNCF is enabled, but you cannot disable it. Contact your support representative for more information.

  2. Select the Country in which Tickets are Issued. Currently, the option is France.

    Note  This field contains a code that sends our RIVA Identifier to SNCF. SNCF assigns this code to online booking systems.

  3. In the Pseudo City for SNCF PNRs box, type the 4 character alpha/numeric pseudo city code (PCC) being used for fulfillment. SNCF assigns this PCC for use within their system. This code is not the Agency GDS PCC.
  4. Enter the Queue Number for SNCF PNRs. Assign a queue number from 200 through 255 to be used by the SNCF PCC for SNCF created PNRs.

  5. Enter the FCE number.

    Note  SNCF assigns an FCE (Fiche Client Enterprise) reference number (e.g., C987254) to contracted companies. When this number is included in the PNR, the sale is reported to SNCF.

  6. In the Show Fare Types list, click the fare types you want displayed in the availability list.

    The options are: First Class - Cheapest, First Class - SemiFlex, Second Class - FullFlex, Second Class - Negotiated, First Class - FullFlex, Second Class - Cheapest, First Class - Negotiated, Second Class - SemiFlex.

    To select multiple items, hold down the Ctrl key while you click other items that you want to select.

  7. In the Fare Type Preference 1, Fare Type Preference 2, and Fare Type Preference 3, lists, select your fare type preferences.

    Notes   

    • Use Fare Type Preference 1, 2, and 3 to identify your preferred rates. (The booking tool displays these rates with a white star inside a purple box.)
    • The system uniquely identifies only the first Fare Type Preference. i.e., If the availability list shows more than one Fare Type Preference, then it identifies the highest ranking Preference.
  8. Click Save.
