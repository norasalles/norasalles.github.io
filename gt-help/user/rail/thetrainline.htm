Path: Rail > thetrainline

To configure thetrainline settings

  1. Select whether to Enable thetrainline. This turns on thetrainline at the Site Administrator level.
  2. If you have a direct contract with thetrainline, type your Corporate Reference ID.
  3. Type your Travel Agent Account Number provided by thetrainline. This number is also considered the Form of Payment ID.
  4. Select Booking Options as follows:
    1. To allow different classes of service to appear together if the original cabin class is not available, select the Display mixed cabin-class options check box.
    2. Select whether to Uniquely identify discounted fares. If you select this option the system identifies discounted fares with a purple star.
  5. Select Other Payment Options as follows:
    1. If you have a direct contract with thetrainline, select whether to Enable Direct Bill for thetrainline bookings.
    2. If you have a direct contract with thetrainline and you chose to Enable Direct Bill for thetrainline bookings, type your Corporate Cost Center Code.

    3. Select which rail cards you would like to enable in the Enable Rail cards list. To select multiple rail cards, hold down the Ctrl key while you click other items that you want to select.

      The information is dynamic with the Home Page.

  6. Select the Class of Service options as follows:
    1. Select whether to Display the First and Standard class search options.
    2. If you select the previous check box, in the Standard Class Availability Options Displayed list, select which brands to display for standard class service.
    3. If you select the previous check box, in the First Class Availability Options Displayed list, select which brands to display for first class service.
  7. Enter Support Information as follows:
    1. If you have a direct contract with thetrainline, type your thetrainline Support E-mail.
    2. If you have a direct contract with thetrainline, type your thetrainline Support Phone Number.
  8. Click Save.
