Feedback

Path: Setup > Accessories > Feedback

Site feedback allows users to e-mail comments to you or your support staff. These comments can range from technical assistance questions to suggestions for new features.

You control the display of the feedback link, including the use of default or custom messaging, categories, and to which e-mail address the feedback is sent.

The feedback categories help you determine the type of feedback the user is submitting. You can use the default categories or up to 6 custom categories. When using the custom categories, users can send feedback to different e-mail addresses based on each category.

When enabled, users click the Send us your feedback link at the bottom of any page to send you feedback.

To configure the site feedback link

  1. Select whether to Show the site feedback link.
  2. If you want to use the default feedback categories, enter an e-mail address in the E-mail feedback to: field.
  3. Select one of the following for the Feedback Submission form:

    • Use default text. The default text for the Feedback Submission form is:

      To make the site easy to use and to include features you need, we need your feedback. Use this form to send us any feedback – positive or negative – about the site’s features, services or technical problems.

    • Use custom text. Enter the message you want displayed on the Feedback Submission form.
  4. Choose one of the following for the Feedback Confirmation form:

    • Select Use default text. The default text for the Feedback Confirmation form is:

      We appreciate your suggestions for improving the site. Your input is extremely valuable to us, as it guides our decisions on product enhancements and new functionality. If you reported a technical problem, it will be addressed as soon as possible. Please check with your travel administrator for any updates.

    • Select Use custom text. Enter the message you want displayed on the Feedback Confirmation form.
  5. Select one of the following for the Feedback Categories:
    • Use default categories. The default categories are:
      • feature I’d like to see added or improved
      • service I’d like to see added or improved
      • system enhancement
      • performance improvement
      • technical assistance
      • other

      Note  If you select Use default categories, you must enter an address in the E-mail feedback to: box at the top of the page.

    • Use custom categories. Enter the categories you want to display and the e-mail address to which the feedback should be sent for each category.
  6. Click Save.
