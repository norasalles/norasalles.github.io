Travel Tools

Path: Setup > Accessories > Travel Tools

Travel Tools is an accessory that you can add to the Home page. The tools consist of a series of links that take the user to travel-related web sites, such as currency rates, weather information, and maps.

If you turn on Travel Tools on the main site, they appear on the Home page of the main site. If you turn on Travel Tools on a subsite, they appear on the subsiteâ€™s Home page.

You have the option of displaying the default links or customizing the labels and/or the web addresses for a majority of the tools.

Display Travel Tools

  1. Select the check boxes for the travel tools you want to display.

  2. Optional: Edit the label and/or the URL for each tool, except the tutorial.

    Note  The limit for the label is 40 alpha-numeric characters. Do not use HTML in the label.

  3. Click Save.
