Trip Templates

Path: Setup > Accessories > Trip Templates

A trip template is a record of a trip that can be used at a later date to make travel reservations for the same type of trip and travel cities.  A trip template speeds up the process of making the reservation because the system already knows which flights, hotels, and rental cars are preferred.

To configure trip templates

  1. Select whether to Enable trip templates.
  2. Select whether to Display the e-mail address on the search results when sharing or acquiring a trip template. If you select this check box, users can see e-mail addresses in the search results during the share and/or acquire trip template process.
  3. Click Save.
