Checkout Page

Path: Setup > Pages > Checkout

You can use the Checkout page to configure:

  • Traveler information
  • Billing information

  • Messaging

For more information, click one of the following:

Traveler Information

Path: Setup > Pages > Checkout | Traveler Information tab

The Traveler Information tab contains settings to control the display of traveler fields on the Trip review and checkout page.

To configure traveler information

  1. Select whether to Permit travelers to edit the Travel Information name fields (first name, last name). If you enable this field, users can change the traveler's name in the Traveler Information section on Trip review and checkout page.

  2. To display the Show Address 2 box, in the Billing Information section of the Trip review and checkout page, select the Show Address 2 check box.

    Note  If you customize the label of Address 2, the setting Show Address 2 reflects the label change. For example, if you change the Address 2 label to Mail Drop, the Show Address 2 setting reads Show Mail Drop.

  3. To require users to enter an e-mail address in the Billing Information section of the Trip review and checkout page, select the Require travelers to provide an e-mail address check box. If users have saved their e-mail address in their profile, the e-mail address is automatically populated in the Billing Information section.
  4. Select whether you want to Require travelers to provide Date of Birth and Gender for TSA Security regulation in the Traveler Information section on Trip review and checkout page.

  5. Select whether to Display General Requests or Special Needs section in the Billing Information section of the Trip review and checkout page. Users can enter up to 390 characters of additional information.

    You can use the text editor to enter or edit the message you want displayed above the general requests or special needs section. There is no default message.

    Note  This setting does not affect the display of this section in the user profile.

  6. To display an Emergency Contact section in the Traveler Information section on the Trip review and checkout page, select the Display Emergency Contact section check box. The Emergency Contact section contains fields for first and last name, telephone number, and country.

    Note  Coordinate the use of this field with your GetThere representative to ensure that this information is correctly passed to the PNR created in the GDS.

  7. If you have GetThere Express enabled, in the Frequent Traveler Information area, select following as needed:
    1. Select whether you want to Permit travelers to provide Frequent Traveler numbers for Air.
    2. Select whether you want to Permit travelers to provide Frequent Traveler numbers for Rail.
    3. Select whether you want to Permit travelers to provide Frequent Traveler numbers for Hotel.
    4. Select whether you want to Permit travelers to provide Frequent Traveler numbers for Car.
  8. In the Checkout Information Customization area, customize the fields with traveler information on the Trip review and checkout page as follows:

    1. In the Default Label column, click the field name.

      A Field Editor dialog box appears.

    2. For e-mail address fields, do any of the following as needed:

    3. In the Example text box, enter instructions and/or examples that appear below the field. For example, you can enter:

      Enter an e-mail address that ends with "fastmail.com", such as "BB123@fastmail.com".

    4. To provide custom validation for the e-mail address fields, select the Custom validation required check box, and do any of the following as needed:
      1. Leave the selection as is in the Type of data users can enter list.

      2. If you want to prohibit the use of one or more e-mail addresses, such as a default e-mail address, select the first check box, and do one of the following:

Billing Information

Path: Setup > Pages > Checkout | Billing Information tab

The Billing Information tab contains settings to control the display of the ticket delivery address, form of payment and purchasing alerts, and messages in the Billing Information section of the Trip review and checkout page.

To configure the billing information

  1. Select whether to Display delivery instructions in the Delivery Address subsection in the Billing Information section of the Trip review and checkout page.

    You can use the text editor to enter or edit the message you want displayed above the ticket delivery section. There is no default message.

  2. Select whether to Allow travelers to request Signature Required for express delivery of travel documents. If you select this check box, the user has the ability to select the option Check if signature is not required for express deliveries. This option is displayed in the Delivery Address section and the default setting is selected.

    Note  Coordinate the use of this field with your GetThere representative to ensure that this information is correctly passed to the PNR created in the GDS.

  3. You can use the text editor to enter or edit the message you want displayed above the charge card information in the Billing Information section. There is no default message.
  4. Select Hide the charge card holder’s name to prevent the credit card holder's from being displayed in the Billing Information section of the Trip review and checkout page. This setting applies only to sites that use a single credit card configuration.
  5. To prevent users from completing a booking when they do not enter credit card information in the Billing Information section of the Trip review and checkout page, select the Require charge cards for all bookings check box. If you do not select this check box, travelers can complete a booking as long as there is no hotel reservation in the itinerary. All hotel bookings are guaranteed.
  6. In the Billing Information box, you can use the text editor to enter or edit the message you want displayed above the Billing Information section of the Trip review and checkout page.
  7. In the Purchase Trip Button Area box, you can use the text editor to enter or edit the message you want displayed above the Purchase Trip button on the Trip review and checkout page.

  8. Select whether you want to Display an alert message when travelers click the Purchase Trip button.

    If you select this check box, you can enter a custom alert message in the Custom Alert Message box, or you can use the default message.

  9. Click Save.

Ground Transportation

Path: Setup > Pages > Checkout | Ground Transportation tab

The Ground Transportation section allows you to control whether a ground transportation section is displayed on the Trip review and checkout page. This section allows travelers to request limousine service and displays regardless of the type of reservation (air, car, hotel, rail) they are making.

To configure the Ground Transportation section

  1. Select whether you want to Display the ground transportation section.

    If you select this check box, you can enter a custom heading or you can use the default heading.

    Note  Coordinate the use of this field with your GetThere representative to ensure that this information is correctly passed to the PNR created in the GDS.

  2. Click Save.

Messaging

Path: Setup > Pages > Checkout | Messaging tab

Note  The types of messages available on your system depends on the features you have enabled. By default these message areas are blank. There are no default messages. You can customize the messages for each language you have enabled.

To enter messages

  1. If applicable, click on the tab for appropriate language.
  2. Do one of the following:

    To add a message

    Do

    That is dependent on one or more conditions and/or requires JavaScript

    Enter information in the Top Dynamic Message box or Bottom Dynamic Message box. For more information, see the Dynamic Text Editor topic or the Velocity Student Guide.

    That always appear regardless of travel choices and does not require JavaScript

    Enter information in the Top Message box or Bottom Message box. For more information, see the Text Editor topic.

  3. To save your messages in Site Administration, at the bottom of the text editor, click Save. (You do not have to click Save each time you enter a message in a different language.)
