Logout Page Setup

Path: Setup > Pages > Logout

This help topic shows you how to configure the following:

  • Logout redirect page
  • Logout Session Expire Page

Redirect Page

Path: Setup > Pages > Logout | Redirect tab

When a user logs out of the booking site, the Logout page appears.

Note  The redirect settings you enter her affect both the desktop and mobile versions of GetThere.

To configure the redirect page

  1. Select the one of the following redirect links when the user logs out of the booking site:
    • Login page – If you select this option, the default message is: Return to the login page (where login page is a link).
    • Alternate Web page – If you select this option, enter the link text that you want displayed on the Logout page. Enter the URL that you want applied to the custom link text. Format the URL as standard text. Do not include HTML tags, but do include the http:// before the URL (e.g., http://www.anysite.com).
  2. Click Save.

Session Expire Page

Path: Setup > Pages > Logout | Session Expire tab

A user receives a session expired page when they are logged into the booking site but have not performed any actions for 20 minutes. A warning pop-up message appears 3 minutes before the session is set to expire.

To configure the Session Expire page

  1. Select one of the redirect links when the session expires:
    • Login page – If you select this option, the default message is: Return to the login page (where the Log In page is a link).
    • Alternate Web page – If you select this option, enter the link text that you want displayed on the Session Expire page. Enter the URL that you want applied to the custom link text. Format the URL as standard text. Do not include HTML tags, but do include the http:// before the URL (e.g., http://www.anysite.com).
  2. Select one of the following for the message:
    • Use default message – If you select this option, the default message is: Session Expired.
    • Use custom message – If you select this option, enter the message you want displayed when the session expires.
  3. Click Save.
