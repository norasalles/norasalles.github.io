Subsite Access Administration

Path: Setup > Site/Subsite Manager > Subsite Access Administration

Superusers can use the Subsite Access Administration page to access Site Administration subsites that are part of the supersite. This allows supersite administrators to change site administration settings for any subsites that are part of the supersite.

Note  This page only appears when in Site Administration on a supersite.

To access a subsite

  1. In the Site Name column, click the link for the subsite.

    Site Administration w page appears

  2. Make changes as needed to the Site Administration of the subsite.
