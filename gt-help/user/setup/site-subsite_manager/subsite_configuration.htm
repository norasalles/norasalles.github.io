Subsite Configuration

Path: Setup > Site/Subsite Manager > Subsite Configuration

You can use the Subsite Configuration page to change the subsite name that is displayed to the user throughout the booking process. You can also reset the password needed to manually add a user to that subsite.

Note  This page only appears when in Site Administration on a subsite.

To change subsite configuration

  1. In the Sub-Site Name box, type a new name for the subsite.
  2. In the Site Key box, type a new password needed to manually add a user to the subsite.
  3. Click Save.
