Subsite Creation Editor

Path: Setup > Site/Subsite Manager > Subsite Creation Editor

Superusers can use the Subsite Creation Editor page to create a subsite. You can create subsites for template sites, test sites, and active customer sites. You should always use a template site to create a new subsite.

Notes   

  • This page only appears when in Site Administration on a supersite.
  • All information in a template subsite copies into the newly created subsite, except for the PNR Acquisition settings on the GT Admin Control page.

To create a subsite

  1. In the New SubSite Information section, enter the following information for the new subsite:
    1. Select an existing Template Site to create the new SubSite. The new subsite that is created has the same settings as the original site, except for the PNR Acquisition settings on the GT Admin Control page.
    2. Enter a unique name for this new SubSite.It must be lowercase and less than 16 characters. It cannot have the same name as the supersite.
    3. Enter the Company Name.
    4. Enter the SubSite creation password. This password is assigned in the SuperSite Configuration Editor. If you do not have this password contact your GetThere representative.
    5. Enter a password that is used when creating a user account on this SubSite. This password is used when creating new users on the User tab. Also, to allow new users to create their own account on the site you must enter a password. This password must be communicated to the new user and they must enter it when creating a new account.
  2. In the New SubSite Administration Information section, enter the following administrator information for the new subsite:
    1. Enter a unique user name for the SubSite administrator.
    2. Enter a password for the SubSite administrator.
    3. Enter the SubSite administrator's e-mail address.
  3. Click Save.
