SubSite Deletion Editor

Path: Setup > Site/Subsite Manager > Subsite Deletion Editor

You can use the SubSite Deletion Editor page to delete a subsite.

Caution  All subsite deletions are final. Users of deleted subsites will no longer have access.

To delete a subsite

  1. Select one or more site's to delete.
  2. Click Save.
