Subsite Status Editor

Path: Setup > Site/Subsite Manager > Subsite Status Editor

You can use the SubSite Status Editor page to maintain control over access to these sites and to set up sites as templates for new sites. This is especially important when working with large sites with multiple subsites.

Note  This page only appears when in Site Administration on a supersite.

To change the status of a subsite

  1. Select one of the following statuses you want to apply to one or more subsites:
    • Open - Unlock your subsite after it has been locked.
    • Locked - Lock your site to prevent users from making bookings. This could be used during maintenance or when changing to a new booking site.
    • Template - Convert the site to a template site to be used to create other sites.
    • Standard - Prevent the site from being used as a template site due to its customizations.
  2. Select the sites to which you want this status applied. To select multiple items, hold down the Ctrl key while you click other items that you want to select.
  3. Click Save.
