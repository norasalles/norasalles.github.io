Supersite Configuration Editor

Path: Setup > Site/Subsite Manager > SuperSite Configuration Editor

Use this page to set restrictions on new subsite creation.

To configure supersite logon page labels

Enter a new label for the SubSite field to alter the field labels that appear in the header throughout the booking tool.

To configure subsite creation

  1. Select What is the initial status of new SubSites when they are created? You can allow users to access the subsite immediately by setting it to Open, or inhibit access by setting it to Locked until you have configured the subsite.
  2. Enter an e-mail address that will be notified when a SubSite is created. An e-mail is sent to this e-mail address when any subsite is created.
  3. Enter a password, key, PIN or ID that permits SubSites to be created.
  4. Select What is the minimum character length that SubSites must use when they create a password, key, PIN, or ID to protect use of their SubSite? You can set the password length from a minimum of 5 characters to a maximum of 10 characters.
  5. The section to enter an instructional or  welcome message is no longer applicable.
  6. Click Save.
