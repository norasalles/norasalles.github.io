Collaboration Tabs Page

Path: Setup > Site Preferences > Collaboration Tabs

You can use the Collaboration Tabs page to enable tabs across the top of the new Home page that gives you access to other functionality, such as Sabre Virtual Meetings and outbound single sign on sites. If you enable Collaboration tabs, the system shows the Travel tab for normal travel functionality. You can enable up to 5 collaboration tabs.

When a user clicks one of the collaboration tabs, the site opens in a new window.

Note  If you add a tab for an outbound SSO, but you do not have outbound SSO enabled, the tab does not appear.

Configure collaboration tabs

  1. Select the Enable Collaboration tabs check box, and click Save.

    The Tabs Setup area appears.

  2. Do one of the following:

    To

    Then

    Add a collaboration tab

    1. Click Add.
    2. In the Tab Name box, type the name you want to appear on the tab. You can use up to 30 characters.
    3. Do one of the following:

      To

      Then

      Add a custom URL for a tab

      1. Click Custom.
      2. In the URL box, type the URL for the site you want to appear. Your URL can have up to 512 characters and must start with "http://" or "https://".

      Add a outbound single sign on destination

       

       

      1. Click SSO.
      2. In the Site list, click the Outbound SSO corresponding to the functionality you want to enable. The list contains all outbound SSOs enabled for your site.

       

    4. Click Save. The tabs appear on the new Home page in the order that you add them.

    Delete a collaboration tab

    1. Select the tab you want to delete, and click Delete.
    2. To confirm the deletion, click OK.

     

  3. Click Save to save the changes.
