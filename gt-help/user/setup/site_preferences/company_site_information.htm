Company Information

Path: Setup > Site Preferences > Company Information

You can use this page to enter the following types of company information: Name/Address/Phone, Contact Information, and Site Information.

To enter the company information

  1. Type the Name/Address/Phone information in the following boxes:
    • Company Name (This name appears at the bottom of all booking tool pages beside the copyright information.)
    • Address 1
    • Address 2
    • City
    • State
    • Zip Code
    • Country
    • Toll-Free Number
  2. Type the Contact Information in the following boxes:
    • Contact Name 1
    • Title
    • E-mail
    • Phone Number
    • Fax Number
    • Contact Name 2
    • Title
    • E-mail
    • Phone Number
    • Fax Number
  3. Type the Site Information in the following boxes:
    • Number of Travelers
    • Air Volume $
    • Number of Locations
    • Standard Browser
    • Standard Platform
  4. Click Save.
