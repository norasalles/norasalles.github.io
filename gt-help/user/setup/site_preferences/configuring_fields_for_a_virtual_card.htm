Configuring Fields for a Virtual Card

Path: Setup > Site Preferences > Payment Options > Multiple Charge Card Configuration tab > Configure Fields link

The main site name and subsite name are always sent to the virtual card vendor. The 8 default fields will also be reported when using GetThere on smartphones. However, you can also send additional fields.

To configure fields for a virtual card

  1. In the Payment Options page, click the Multiple Charge Card Configuration tab.

  2. In the Custom Data Fields column, click the Configure Fields link for the virtual card.

    The Configure Fields page appears for the virtual card.

  3. Do one of the following:

    To

    Do

    Add a custom data field

    Note  You can add up to 120 custom data fields.

    1. Click Add.
    2. In the Field Label box, type a label for the custom data field. You can enter up to 50 characters. The label must be unique for the virtual card.
    3. In the Field Expression box, type the field expression for retrieving the data and sending the data as follows:
      • The LVIS tags used for PNR Editor Strings should be used for Field Expressions.
      • If the result for the configured field expression is greater than 200, the data will be truncated to 200 characters and sent to Conferma.
      • JavaScript is not allowed.
      • The following is list of field expressions that have been created specifically for reporting. These should not be used in PNR Editor Strings.
        • itin.hotel.chain
        • itin.hotel.checkinDate
        • itin.hotel.checkoutDate
        • itin.hotel.ssr.count
        • itin.hotel.ssr
        • itin.hotel.room_count
        • itin.hotel.property_id
        • itin.hotel.rate
        • itin.hotel.address
        • itin.hotel.phone_number
        • itin.hotel.room_desc

        There are corresponding PNR editor tags that are available, and they should not be configured in a field expression.

    4. Click Save.

    Edit a custom data field

    1. In the Field Label column, click the link of the field you want edit.
    2. Change the field label and/or field expression as needed.
    3. Click Save.

    Delete a custom data field

    1. In the first column, select the check boxes for the custom data fields you want to delete.

    2. Click Delete.

      A confirmation dialog appears.

    3. Click OK.
  4. Click Save.
