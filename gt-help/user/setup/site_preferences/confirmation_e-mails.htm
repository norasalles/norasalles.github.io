Confirmation E-mails

Path: Setup > Site Preferences > Confirmation E-mails

You can use the Confirmation E-mails page to customize the headers and footers of e-mails the system sends when a reservation is completed or changed.

To setup confirmation e-mails

  1. Select whether you want to include a header message in the confirmation e-mail. If so, select one of the following:
    • Use default message – If you select this option, the default message is: Thank you for using the online booking tool for your travel arrangements. Your reservation details appear below. Please keep this e-mail in a safe place so you can refer to it later.
    • Use custom message – If you select this option, type the message you want sent in the header when a reservation is completed or changed. HTML is not supported in this field.
  2. Select whether you want to include a footer message in the confirmation e-mail. If so, select one of the following:
    • Use default message – If you select this option, the default message is: Have a great trip!
    • Use custom message – If you select this option, type the message you want sent in the footer when a reservation is completed or changed. HTML is not supported in this field.
  3. Select Save.
