Express Checkout Labels

The Express Checkout section contains labels that affect the Trip review and checkout page when you have Express Checkout selected.

The custom label feature allows you to replace the system default labels with customized text.

Note  Only the user interface is changed, the original field names remain unchanged in the database.

 To set up express checkout labels

  1. Enter or edit custom labels for the following items:

    Note  You can enter up to 120 characters, including special characters.

    • Trip Details
    • Policy Compliance
    • Rail Seating Preference
    • Traveler Information
    • Billing Information
    • Delivery Information
    • Company Information
    • Request Limousine Service
    • Additional Information
    • Estimated Trip Cost
  2. Click Save.
