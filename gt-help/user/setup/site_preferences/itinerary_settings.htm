Itinerary Settings

Path: Setup > Site Preferences > Itinerary Settings

You can use the Itinerary Settings page to set options that have impact to multiple pages within the booking process.

To configure itinerary settings

  1. Select whether to Display travel agency information on the Reservation Complete and Trip Details pages. This information includes: Agency name, address, city, state, zip/postal code, country, phone number(s) and hours of operation.
  2. Select whether to Display travel agency fax number on the Reservation Complete and Trip Details pages.
  3. Select whether to Display the iCalendar link on the Reservation Complete and Trip Details pages, which allows travelers to download trip information to Outlook/Smartphone calendars.
  4. Select whether to Display the E-mail Itinerary section on the Reservation Complete and Trip Details pages, which allow travelers to send trip information for calendars by e-mail.
  5. Select whether to Display links to social networking sites on the Reservation Complete and Trip Details pages. You can choose up to 5.
  6. Select whether to Display the TripCase itinerary management section on the Reservation Complete and Trip Details pages, which allows travelers to send their itineraries to their TripCase account..
  7. Select whether to Warn users about itineraries that overlap existing trips on the Trip review and checkout page. If you select this check box, users see a warning message on every shopping page after they select an overlapping flight. Users must select a check box at the bottom of the Trip review and checkout page to acknowledge the overlap before they will be able to purchase the trip or put the trip on hold.
  8. Select whether to Prohibit users from booking itineraries that overlap existing trips. If you select this check box, users see a warning message on every shopping page after they select an overlapping flight and the Trip review and checkout page. Users are not able to purchase the trip or put the trip on hold. This applies to the trips that:
    • Leave on the same day from the same location as an existing trip.
    • Contain flights that depart less than 30 minutes after the arrival time of a previous flight.
    • Contain trains that depart less than 30 minutes after the arrival time of a previous train.

  9. You can add custom link text and an alert message you want displayed when hotel bookings cannot be confirmed. Until we receive the confirmation number from the vendor, the system displays a link that opens a pop-up alert.

    The default link is: On Request.

    The default alert message is: Please contact your agency to confirm your reservation. This link is on the Checkout and Reservation Complete pages. The booking request is not sent to the vendors until the user clicks Purchase Trip.

  10. You can add a custom link text and an alert message you want displayed when rental car bookings cannot be confirmed. Until we receive the confirmation number from the vendor, we display a link that opens a pop-up alert.

    The default link is: On Request.

    The default alert message is: Please contact your agency to confirm your reservation. This link is on the Checkout and Reservation Complete pages. The booking request is not sent to the vendors until the user clicks Purchase Trip.

  11. Click Save.
