Subsite Labels

You can use the Labels page to replace the system default labels for the subsite with customized text.

 To set up site labels

  1. Enter or edit custom labels as applicable.

    Item

    Default Label

    Displays

    Subsite Header Label

    Subsite

    On all pages throughout the booking tool when on a subsite.

    Traveler Information

     

     

     

     

     

     

    Address1

    In the Personal Information page of the user profile and the Delivery Address section of the Trip review and checkout page.

    Address2

    In the Personal Information page of the user profile and the Delivery Address section of the Trip review and checkout page.

    City

    In the Personal Information page of the user profile and the Delivery Address section of the Trip review and checkout page.

    State/Province

    In the Personal Information page of the user profile and the Delivery Address section of the Trip review and checkout page.

    Postal Code

    In the Personal Information page of the user profile and the Delivery Address section of the Trip review and checkout page.

    Country

    In the Personal Information page of the user profile and the Delivery Address section of the Trip review and checkout page.

    Work Phone

    In the Personal Information page of the user profile and the Delivery Address section of the Trip review and checkout page.

    E-mail Address

    E-mail Address

    In the E-mail Settings page of the user profile, the Delivery Address section of the Trip review and checkout page and the Add a New User page in Site Administration.

    Emergency Contact

     

    Country

    In the Emergency Contact Information area of the Personal Information page of the user profile and the Specify an emergency contact for this trip section of the Trip review and checkout page.

    Telephone Number

    In the Emergency Contact Information area of the Personal Information page of the user profile and the Specify an emergency contact for this trip section of the Trip review and checkout page.

  2. Click Save.
