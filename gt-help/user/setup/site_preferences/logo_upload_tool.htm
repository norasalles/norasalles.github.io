Logo Upload Tool

Path: Setup > Site Preferences > Logo Upload

You can use the Logo Upload Tool page to to upload company and product logos to a booking site. The images shown in the Image column are the images currently displaying on your site.

You can upload custom images for the following items:

  • Title logo – This image appears first in the upper left corner of the Login page, and then on every other succeeding page. This is where the company logo normally appears.
  • Register logo – This image no longer applies.
  • HDR product name logo – This header (hdr) product name image appears first on the Login page in the upper right hand corner, and then on every other succeeding page in the upper left corner next to the Title logo.

You can also delete a logo by replacing it with a blank image.

To replace a logo file

  1. Click the Customize button to the left of the logo you want to replace.
  2. To select the file you want to use for the logo, click Browse, and select replacement logo.
    • You must name the replacement logo the same name as the existing logo.  For example, if you want to replace the title logo, name and save the replacement logo with the same name as the existing logo listed in the File Name column, title.logo.gif, before it is loaded.
    • The logo must be a .gif file.
    • To ensure your logo displays properly, you must use the image size listed in the Preferred Size column.  If you use a size other than the one recommended, your image may be stretched or compressed to fit the preferred image size.  Do not use an image larger than 5KB as this causes your site to display more slowly.

  3. Click Upload.

    A preview page appears.

  4. If the logo is correct, click Approve.

    Note  It can take up to 24 hours before the new logo is viewable on your site.

  5. Repeat the above steps to add other logos.

To delete a logo file

  1. Create a blank .gif file.
  2. Click the Customize button to the left of the logo you want to replace.
  3. Click Browse, and select blank .gif file.
    • You must name the replacement logo the same name as the existing logo.  For example, if you want to replace the title logo, name and save the replacement logo with the same name as the existing logo listed in the File Name column, title.logo.gif, before it is loaded.
    • The logo must be a .gif file.
    • To ensure your logo displays properly, you must use the image size listed in the Preferred Size column.  If you use a size other than the one recommended, your image may be stretched or compressed to fit the preferred image size.  Do not use an image larger than 5KB as this causes your site to display more slowly.

  4. Click Upload.

    A preview page appears.

  5. If the logo is correct, click Approve.

    Note  It can take up to 24 hours before the new logo is viewable on your site.

  6. Repeat the above steps to delete other logos.
