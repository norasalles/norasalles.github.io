Language

Path: Setup > Site Preferences > Language

If you enable two or more languages on your site, users have the ability to set their preference in their profile. The Preferred Language setting appears on the Display Preferences page of the Profile.

The default and additional languages selected on the main site control the Log In Page. To avoid user confusion, please ensure consistency across the main site and subsites. Please use caution when disabling a language because users may have set that language as the default in their Profile.

We support the following languages:

  • Arabic
  • Chinese (Simplified Han)
  • English (U.S.)
  • English (Great Britain)
  • French (European)
  • French (Canada)
  • German
  • Italian
  • Japanese
  • Korean
  • Nederlands / Dutch
  • Nederlands / Dutch (Belgian)
  • Portuguese (Brazilian)
  • Russian
  • Spanish (Mexican)

Session defined language

The Log In page is displayed according to:

  1. The user’s browser setting (if that language is enabled on the site)
  2. The site’s default language

The Home page is displayed according to:

  1. The language selected by the user on the Log In page
  2. The language setting in the user profile
  3. The user’s browser setting (if that language is enabled on the site)
  4. The site’s default language

To set up site languages

  1. In the SIte Default Language list, click the language you want to have as the default language for the site. The default language is automatically selected as one of the Additional language options you want displayed on the Login and Home pages.
  2. Select the additional languages from which travelers can choose.  
  3. Click Save.
