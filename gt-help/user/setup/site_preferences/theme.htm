Theme

Path: Setup > Site Preferences > Theme

You can use the Theme page to choose a theme, brand, and color style (a unique set of colors) to be applied to a site or subsite.

Note  The main site theme settings are applied to the Log in page.

Select a theme

  1. In the Color Style list, select the color you want applied to your site.

    Note  This only applies to pages that have not be redesigned as part of the User Experience initiative.

  2. In the Primary Accent Color list, click the color you want for the navigation banner (the rectangular color stripe) below the header section at the top of each page. You can select a color using two color selectors (hue and intensity) or by entered using a color RGB hex code.
  3. In the Secondary Accent Color list, click the color you want for the primary action buttons shown on each page. You can select a color using two color selectors (hue and intensity) or by entered using a color RGB hex code.
  4. Click Save.
