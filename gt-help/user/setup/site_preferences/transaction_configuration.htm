Sites/Carriers Configure

Path: Setup >  Site Preferences > Air Connect > (Configure Link)

After you add a site with Air Extras, you can configure settings for the Air Extras on the Sites/Carriers Configure page.

To configure Air Extras

  1. On the Air Connect page on the Sites tab, in the Transaction Configuration column, click the Configure link for the site.
  2. Select Which account type should be used to book on the external site.
    • Personal Only – This means use only the Traveler’s login information. (Not Recommended)

      Note  If you configure Personal only and the Traveler does not have a Personal login, then no Web fares are returned.

    • Personal if available, otherwise don't log in – If the Traveler’s personal login is available, use it.  If not, then login anonymously.(Best practice)
    • Person if available, otherwise use default – If the Traveler’s personal login is available, use it.  If not, then use default login setup below.

    • Always Default – For Agency Site Access Only – For Agency Sites enter the Default Name and Default password. Maximum of 32 characters permitted in each field.

      Notes   

      • This setting is used to default all searches to an Agency site that requires a log-in.
      • Germanwings – A login is required to access Germanwings’ information.  Current settings need to be Personal if available, otherwise use default (access is their consumer site).
      • EasyJet – A login is required to access EasyJet information. The site can be configured to either Personal Only or Personal if available, otherwise use default.
  3. If required by the carrier, in the Corporate or Agency ID/CD box, type your 8-digit company ID.
  4. If required by the carrier, in the Agent Username box, type the agent user name.
  5. If required by the carrier, in the Agent Password box, type the agent password.
  6. If required by the carrier, in the Default Name box, type the default user name.
  7. If required by the carrier, in the Default Password box, type the default password.

  8. If required by the carrier, in the Which form of Payment should be used? list, select one of the following:

    Note  This menu only shows those payment options that are supported by the carrier being configured.

    • Instant Payment to Supplier – The charge card is processed by the supplier when the reservation is created.
    • Instant Payment through an Intermediary – The charge card is processed by an intermediary on behalf of the supplier when the reservation is created.
    • Direct Bill – The account of the company who implemented the booking tool or the agency is used to secure the reservation and then is invoiced by the supplier.
  9. Depending on the vendor selected, do one of the following:
    If you selectDo
    Air Aurigny
    1. In the Which account type should be used to book on the external site? list, click Personal if available, otherwise don't log in.
    2. In the Which form of Payment should be used? list, click Instant Payment through an Intermediary.
    FlyDubai
    1. In the Which account type should be used to book on the external site? list, click Personal if available, otherwise don't log in.
    2. In the Which form of Payment should be used? list, click Instant Payment through an Intermediary.
    Germanwings
    1. In order to retrieve Germanwings bookings from their B2C booking site, a 10 digit Customer ID is required. The Customer ID is not required to obtain availability and to book.  This is only required if the user wants to display their reservation from the Germanwings B2C booking site.

      Note  The Default Name and Default Password are not required.

    2. Which account type should be used to book on the external site? The recommended setting is Personal if available, otherwise use default.

      Note  The user's login credentials from their profile as well as the Default Name and Default Password are not sent to the vendor.

    3. Agent Code . Enter the 10 digit Customer ID.
    GoAir
    1. In the Which account type should be used to book on the external site? list, click Personal if available, otherwise don't log in.
    2. In the Which form of Payment should be used? list, click Instant Payment through an Intermediary.
    GoAirb2b
    1. In the Agent IATA Number box, type the Agent IATA number.
    2. To ensure you can access corporate fares, type the User Login and User Password.

    Note  A separate corporate negotiated fares code or promo code is not currently supported by GoAirb2b.

    IndiGo
    1. In the Which account type should be used to book on the external site? list, click Personal if available, otherwise don't log in.
    2. In the Which form of Payment should be used? list, click Instant Payment through an Intermediary.
    SpiceJet
    1. In the Which account type should be used to book on the external site? list, click Personal if available, otherwise don't log in.
    2. In the Which form of Payment should be used? list, click Instant Payment through an Intermediary.
    Voxel Carriers, such as Azul and Gol

    1. In the International Fare Code box, type the negotiated fare code for international flights.

      Note  Only one Fare Code can be entered. This box does not appear for Passaredo.

      Caution  If the value entered is incorrect, the negotiated fares are not returned by Voxel and users are not able to shop and book flights with negotiated fares.

    2. In the Domestic Fare Code box, type the negotiated fare code for domestic flights.

      Note  Only one Fare Code can be entered. This box does not appear for Passaredo.

      Caution  If the value entered is incorrect, the negotiated fares are not returned by Voxel and users are not able to shop and book flights with negotiated fares.

    Azul, Gol, Avianca, and Passaredo

    If you only want purchases to be booked and not ticketed (because some require the agency to ticket), select the Enable book only for this carrier check box.

    Note  Once a "book only" reservation is completed, the traveler sees a PNR on the confirmation page. These bookings are shown in the Active Trips section on the Trips page, and their status will not change regardless of whether the agency actually tickets them.

    Southwest

    1. Indicate whether you want to Enable Cancel for this carrier.

      Note  The Cancel trip button only appears on the Trip details page if all elements of the trip (flights, cars, hotels) can be canceled.

    2. Indicate whether you want to Enable Book with RTF for this carrier. This allows you to use some of the value of previous trips booked and canceled with Southwest Airlines on future Southwest Airlines flights.

      Note  You can only select the Enable Book with RTF for this carrier check box, if you selected the Cancel trip check box.

    For more information, see the Southwest via Air Connect student guide.

  10. Click Save.
  11. To go back to the Air Connect page, click the Sites/Carriers link.
