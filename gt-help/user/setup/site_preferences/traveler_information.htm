Traveler Information

Path: Setup > Site Preferences > Traveler Information

You can use the Traveler Information page to configure the following:

  • Traveler Information Settings
  • Expiration Reminders

For more information, click one of the following:

Traveler Information Settings

Path: Setup > Site Preferences > Traveler Information | Settings tab

You can use the Settings tab to configure salutation, charge card, and e-mail confirmation settings that affect multiple pages within the booking process.

To configure traveler information settings

  1. In the Settings tab, select whether to Display traveler's title/salutation on the Checkout and Profile pages. If you select this check box, a list is displayed on the Trip review and checkout page and the Personal Information section of the user profile. The available options are Mr, Ms, Mrs, Miss, and Dr.
  2. Select how to display Charge Card Numbers on the Trip review and checkout page and the Charge cards page of the user profile. You can choose to replace the number with the text On File, or you can display the last four digits of the number.
  3. To replace the credit card number with asterisks (****) as the traveler enters the number on the Trip review and checkout page, select the Mask card number with asterisks when entering card information check box.
  4. In the Additional E-mail Confirmations section, select whether to allow travelers to send a copy of the their itinerary to up to three people. The additional e-mail boxes appear in the Additional E-mail Confirmations section of the Trip review and checkout page and the E-mail settings page of the user profile.
  5. Click Save.

Expiration Reminders

Path: Setup > Site Preferences > Traveler Information | Expiration Reminders tab

You can use the Expiration Reminders tab to set a system warning that notifies users upon log in that their passport, visas, or charge cards will expire soon.

To configure expiration reminders

  1. Select when to display an expiration reminder for Passport, Visa, or Charge Card. The options are: 30, 60, 90, 120, or 180 days before expiration for passports and visas. The options are: 30, 60, 90, or 120 days before expiration for charge cards. Users see the message on the Home page or when they access their user profile. Users can deactivate this feature by turning it off in the Personal Information page of the user profile.
  2. Click Save.
