Air Connect

Path: Setup >  Site Preferences > Air Connect

The Air Connect page only appears if your site has Air Connect enabled.

Note  Preferred markets must be set at Shuttle, City, Country, Region, and Global preferences levels to enable Air Connect fares for those markets.

For more information, click one of the following:

Setup Tab

Path: Setup > Site Preferences > Air Connect | Setup tab

Search Options

  1. Select whether to Require the user to manually request Air Connect options.
    • If you clear this check box, the system always displays Air Connect options to the Traveler if applicable.
    • If you select this check box, the traveler must click the Show me fares from other internet sites check box on the Home page to display Air Connect options.

Display

  1. Select whether to Show GDS responses for Flexible One Way Fare carriers on Search by Time. If you select this check box, GDS branded flights for Flexible One Way Fare carriers that are also web fare displays.
  2. Select whether to Enable filtering of Web fares on Search by Price. If you select this check box, then configure the following two items:

    Note   Without filtering enabled, all options returned are displayed.

    1. Select whether to Block Web Fare Itineraries. If you select this check box, a web fare that is the same as a fare found in the GDS does not display.
    2. In the Minimum savings needed to display a Web Fare list, click a savings amount that must be met before the Web fare is displayed.

      Note  This amount is in USD and the application uses the GDS to convert the USD $ amount to the currency setting for the site.  For example, if you wish to block fares under Â£50.00, select $100.00 USD as this is the approximate conversion.

  3. To display the Fare Details link within the Trip details section on the Trip review and checkout page, select the Enable Fare Details check box.

    Note  This setting applies to the Auto Book vendors that contain an asterisk (*) and VIA Rail.

  4. Select whether to Show the Supplier's booking location on My Trip, the Confirmation page and the confirmation e-mail.  If you select this check box the supplier's name appears next to the web fare option.

    Note  This is applicable to Auto and Mediated bookings only.

  5. In the Maximum number of seconds the system should wait for response list, click the amount of time the system waits for a response from your designated Air Connect travel site(s). Depending on the number of carriers configured in Market Based Preferences, 35 to 60 seconds is recommended.

    Note  The system waits for a response from all travel sites designated for a specific market.  At the maximum time allotted, GetThere displays web fares from the Travel sites that have responded. If no response is received from a supplier, no web fares from that supplier are displayed.

Non-GDS Carrier

  1. Select the time window that should be considered in the itinerary evaluation process. Choose the time window for web sites to return applicable itineraries.  You can set this option from 1 hour to 8 hours before and after the requested time.

    Note  A minimum of 4 hours is recommended.

 Configuration

  1. Select whether to Create a passive air segment in the GDS PNR to record the Air Connect booking.
    • If you select this check box, the system creates a passive PNR in the GDS.  
    • If you clear this check box, the system continues to store the booking data in the database, and the reservation does not appear in the user's Trips page.  An Air Connect confirmation number is displayed versus a GDS PNR Locator number.

    Note  This is applicable for Auto and Mediated bookings only.

  2. Click Save.

Sites Tab

Path: Setup > Site Preferences > Air Connect | Sites tab

On the Sites tab, you can select one or more sites that you want Air Connect to search.  You can enable up to 15 sites.

The more sites that are enabled, the slower the response may be.

Depending on the type of booking, click one of the following for more information:

    1. On the Sites tab, click Add.

      Note  As default, all Air Extras are disabled. All possible Air Extras that may be configured are displayed, regardless if the supplier offers the Air Extra or not. If an Air Extra is configured but it is not returned during the booking process, it is not displayed to the user. However, some suppliers may have unique requirements on how they charge for their Air Extras.

Link Booking

Link booking sites require no setup.

  1. On the Sites tab, click Add.

  4. Click Save.  

Messaging Tab

Path: Setup > Site Preferences > Air Connect | Messaging tab

Enter any unique messages or leave the message boxes blank to have the default messages display.

My Trips

  1. Enter the message you want displayed on the My Trips page when Air Connect is enabled. You can enter up to 255 characters.

    Note  When messages are created, do not request any credit card information for the fields that are not designed specifically for credit card data. This is a requirement for PCI certification.

Confirmation and Confirmation email

  1. Enter the message you want displayed on the Confirmation page and the user's confirmation email (Auto Booking only). You can enter up to 255 characters, and HTML is not allowed.

Popup

  1. Enter the message you want displayed in the popup when the user selects a Air Connect flight (Auto Booking only).
