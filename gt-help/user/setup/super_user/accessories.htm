Accessories

Path: Setup > Super User > Accessories

You can use the Accessories page to enable meetings management and social networking.

To enable meetings management and social networking

  1. On the Meetings Management tab, select whether to Enable Meetings Management. If you select this check box, Users are required to login with their user ID and password.

    Note  This functionality applies to MidMarket customers only.

  2. In the Social Networking tab, do the following:
    1. Select whether to Enable Social Networking.

    2. If you enable social networking, in the URL box, type the URL provided by Cubeless when the request for a new site is sent. This request is found on SharePoint.

      Note  This functionality applies to MidMarket customers only.

  3. Click Save.
