Accounting Code Validation via Web Service

Path: Setup > Super User > Accounting Code Validation

You can use the Accounting Code Validation via Web Service page to enable Accenture Live Accounting Code Evaluation of the users' WBS Code (cost center/company code) at the subsite level.

To enable accounting code validation via web service

  1. Select whether to Enable Live Accounting Code Validation via Web Service. If you select this check box, Site Administrators have the ability to enable/disable the functionality of invoking a Web Service to validate the WBS code at the subsite level.

  2. In the Select the customer name list, select the customer for whom you want to activate validation.

    Note  If you want the option to include the employee number in the validation test, select Accenture via REST.

  3. Click Save.
