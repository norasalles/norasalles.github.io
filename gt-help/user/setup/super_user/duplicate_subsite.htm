Duplicate SubSite

Path: Setup > Super User > Duplicate SubSite

You can use the Duplicate Subsite page to copy a subsite from one SuperSite to another. The majority of all fields are copied.  There are certain fields and data that are not copied.  The following item are not copied to the new subsite.

  • TA Groups
  • SuperPNR Configuration
  • Bulk CRS Sync settings
  • PNR Acquire settings
  • Preferred Hotel Upload Tool
  • Accounting Code Validation
  • Logos
  • Direct Data configuration

The Duplicate Subsite tool is available to use between 6pm – 6am CST Monday to Friday and all day Saturday and Sunday.

Note  When duplicating a subsite, make sure that you are logged into the site where you wish to create the new subsite.

To configure inbound SSO

  1. Enter the SuperSite name to copy from.
  2. Enter the SubSite name to copy from.
  3. Enter the Destination Subsite Name.

  4. Enter the Company Name.

    Note  The maximum number of characters allowed is 64.

  5. Enter a password that is used when creating a user account on this SubSite.

  6. Enter a unique user name for the SubSite administrator.

    Note  You can use alphanumeric or special characters.

  7. Enter a password for the SubSite administrator.

    Note  You can use alphanumeric or special characters.

  8. Enter the SubSite administrator’s e-mail adress.
  9. Click Save.
