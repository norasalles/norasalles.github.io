GetThere Experience

Path: Setup > Super User > GetThere Experience

  1. To display the Trips tab on the new Home page, select the Enable Trips on the New Home Page check box.

  2. To use Bing maps instead of MapQuest maps, select the Enable Bing Maps check box.

    Note  Bing maps are currently only authorized for Microsoft sites only.

  3. Select whether to Enable the newest version of the Trip review and checkout page.

    Warning  Do not select this check box at this time. This is check box is for future use.

  4. Select whether to Enable the newest version of the Profile pages. If you select this, JavaScript is not allowed.

  5. To enable sites to use Demand Management, select the Enable Demand Management check box. For more information, see Demand Management.
  6. To enable smart messaging, select the Enable Smart Messaging check box. For more information, see Smart Messaging.

  7. If the customer is a TripCase Corporate customer and you want to automatically send itineraries to TripCase when you book, select the Enable TripCase Corporate Auto-Push check box. This also sends cancellations made in GetThere to TripCase.

    You can still manually send the itinerary to TripCase from the Trip Details page. You can also send it from the Reservation Complete page if there is an error when the system tries to automatically send the itinerary. Otherwise, you no longer see this section on the Reservation Complete page.

  8. To enable inbound single sign-on for TripCase Corporate customers, select the Enable TripCase Inbound SSO check box.
  9. Click Save.
