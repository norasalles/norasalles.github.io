Single Sign On (Site Certificates)

Path: Setup > Super User > Single Sign On

Note  The information contained in this section only explains the pages that are viewable.  For complete configuration of SAML SSO, please contact Data Integration.

To configure single sign on

  1. In the Security Contact tab, in the Name box, type the first and last name of the security contact. You can type up to 64 characters.
  2. In the Title box, type the title of the security contact. You can type up to 32 characters.

  3. In the Phone Number box, type the complete phone number of the security contact including the country code if applicable. You can type up to 24 characters.

    Note  This is extremely important because it is used in Step 3 of the Verification process

  4. In the E-mail box, type the e-mail address of the security contact. You can type up to 64 characters.
  5. Click Save.

To view certificate status

  • Click the Verify Site Certificates tab.

    The Data Integration Team will Validate Site Certificates.
