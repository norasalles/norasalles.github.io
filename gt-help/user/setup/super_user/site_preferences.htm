Site Preferences

Path: Setup > Super User > Site Preferences

You can use the Site Preferences page to enable Auto Complete for airport and city code searches.

Auto Complete only supports English, Spanish, French and Italian languages and is only available on certain pages, depending on the default site language configured as follows:

  • English:  Air and Hotel search pages
  • Spanish, French, Italian: Air search page

To configure autocomplete settings

  1. If you want the system to display words or phrases that a user may want to enter in the search fields after they have entered at least three characters, select the Enable Autocomplete check box.

  2. If you want to allow users to pay with virtual cards on the site, select the Enable Virtual Payments check box.

    Note  Virtual Payments cannot be disabled when virtual cards exist in Site Payments.

  3. Click Save.
