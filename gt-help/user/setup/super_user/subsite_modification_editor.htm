Subsite Modification Editor

Path: Setup > Super User > Subsite Modification Editor

To enable Subsite Modification Editor

  1. Select whether to Enable Subsite Modification Editor for this site. If you select this check box, the SubSite Modification Editor page is displayed at Setup > Site/Subsite Manager > Subsite Modification Editor.
  2. Click Save.
