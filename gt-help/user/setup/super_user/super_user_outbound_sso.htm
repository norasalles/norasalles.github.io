Outbound SSO

Path: Setup > Super User > Outbound SSO

Outbound SSO allows customers who need access to the vendor web sites to able to set and configure the access to those service provider web sites, allowing their travelers to be redirected and logged in automatically.

The links are shown and accessible from the More Booking & Planning Tools section on the Home page.

  • Select the SSO vendor from the table that is to be configured.

To enable and configure most Outbound SSO settings

  1. To enable outbound SSO for the site, select the Enable Outbound SSO for the site check box. All vendors configured in your site appear.
  2. To enable and configure each vendor, do the following:
    1. Click the name of the service provider.
    2. To enable the vendor, select the Display this Outbound SSO link in the More Booking & Planning Tools section on the Home page check box.
    3. Select who will see the link on the Home page as follows:
      • All users.
      • Subsite Administrators and Meeting Planners only.
      • Subsite Administrators only.
      • Meeting Planners only.
    4. If the service provider needs a special ID to better identify the Customer to which the travelers belongs, enter that ID from the Company ID box.
    5. If users must send additional information when connecting to the service provider, you can select those items in the Custom Fields area. To add items to the Custom Fields area, go to Setup > Site Preferences > Custom Field Editor.
    6. Click Save.
  3. After you make all necessary changes to service providers, click Save.

To enable and configure S20 or SAML SSO partner settings

  1. To enable outbound SSO for the site, select the Enable Outbound SSO for the site check box. All vendors configured in your site appear.
  2. To enable and configure each vendor, do the following:
    1. Click the name of the service provider.
    2. To enable the vendor, select the Enable this Outbound SSO site in the More Booking & Planning Tools section, on the Home page check box.
    3. In the User Name and Password boxes, enter the login information.
    4. In the BMIS Number box, enter your customer ID.
    5. Click Save.
  3. Click Save.
