Super User Car Settings

Path: Car > Super User

You can use the Car Super User page to control the source of third-party car content.

To configure car settings

  1. To enable third-party car content, select the Enable search for taxis, limousines, and/or executive sedans check box.
  2. If you have the new Home page enabled and want to disable car service in the More Tools tab on the new Home page, clear the Enable search for taxis, limousines, and/or executive sedans on the new Home page check box.
  3. In the Vendor list, click the vendor you want.
  4. Enter the Provider's URL. You can use up to 128 characters.

  5. Optional: Enter a custom label that appears on the Home page. Use up to 32 alphanumeric characters. The default label is:

    Taxis, Limos, & Executive Sedans

  6. Indicates whether to Enable search for Mine Spec vehicles. Several car vendors offer a specific vehicle type for “Mine Equipped” vehicles (Budget, Avis, Hertz, Thrifty), which is mainly requested for specific regions in Australia. These are trucks that offer special features and are available using specific SIPP codes at specific locations.
  7. Click Save.
