Super User Policy

Path: Policy> Super User

You can use the Policy Super User page to do the following:

The Agency Vendor Preferences section allows you to configure settings that affect how preferred airlines (as defined in the Vendor Preferences section) are displayed in the search results.

You can use the Agency Vendor Preferences to bias air availability displays to promote agency preferred airlines, demote non-preferred ones, and exclude specific airlines. This functionality works in conjunction with Market Based Preferences (Corporate level) and traveler preferred carriers. Corporate preferences in Market Based Preferences and traveler preferences take precedence over agency preferences. When there is a conflict, corporate preferences override agency preferences. Airlines cannot be treated as both, “promoted” and “demoted”. The “promoted” setting takes precedence over the “demoted” setting.

To enable vendor preferences

  1. To allow site administrators to set up and configure preferred vendors for air travel, select the Enable Agency Vendor Preferences for Air check box.
  2. To allow site administrators to set up and configure preferred vendors for hotel bookings, select the Enable Agency Vendor Preferences for Hotel check box.
  3. To show the link to Policy Administration at the top of the traveler application and on the Home page for Site Administration, select the Enable Policy Administration Link check box.
  4. To enable the Policy Decision Engine (PDE) based policy evaluation, select Enable PDE based policy evaluation.
  5. To enable policy to evaluate the ideal time window, select the Use the traveler's selected flight option times for ideal time window evaluation when shopping by time check box. If you select this check box, the ideal time window is evaluated as follows:
    • If the traveler uses the Shop by Price path, the system will base the ideal time window on the search criteria entered on the Home page.
    • If the traveler uses the Shop by Schedule path, the system will base the ideal time window on the traveler's selected flight times on the Select Flight pages.
  6. To allow the evaluation of the number of passengers on a flight, as configured in the Travel Policy rule on each subsite, select the Enable Maximum Number of Travelers check box. You must setup this option at the Main site level in order to determine which sites are to be evaluated for the passenger flight count.
  7. Click Save.
