Super User Rail Settings - Amtrak

Path: Rail > Super User > Amtrak

Amtrak is only offered through Sabre.

To configure Amtrak Settings

  1. Select whether to Enable Amtrak.
  2. In the Data Source list, click Sabre.

  3. In the TA Group box, type the code for the TA group and the PCC to be used for making Amtrak bookings.

    The format is: TA Group, 4 slashes, and the pseudo city code. For example:

    <test>////<PCC>

  4. If you want site administrators to be able to turn Amtrak on and off, select the Allow Site Administrators to enable Amtrak check box.
  5. Click Save.
