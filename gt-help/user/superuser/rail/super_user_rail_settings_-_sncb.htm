Super User Rail Settings - SNCB

Path: Rail > Super User > SNCB

GetThere offers its customers the ability to make rail bookings on Société Nationale des Chemins de Fer Belges (SNCB). BeNe User Name, Password, and Distributor are provided by the TMC supporting the corporate customer. Agencies have access to an SNCB point of sale application (linked to the credentials configured in GetThere) that is used for aftersales transactions, such as changes, cancellation, reporting, and invoicing.

To configure SNCB settings

  1. Select whether to Enable SNCB.
  2. Select whether to Enable Low Fare Search for SNCB.
  3. If you want site administrators to be able to turn SNCB on and off, select the Allow Site Administrators to enable SNCB check box.
  4. Select whether to Enable Passive Segments for SNCB.

    Note  This option is only available to sites using GTX2 Booking and whose GDS is Sabre or Amadeus and when booking only 1 traveler. When enabled, a PNR is created in the GDS with passive segments for the SNCB segments.

  5. In the BeNe User Name box, type the BeNe user name provided the TMC supporting the coporate customer.
  6. In the BeNe Password box, type the BeNe password provided the TMC supporting the coporate customer.
  7. In the BeNe Distributor box, type the BeNe user name provided the TMC supporting the coporate customer.
  8. Click Save.
