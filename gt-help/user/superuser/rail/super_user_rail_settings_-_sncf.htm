Super User Rail Settings - SNCF

Path: Rail > Super User > SNCF

All SNCF bookings made through GetThere are made in the GetThere SNCF pseudo city code (PCC).  In order for an agency to be able to access the record, they must be queued to the Agency SNCF PCC.  The Branch Access Editor allows the GetThere Super User to open or close branch access between the GetThere SNCF PCC and the Agency SNCF PCC.  Branch Access must be opened to the PCC listed in the Pseudo City for SNCF PNRs field.

To configure SNCF settings

  1. Select whether to Enable SNCF.
  2. Select whether to Enable Low Fare Search for SNCF.
  3. If you want site administrators to be able to turn SNCF on and off, select the Allow Site Administrators to enable SNCF check box.
  4. Select whether to Enable Passive Segments for SNCF.

    Note  This option is only available to sites using GTX2 Booking and whose GDS is Sabre or Amadeus.  When enabled, a PNR is created in the GDS with passive segments for the SNCF segments.

  5. In the GetThere SNCF PCC (to open branch access to the Agency SNCF PCC) box, type a maximum of 4 alphanumeric characters, and click Open.  You must open branch access from the GetThere SNCF PCC to the Agency SNCF PCC so that the agency can access the bookings.
  6. To disable branch access, in the GetThere SNCF PCC (to close branch access to the Agency SNCF PCC) box, type a maximum of 4 alpha/numeric characters, and click Close.  
  7. To display a list of all SNCF PCCs where branch access has been opened between the GetThere SNCF PCC and the Agency SNCF PCC, click the List of PCCs that have SNCF branch access link.
  8. Click Save.
