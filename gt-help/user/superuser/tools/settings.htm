GDS Profile Synchronization Settings

Path: Tools > Profile Synchronization > GDS Profile Synchronization Settings

GDS Profile Synchronization allows updates from the Site to the GDS profiles with the Host Travel Agency through user action, User Database Upload Tool, or Mass Synchronization.

Note  GDS Profile Synchronization is not applicable to Amadeus.

Supersite administrators can view the settings, but cannot change them.

To set up GDS Profile Synchronization Settings

  1. Select whether to Perform mass GDS Profile Synchronization. This setting controls synchronization of the entire user profile database. The system performs this process only one time after it is enabled. After the initial synchronization is complete, you cannot repeat the process unless the toggle is disabled and re-enabled.
  2. Select whether to Perform continuous GDS Profile Synchronization. This setting control whether the system synchronizes data whenever user information is modified through the user profile.
  3. Click Save.

About the GDS Profile Synchronization Log

The GDS Profile Synchronization Log lists the status of the mass GDS Profile Synchronization request items as follows:

  • Synchronization start time
  • Synchronization end time
  • Number of profiles on queue for synchronization
  • Number of profiles processed
  • Total number of profiles for the site
  • Status
