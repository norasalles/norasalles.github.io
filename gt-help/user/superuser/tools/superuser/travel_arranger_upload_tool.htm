Travel Arranger Upload Tool

Path: Tools > Super User > Travel Arranger Upload Tool

The Travel Arranger Upload Tool provides customers and resellers the ability to upload their own travel arranger associations instead of Services doing it for them. It also provides the option of deleting associations, which has to be done by DBE in today’s environment. You can upload up to 50,000 associations per file.

Note  Only GetThere personnel can enable the Travel Arranger Upload Tool. To enable this feature, contact your Service Representative.

To enable the Travel Arranger Upload Tool

  1. Select whether to Enable Travel Arranger Upload Tool.
  2. Click Save.
