User Data Extract

Path: Tools > Super User > User Data Extract

The GetThere User Database Extract Tool provides customers and agencies selfservice, administrative access to user data – reducing turnaround time and streamlining user data maintenance.

Note  Only GetThere personnel can enable the User Data Extract Tool. To enable this feature, contact your Service Representative.

To enable user data extract requests

  1. Select whether to Enable User Data Extract requests.
  2. Click Save.
