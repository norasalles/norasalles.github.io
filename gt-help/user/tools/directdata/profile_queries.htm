Profile Queries & Imports

Path: Tools > DirectData > Booking Queries & Imports

DirectData (DD) has two functions. The first function provides profile synchronization from a customer data source into the booking tool for new user adds, updates, and deletions. Profile synchronization also happens when a profile is modified online within the booking tool. An event is added to the EAP that you can query. The second function that DD provides is booking information export. Both online bookings as well as agency-initiated bookings (with PNR acquire enabled) are available for you to query.

Note  You must be an Agency Superuser or above to change DirectData settings. To enable this feature, contact your Service Representative.

To configure profile query and import settings

  1. In the Username and Password boxes, type a username and password for use on all profile query requests and imports to the data manager program.
  2. Select whether to Enable profile data booking/exporting for the main site.
    1. If you select this check box, select whether to Disable profile data booking/exporting for all subsites, Enable profile data booking/exporting for all subsites, or Enable profile data booking/exporting for some subsites.
    2. If you Enable booking data exporting for some subsites, in the Subsites list, select the sites for which you want to enable booking data exporting. To select multiple items, hold down the Ctrl key while you click other items that you want to select.
  3. Click Save.
