GDS Profile Synchronization Editor

Path: Tools > Profile Synchronization > GDS Profile Synchronization Editor

GDS Profile Synchronization allows updates from the Site to the GDS profiles with the Host Travel Agency through user action, User Database Upload Tool, or Mass Synchronization.

Use the GDS Profile Editor to configure the fields and formats needed to synchronize to the host GDS.

The system makes the synchronization commands when changes are saved in an online profile or when profile updates are received from the User Database Upload Tool.

Synchronization can only be successfully accomplished when all profiles are uniquely named and are standardized in line number and format.

Notes   

  • GDS Profile Synchronization is not applicable to Amadeus.
  • You may program a maximum of 40 strings.

To use the GDS Profile Editor

  1. In the Profile Name, type the string used to create the profile.
  2. In the Line Layout box, type how many of each profile line type the GDS profile will have. This only applies to Sabre and Abacus.
  3. In the Profile String box, type the string based on the format and data required using QuarterMaster language.
  4. In the Field Type list, click the field Type that will be entered into the GDS profile as follows:
