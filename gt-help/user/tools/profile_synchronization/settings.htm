GDS Profile Synchronization Settings

Path: Tools > Profile Synchronization > GDS Profile Synchronization Settings

You can use the GDS Profile Synchronization Settings to configure updates from the Site to the GDS profiles with the Host Travel Agency through user action, User Database Upload Tool, or Mass Synchronization.

Note  GDS Profile Synchronization is not applicable to Amadeus.

If you are a supersite administrator you can view the following options:

  • Perform mass GDS Profile Synchronization – This setting controls synchronization of the entire user profile database. The system performs this process only one time after it is enabled. After the initial synchronization is complete, you cannot repeat the process unless the toggle is disabled and re-enabled.  
  • Perform continuous GDS Profile Synchronization – This setting control whether the system synchronizes data whenever a user information is modified through the user profile.

To change these settings, contact your Service Representative.

About the GDS Profile Synchronization Log

The GDS Profile Synchronization Log lists the status of the mass GDS Profile Synchronization request items as follows:

  • Synchronization start time
  • Synchronization end time
  • Number of profiles on queue for synchronization
  • Number of profiles processed
  • Total number of profiles for the site
  • Status
