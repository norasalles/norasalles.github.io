Travel Arranger Upload Tool

Path: Tools > Travel Arranger Upload Tool

The Travel Arranger Upload Tool provides customers and resellers the ability to upload their own travel arranger associations instead of Services doing it for them. It also provides the option of deleting associations, which has to be done by DBE in todayâ€™s environment. You can upload up to 50,000 associations per file.

Note  Only GetThere personnel can enable the Travel Arranger Upload Tool. To enable this feature, contact your Service Representative.

To add or remove Travel Arranger associations

  1. To download and open a template csv file, click the Travel Arranger Upload Template link.
  2. In the template, click the Template sheet. You can also click the Key sheet to view information and examples for entering information into the Template sheet.
  3. For each association you want to add or remove, do the following:
    1. In the Arranger ID column, type the travel arranger's user ID that they use when logging in to the booking tool.
    2. In the Traveler ID column, type the user ID for the traveler for which you want to add or remove an association with the specified travel arranger.
  4. Save the file as follows:

    If you are

    Then

    Adding associations between travel arrangers and travelers

    Save the file as a CSV file and with a file name that starts with "ADD_" (for example, "ADD_20120301.csv").

    Deleting associations between travel arrangers and travelers

    Save the file as a CSV file and with a file name that starts with "DEL_" (for example, "DEL_20120301.csv").

  5. In the Travel Arranger Upload tab, do one of the following:

    If you are

    Then

    Adding associations between travel arrangers and travelers

    Click Add New.

    Deleting associations between travel arrangers and travelers

    Click Delete Existing.

  6. To select the CSV file, click Browse, and then select the file with the associations you want to add or delete.

  7. Click Upload.

    The system sends a confirmation e-mail to the e-mail address in the online profile of the user requesting the change.
