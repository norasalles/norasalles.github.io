Change/Cancel

Path: Trip Manager > Change/Cancel

You can use the Change/Cancel page to control the user’s ability to change or cancel a selected itinerary. Settings on this page can only be set by a GT Superuser.

Note  GT Ticket Manager is currently available on Sabre and Apollo.

For more information, click one of the following:

To configure user's ability to cancel a selected itinerary

  1. In the Permit users to cancel air bookings list, click one of the following:
    • No – Traveler is unable to cancel a booking
    • Yes, unticketed bookings – Traveler can cancel reservations that have not been ticketed.

    • Yes, refundable bookings – Traveler is able to only cancel refundable bookings.

      Note  If the ticket is refundable to a voucher only, a user cannot cancel online.

    • Yes, all bookings – Traveler is able to cancel any booking including instant ticketing carriers and WN.

      Note  If the ticket is refundable to a voucher only, the user can cancel online, but does not receive any refund calculation and will not be on the GTM Refund path.

  2. Select whether to Permit users to cancel car bookings. If you select Yes, then do the following:
    1. Select whether users can cancel car bookings containing passive segments.
    2. Select whether users can cancel car bookings that include ticketed air reservations.
    3. Select whether users can cancel car bookings that include restricted air reservations.
  3. Select whether to Permit users to cancel hotel bookings. If you select Yes, then do the following:
    1. Select whether users can cancel hotel bookings containing passive segments.
    2. Select whether users can cancel hotel bookings that include ticketed air reservations.
    3. Select whether users can cancel hotel bookings that include restricted air reservations.
  4. Select whether to Permit users to cancel rail bookings.
  5. Click Save.

To configure user's ability to change a selected itinerary

  1. In the Permit users to change air bookings list, click one of the following:

    Note  This only applies if Refund, Void, and Exchange or Exchange only is set.

    • No – Traveler is not able to change the booking.
    • Yes, unticketed bookings – Traveler can change only unticketed bookings.
    • Yes, exchangeable bookings – If the system is able to determine if this is an exchangeable booking, the traveler can change the booking and the exchange is processed.  If the system is unable to determine whether the booking is exchangeable, changes are not made but the PNR is documented.
    • Yes, any bookings – Traveler can change any booking, and the PNR is either documented and/or exchanged.

    If you chose any of the three options with Yes, do the following:

    1. Select whether users can change their air bookings by segment.
    2. Select what types of changes are allowed as follows:
      • Permit any changes – Traveler can make any change to the reservation.
      • Limit to date and time changes only – Traveler cannot change the city pairs or airline, they can only change the date and/or time.
  2. Select whether to Permit users to change car bookings. If you select Yes, then do the following:
    1. Select whether users can change car bookings containing passive segments.
    2. Select whether users can change car bookings that include ticketed air reservations.
    3. Select whether users can change car bookings that include restricted air reservations.
  3. In the Permit users to change hotel bookings list, click one of the following:
    • No
    • Yes, permit any changes – Traveler can make any change to the reservation.
    • Yes, but limit to date and time only – Traveler cannot change the city or hotel. They can only change the date.

    If you chose any of the three options with Yes, do the following:

    1. Select whether users can change hotel bookings containing passive segments.
    2. Select whether users can change hotel bookings that include ticketed air reservations.
    3. Select whether users can change hotel bookings that include restricted air reservations.
  4. Click Save.
