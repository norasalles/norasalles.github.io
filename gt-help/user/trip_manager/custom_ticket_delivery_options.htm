Custom Ticket Delivery Options

Path: Trip Manager > Custom Ticket Delivery Options

Custom Ticket Delivery Options are displayed to users on the Trip review and checkout page if their itinerary matches the criteria specified in one of the Delivery Options. Once they select one of these options from the drop-down menu, the system can apply specified ticketing instructions in the Delivery Options.

You can use the Custom Ticket Delivery Options page to offer up to 10 specific delivery options, such as ETICKET or OFFICE PICKUP, to your users based on their itinerary characteristics.

Take all possible itinerary options into consideration when setting up Custom Ticket Delivery options. If a condition exists on the itinerary that is not accounted for in the Custom Ticket Delivery Options, then no delivery option is displayed. In these situations, users may not be able to complete some reservations because the GDS does not receive a delivery option.

For more information, click one of the following:

Create Custom Ticket Delivery Options

  1. In the Delivery Option Message box, type the delivery option label. This option appears in the drop-down menu on the Trip review and checkout page. Make sure each Delivery Option Message is unique so you do not confuse users.
  2. Select the delivery option Departure time frame. These selections determine the amount of time before flight time for the Delivery Option to be applied. The first field refers to the minimum amount of time before departure and the second field refers to the maximum amount of time. For example, you might specify that e-ticketing is the default when there is at least 4 hours before a flight and no more than 30 days before a flight.
  3. In the Itinerary Option list, click the type of itinerary to which this delivery option applies as follows:
    • All applies to all itinerary types.
    • Domestic refers to flights originating and ending in the home country for this site (e.g. for U.S home country NYC to SFO).
    • Domestic Departure refers to flights that originate in the home country (e.g. NYC to LON).
    • International refers to flights that originate and end outside of the home country (LON to TOK).
    • International Departure refers to flights that originate outside of the home country (LON to NYC).
    • Non Air refers to itineraries that do not include a flight (e.g. hotel reservations or car rentals).
    • Supplier Website refers to itineraries booked via Air Connect.
  4. In the Carrier Option list, click the selection criteria for specific airlines as follows:
    • Use Any Carriers to include flights from any carrier.
    • Use All Legs Use One of the Carriers Listed to include itineraries where flight legs use a listed carrier from the Carrier List, but not necessarily the same carrier.
    • Use All Legs Use a Single Carrier Listed to include itineraries where all flight legs use just one of the carriers listed.
    • Use Some Legs Use Carriers Which are not Listed to include itineraries where at least one of the legs uses a non-listed carrier.
    • Use No Legs Use Carriers Which are Listed to include itineraries where none of the flight legs use a listed carrier.

      Note  Use this list in conjunction with the Carrier List. If you enter UA in the carrier list and select All Legs Use a Single Carrier Listed, then you could apply this delivery option to round trip tickets on United Airlines.

  5. Select the E-Ticket Indicator. The airline returns an E-ticket indicator if users can purchase a flight as an E-Ticket. However, some itineraries do not qualify for E-Ticketing (international flights, for example).

    If you set this field to All Segments Have E-Ticket Indicator, then the system uses this setting to identify itineraries where the GDS indicates that a segment is authorized for E-Ticketing.

  6. If you want to restrict the use of delivery options to specific airlines, leave the E-Ticket Indicator field blank and enter the Carrier List. Separate multiple airline codes with a comma.
  7. If you want to set up a delivery option based on your office schedule, select the Office Hours check box. If you select this check box, make sure you configure office hours (Agency > Agency Details | Office Hours Configuration).
  8. In the Miscellaneous conditionsbox, type QuarterMaster codes. If you need assistance with Quartermaster commands, contact your Service representative.
  9. Select whether to Limit to this delivery Option. Some bookings may meet the conditions of more than one delivery option. The system makes comparisons sequentially, beginning with your first option. Use this field to prevent the system from comparing options that follow this one if the conditions match. Therefore, if you enable this field and this Delivery Option is applicable and there are no other Delivery Options with a lower number, then only this option is displayed to users on the Trip review and checkout page.
  10. Click Save.

Configure Amtrak Ticketing Options

  1. If you want to display the option for a travel agency to issue the ticket to the traveler in the drop-down menu on the Trip review and checkout page, do the following:

    1. Select the Please have the travel agency issue my ticket check box.
    2. Type the text that is displayed to the user to indicate this option (the label).
    3. Select the delivery option Departure time frame. These selections determine the amount of time before departure time for the Delivery Option to be applied. The first list refers to the minimum amount of time before departure and the second list refers to the maximum amount of time. If you set this field to No limit, this ticketing option is available to travelers at all times, regardless of the traveler's departure date and time.
    4. If you want to set up a delivery option based on your office schedule, select the Office Hours check box. If you select this check box, configure office hours (Agency > Agency Details | Office Hours Configuration).
  2. If you want to display the option for the traveler to pick up their ticket at the station in the drop-down menu on the Trip review and checkout page, do the following:
    1. Select the I will pick up the ticket at the Amtrak station check box.
    2. Type the text that will be displayed to the user to indicate this option (the label).
    3. Select the delivery option Departure time frame. These selections determine the amount of time before departure time for the Delivery Option to be applied. The first field refers to the minimum amount of time before departure and the second field refers to the maximum amount of time. If you set this field to No limit, this ticketing option is available to travelers at all times, regardless of the traveler's departure date and time.
    4. If you want to set up a delivery option based on your office schedule, select the Office Hours check box. If you select this check box, configure office hours (Agency > Agency Details | Office Hours Configuration).
  3. Click Save.

Configure SNCF Ticketing Options

Note  At this time, SNCF tickets are issued by travel agencies.

  1. If you want to display the option for a travel agency to issue the ticket to the traveler in the drop-down menu on the Trip review and checkout page, do the following:
    1. Select the Agree to have the travel agency issue my ticket check box.
    2. Type the text that is displayed to the user to indicate this option (the label).
    3. Select the delivery option Departure time frame. These selections determine the amount of time before departure time for the Delivery Option to be applied. The first list refers to the minimum amount of time before departure and the second list refers to the maximum amount of time. If you set this field to No limit, this ticketing option is available to travelers at all times, regardless of the traveler's departure date and time.
    4. If you want to set up a delivery option based on your office schedule, select the Office Hours check box. If you select this check box, configure office hours (Agency > Agency Details | Office Hours Configuration).
  2. If you want to display the option for the traveler to pick up their ticket at the agency office in the drop-down menu on the Trip review and checkout page, do the following:
  3. Select the I will pick up the ticket from my agency's office check box.
    1. Type the text that is displayed to the user to indicate this option (the label).
    2. Select the delivery option Departure time frame. These selections determine the amount of time before departure time for the Delivery Option to be applied. The first list refers to the minimum amount of time before departure and the second list refers to the maximum amount of time. If you set this field to No limit, this ticketing option is available to travelers at all times, regardless of the traveler's departure date and time.
    3. If you want to set up a delivery option based on your office schedule, select the Office Hours check box. If you select this check box, configure office hours (Agency > Agency Details | Office Hours Configuration).
  4. Click Save.
