Error Queues

Path: Trip Manager > Queues > QueueMaster

An Error Queue is available for use when there is a communication error with the virtual payment vendor. For more information about virtual cards, see Payment Options.

To configure an error queue

  1. In the Pseudo City Code box, type the pseudo city code (PCC).

  2. In the Queue Number box, type the queue number in which the PNR is to be placed.

     

    Depending on GDS, a prefatory code may be required.

    It is recommended that a prefatory code be included as follows:

     

    The following format adds a custom prefatory code, numbered 200 in Sabre:

    QI/200/A-MANUALLY UPDATE DEPLOYMENT AND RESEND FAX«

     

    The following format rewrites prefatory code 200 with this new text:

    QI/200/C-MANUALLY UPDATE DEPLOYMENT AND RESEND FAX«

     

    Agencies can choose any prefatory code number above 200 and use that in the error queue to send the failed payment transactions.

     

  3. Click Save.
