Modify PNR Editor

Path: Trip Manager > PNR Editor > Modify PNR Editor

Modify PNR Editor allows you to control the information inserted into existing reservation records (PNR's) when they are changed using the new architecture programming language.

Note  You may create a total of 50 PNR format strings.

To enter strings

  1. In the PNR Format String box, type a string based on the GDS format and field data.
  2. In the PNR Field list, click a field type to be used when entering a string.
  3. In the GDS list, click the GDS that receives the programmed string.

    Note  This is only available if the site has SuperPNR enabled.

To exclude fields

  1. In the PNR list, click a PNR field if the system default entry is not to be used.

    Note  A PNR Format String should be created in place of the excluded system default entry.

  2. In the GDS list, click the GDS for which the excluded field is not to be used.

    Note  This is only available if the site has SuperPNR enabled.

  3. Click Save.
