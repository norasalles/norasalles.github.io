Ticket Manager – Messaging

Path: Trip Manager > Ticket Manager > Messaging

By entering custom text, you can display a message on any of the following pages:

  • Home page
  • Flight Availability pages
  • Low Fare Options page
  • Checkout pages
  • Cancellation Details page

Use the text editor to enter or edit the message you want displayed on these pages. There are no default messages.
