Travel Arranger Settings

Path: User > Travel Arranger Settings

You can use the Travel Arranger Settings page to configure the following:

  • Preferences
  • Messaging

A Travel Arranger is any registered site user who is assigned by another registered site user to make travel arrangements on their behalf. Travelers can select their arrangers, or travelers can designate themselves as another user’s Travel Arranger.

For more information, click one of the following:

Preferences

To configure Travel Arranger preferences

  1. To turn on Travel Arranger functionality, select the Allow travelers to assign travel arrangers check box.

    Notes   

    • This setting adds the Profile link in the main navigation; it overrides the setting (under User > Profile Pages > General Settings) that turns off the Profile link.
    • When enabled, an Arranger Settings section is added to the profile. When the user clicks My Arrangers, they can search for and add travel arrangers. They can also remove travel arrangers.

  2. If you enabled Travel Arranger functionality, select whether to Allow travelers to designate themselves as travel arrangers for others.

    Notes   

    • If you select this check box, a traveler can search for and designate themselves as an arranger for another traveler. A button is added to the My Travelers / People Who Allow Me to Arrange Their Travel section. The user can click Add a Traveler to search for and add a traveler. When a traveler self-designates, the person who they have chosen as one of their travelers automatically receives an e-mail notifying them of the self-designation.
    • You can only add travelers who have selected Yes in the Can others designate themselves as my travel arranger? list in the My Arrangers page of the Profile.
  3. If you enabled Travel Arranger functionality, select whether you want to Allow travel arrangers to edit their travelers' profiles.

    Note  When you select this check box, a travel arranger can edit their travelers' profile. The travel arranger can only edit profiles for those travelers who have granted permission. The traveler grants permission in the My Arrangers page by making a selecting in the Who can change my account settings list.

  4. Select whether you want to Substitute the arranger's profile fields for the traveler's profile fields.

    If you select this check box, select the fields you want substituted. For example, use the arranger's e-mail address for notifications instead of the traveler's.

  5. Select whether to allow travel arrangers to duplicate a trip for another traveler. If you select this check box, travel arrangers are allowed to duplicate a reservation for another traveler that is assigned to them. Once a reservation has been completed, the travel arranger is able to select another traveler and duplicate that same reservation for that other traveler.

  6. In the Global Travel Arranger Settings list, select sites for which travel arrangers can arrange travel. This is in addition to the site currently used by the travel arranger.

    Warning  Make sure that all customers and/or agencies that have access to the SuperSite are allowed to have access to all subsites. Once enabled, any agency superuser access at the main site level can allow arrangers to cross over to any subsite enabled. If there is a shared supersite with multiple customers and/or agencies, arrangers could possibly gain access to another customer’s/agency’s site.

    Note  Global Travel Arranger functionality cannot be used together with subsite crossing.

  7. Click Save.

Messaging

You can display a custom message on the travel arranger setting pages in the Profile:

Type

Appears

Travel Arranger Setting

On the My Travelers and My Arrangers profile pages where you create travel arranger settings

Note  The types of messages available on your system depends on the features you have enabled. By default these message areas are blank. There are no default messages. You can customize the messages for each language you have enabled.

Enter messages

  1. If applicable, click on the tab for appropriate language.
  2. Use the text editor to enter or edit your message.
  3. Click Save at the bottom of the text editor to save your messages in Site Administration. (You do not have to click Save each time you enter a message in a different language.)
