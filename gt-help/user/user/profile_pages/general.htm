General Settings

Path: User > Profile Pages > General Settings

The booking tool Profile allows users to enter personal information and configure system preferences. The system uses this information throughout the booking process.

By configuring the Profile pages, you can control whether users can access their Profiles and you can control the options included on the Profile pages. You can also add announcements to Profile pages. There are additional settings that affect the Profile pages in Setup > Site Preferences.

Note  Information that has been entered and not deleted, even if it is not displayed to the user, is used by the system.

To setup general Profile settings

  1. To allow users can access their profiles by clicking the Profile link in the main navigation bar of the booking tool, select the Display the Profile link in the main navigation check box.

    Notes   

    • The Profile link appears even if this option is not selected if you allow users to change their passwords, have access to Profile History, or set up Traveler Arrangers.
    • If you have the new Home page enabled, the Profile tab appears even if this option is not selected if you enable certain password, profile history, or  travel arranger settings or when travel arrangers are allowed to edit their travelers profile settings.
  2. Use the text editor to enter or edit your Profile Announcements.

    Type

    Appears

    Top Message

    On the top of all profile pages

    User Profile Message

    Below the top message on the main Profile page

    Bottom Message

    At the bottom of all Profile Pages

  3. If you want to display a message next to the Save button on the Profile pages, select the Display a profile synchronization message when profiles are saved check box.

    If you are displaying the profile synchronization message, do one of the following:

    • In the Custom Message box, type a custom alert message.

    • Use the following default message:

      "Please Note: After you click Save, the system stores your changes in your profile, but does not send them to the travel agency."

  4. Click Save.
