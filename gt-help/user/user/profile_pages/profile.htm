Profile Settings

Path: User > Profile Pages > Profile Settings

You can use the Profile Settings page to configure the following:

  • Personal Information
  • Password & Charge Cards
  • E-mail, Display, & History

For more information, click one of the following:

Personal Information

Path: User > Profile Pages > Profile Settings | Personal Information tab

To configure personal information settings

  1. In the Personal Information tab, select whether to Allow users to edit their names in their profiles.
  2. Optional: Use the text editor to enter or edit the message you want displayed on the Personal Information page at the top of the Name, Address and Phone Number section. There is no default message.
  3. Optional: Use the text editor to enter or edit the message you want displayed on the Personal Information page above the ticket delivery address information. There is no default message.
  4. Identify the optional personal information fields you want displayed on the Profile pages by clicking the link and customizing the label, if applicable.

    Note  GetThere DirectObserver does not use custom labels and the hierarchical reporting structure is not changed. The hierarchy is: Site, SubSite, Company, Division, Cost Center, and Department (personal information fields affected are highlighted in bold). Keep this hierarchy in mind when modifying field labels.

    The following fields are always displayed in the Profile pages. You can, however, change the labels.

    • Home Phone
    • Fax Phone

    You can choose to change the labels, display or hide, or make read-only the following fields:

    • Company
    • Corporate ID
    • Employee Number
    • Division
    • Department
    • Cost Center
    • Job Title
    • Project Code

  5. Select whether to Display the Emergency Contact section on the Personal Information page. This check box allows users to enter the name, phone number, and country of residence of a person to contact in case of an emergency.

    Optional: If you select this check box, you can use the text editor to enter or edit the message you want displayed above the emergency contact information. There is no default message.

  6. Select whether to Display the General Requests or Special Needs section on the Personal Information page. Users can enter up to 390 characters of additional information.

    If you select this check box, you can use the text editor to enter or edit the message you want displayed above the General Requests and Special Needs section. There is no default message.

  7. Click Save.

Password & Charge Cards

Path: User > Profile Pages > Profile Settings | Password & Charge Cards tab

  1. In the Password & Charge Cards tab, select whether to Display the Change Password section. When you enable this field, a Change Password link is displayed on the main Profile page.  When users click the link, they have the ability to change their passwords for the booking tool without assistance.
  2. Select whether to display the Charge cards link on the Profile tab on the Home page. If selected, users can go to the Charge cards page and manage the charge cards they can use for travel bookings.
  3. Select whether to Allow users to modify and remove the first charge card listed on the Charge cards page in their profile. Most companies allow users to add, edit, or remove all charge cards on the Charge cards page in their profile. However, if you load charge cards for your users and do not want them to edit or remove the first card, you can clear this check box.
  4. Optional: Use the text editor to enter or edit the message you want displayed at the top of the Charge Cards page. There is no default message.
  5. Identify the optional charge card fields you want displayed on the Profile pages by clicking the link and determining if the field is optional or required.
  6. Click Save.

E-mail, Display, & History

Path: User > Profile Pages > Profile Settings | E-mail, Display, & History tab

  1. In the E-mail, Display, & History tab, you can use the text editor to enter or edit the message you want displayed in the E-mail Settings page of the profile pages. There is no default message.

  2. Select whether to Display the Default Currency option so users can select their preferred default currency.  If the user’s selection is different from the site’s currency, both currencies are displayed.

    Optional: Use the text editor to enter or edit the message you want displayed in the Display Preferences page of the profile pages. There is no default message.

  3. Select whether to Display the Profile History section. When you select this check box, a Profile History link is displayed on the main Profile page.  When users click the link, they can view a list of the dates and times changes were made to their profile and the names and e-mail addresses of who made the changes.
  4. Click Save.
