Travel Preferences

Path: User > Profile Pages > Traveler Preferences

You can use the Travel Preferences page to configure the following:

  • Frequent Traveler Settings
  • Passport & Visa Settings
  • Flight Preferences
  • Hotel Preferences
  • Rental Car Preferences
  • Rail Preferences

Note  We recommend that after you configure the traveler preferences, do not make changes because users may become confused if they cannot find information they had previously entered into the system.

For more information, click one of the following:

Frequent Traveler Programs

Path: User > Profile Pages > Traveler Preferences | Frequent Traveler tab

You can enter custom text to display as a message at the top of the Frequent Traveler page.

To configure frequent traveler settings

  1. Optional: Use the text editor to enter or edit the message you want displayed at the top of the My Frequent Traveler Programs section of the Frequent Traveler page. There is no default message.
  2. Select the maximum number of travel reward memberships users can enter for each travel type (frequent flyer, rental car, hotel chain, and rail) in their profile (up to 12).
  3. Click Save.

Path: User > Profile Pages > Traveler Preferences | Passports & Visas tab

Flight Preferences and Special Instructions

Path: User > Profile Pages > Traveler Preferences | Flight tab

You can enter custom text to display as a message at the top of the Flight Preferences page. The flight special instructions text box allows users to enter special information, instructions, and/or requests for air reservations.

Configure Flight Preferences and Special Instructions

  1. Optional: Use the text editor to enter or edit the message you want displayed at the top of the Flight Preferences page. There is no default message.
  2. Select whether to Display the flight special instructions section.

    Optional: If you select this check box, use the text editor to enter or edit the message you want displayed above the Special Instructions box. There is no default message.

    Note  Coordinate the use of this field with your GetThere representative to ensure that this information is correctly passed to the PNR created in the GDS.

  3. Click Save.

Hotel Preferences and Special Instructions

Path: User > Profile Pages > Traveler Preferences | Hotel tab

You can enter custom text to display as a message at the top of the Hotel Preferences page. The Hotel Special Request section allows users to request specific amenities, such as a non-smoking room. The Special Instructions box allows users to enter special information, instructions, and/or requests for hotel reservations.

To configure hotel preferences and special instructions

  1. Optional: Use the text editor to enter or edit the message you want displayed at the top of the Hotel Preferences page. There is no default message.
  2. Select whether to Display the hotel special requests section. The hotel special requests is a pre-populated list of hotel request items, such as a non-smoking room and king size bed. You can display this list in the Profile, as well as the Hotel Search page.  If you display the list in the Profile, any selections the user makes are automatically prepopulated on the Hotel Search page.
  3. Select whether to Display the hotel special instructions section.

    Optional: If you select this check box, use the text editor to enter or edit the message you want displayed above the Special Instructions section. There is no default message.

    Note  Coordinate the use of this field with your GetThere representative to ensure that this information is correctly passed to the PNR created in the GDS.

  4. Click Save.

You can display this list in the Profile, as well as the Car Search page.  If you display the list in the Profile, any selections the user makes are automatically prepopulated on the Car Search page.
  3. Select whether to Display the rental car special instructions section.

    Optional: If you select this check box, use the text editor to enter or edit the message you want displayed above the Special Instructions box. There is no default message.

    Note  Coordinate the use of this field with your GetThere representative to ensure that this information is correctly passed to the PNR created in the GDS.
