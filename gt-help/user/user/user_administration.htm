User Administration

Path: User > User Administration

You can use the User Administration page to search for a user. Once displayed, you can delete a user, change a password, unlock a user, or change a user’s security level.

To search for a user

  1. To limit your search to users on a specific site, Select the user's subsite. If you want to search all subsites, leave this blank (not recommended).

  2. Enter one or more of the following search criteria:

    Note  The more search criteria you enter, the narrower your search results will be.

    Field Label

    Enter the

    Search for a user with the following user name

    User name for the user as defined in the user's profile.

    Search for a user with the following employee number

    Employee number for the user as defined in the user's profile.

    Search for a user with the following first name

    First name for the user as defined in the user's profile.

    Search for a user with the following last name

    Last name for the user as defined in the user's profile.
  3. Select the maximum number of matches to return for review.
  4. Select whether you Should search results only display exact matches?.

  5. Click Search.

    The search results appear. You can then delete a user, change a password, unlock a user, or change a user’s security level.
