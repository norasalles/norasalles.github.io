Change Password

Path: User > User Administration > (Search Results) > (Change Password)

You can use the Change Password page to change the password for an existing user. You must also change the password when restoring site access for a user.

To change the password for a user

  1. Do one of the following

    If you want to

    Do

    Change the password for a user still has site access

    		Search for users.

    The User Administration search results page appears

    Restore site access for a user

    		Search for users.

    The Restore User search results page appears.

  2. Click the Select option button for the user for which you want to change the password.

  3. Click Change Password.

    The Change Password page appears.

  4. In the New Password box, type the new password for the user.
  5. In the Retype New Password box, type the new password for the user again.

  6. Click Change.

    The message The system has saved your changes. appears.
