User Administration - Search Results

Path: User > User Administration > (Search Results)

After you search for users, any users that match the search criteria appear. Once displayed, you can delete a user, change a password, unlock a user, or change a user’s security level.

For more information, click one of the following:

Delete a user

  1. Search for users.

    The User Administration search results page appears

  2. Click the Select option button for the user you want to delete.

  3. Click Delete User.

    A confirmation window appears.

  4. Click OK.

    The selected user disappears from the list of users, and the message Your changes have been saved. appears.

Change the password for a user

  1. Search for users.

    The User Administration search results page appears

  2. Click the Select option button for the user for which you want to change the password.

  3. Click Change Password.

    The Change Password page appears.

  4. In the New Password box, type the new password for the user.
  5. In the Retype New Password box, type the new password for the user again.

  6. Click Change.

    The message The system has saved your changes. appears.

Unlock a user

  1. Search for users.

    The User Administration search results page appears

  2. Click the Select option button for the user you want to unlock.

  3. Click Unlock User.

    The message Your changes have been saved. appears.

Change a user's security level

  1. Search for users.

    The User Administration search results page appears

  2. Click the Select option button for the user for which you want to change the security level.

  3. Click Change User Security Level.

    The Change User Security Level page appears.

  4. In the New Security Level list, click the new security level for the user. For a description of the security levels, click here.
    • Traveler – Can make reservations on the booking tool.
    • Travel Arranger – Person that can make reservations for another person or themselves.
    • Travel Approver – Person that can make reservations for themselves and approve reservations for others.
    • Agency Travel Counselor – Does not have Site Administration access but can make reservations for a user.
    • Profile Administrator – Can add users, keep Traveler profiles up-to-date.  Can book reservations for other users and guests.
    • Customer Help Desk – Can add users, keep profiles up-to-date, change passwords, and restore user accounts if they get locked out.
    • SubSite Administrator – Has administrative access to a specific SubSite only.
    • SuperSite Administrator – Has access to the Main site and any SubSites under the Main.
    • Agency Help Desk – Can add users, keep profiles up-to-date, change passwords, and restore user accounts if they get locked out.
    • Agency SuperUser – Has access to the Main site and any SubSites under the Main.  Can also view the Agency Tab.

  5. Click Assign.

    The message The system has saved your changes. appears.
