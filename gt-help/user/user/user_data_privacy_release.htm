User Data Privacy Release

Path: User > User Data Privacy Release

Site Administrators can require users to accept a Data Privacy statement prior to making bookings on the site.

Enable User Data Privacy Releases

  1. Select whether to Enable the Data Privacy Release.
  2. In the Display release list, click a release message to enable.
  3. Click Save.

Create a User Data Privacy Release Message

  1. Click Add.
  2. In the Release Name box, type the name you want to use for the release. You can type up to 100 characters.
  3. In the Release Title box, type the title for the release. You can type up to 100 characters.
  4. In the Release Message box, type the message text for the release.
  5. In the Decline Release Title box, type the title for the message a user sees if they decline the message release. You can type up to 100 characters.
  6. In the Decline Release Message box, type the message text a user sees if they decline the message release.
  7. Click Save.
  8. To select the release message you just created, in the Display release list, click a release message to enable.

Edit a User Data Privacy Release Message

  1. In the table, click the link in the Name column for the release you want to modify.
  2. Change the release information as needed.
  3. Click Save.

Delete a User Data Privacy Release Message

  1. Select the check boxes for the releases you want to delete.

  2. Click Delete.

    A confirmation dialog box appears.

  3. Click OK.

Delete all User Data Privacy Release Messages

  1. Select the check box in the header row.

  2. Click Delete.

    A confirmation dialog box appears.

  3. Click OK.
