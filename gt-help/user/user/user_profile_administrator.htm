User Profile Administration

Path: User > Profile Administrator Access

You can configure your site to allow different users access to edit Profiles. You can grant this privilege to registered users by searching for their names on the User Administration page and changing their security level to Profile Administrator. Alternatively, you can add a new user with Profile Administrator privileges.

Use the controls on this page to hide or display fields to the Profile Administrator when they access and edit user profiles.

  1. Select the check boxes for the fields you want Profile Administrators to be able to complete and edit.
