Using the Help System

How Do I Use This Help System?

  • Click the Contents button and then use the displayed links to navigate through the Help system. The table of contents  contains the following icons:
    • The book is similar to a chapter in a book. The book indicates that there are one or more topics related to the book's subject. You click a closed book to open it and view the topic list, or you click an open book to close it and hide the topic list.
    • The page indicates a topic in the Help system. Click the page icon or the text next to the icon to display the topic.
  • Click the Index button to search for a word or phrase and view a list of related Help topics. To search the index, type part or all of a word or phrase in the entry box at the top of the index page. To view the topic associated with the index entry, click the index entry.
  • Click the Search button (or use the Search box) to search for all Help topics that contain a specific word or phrase. Enter text in the field and then select Go to view a list of topics containing the text you entered in the field. For more information, see Performing Full-Text Searches.
  • Click the Glossary button to view a list of terms used in site administration and their definitions. Click a term in the upper half of the window to view the definition in the lower half of the window.
  • Click the Print button to print the displayed Help topic. Make sure you click in the topic you want to print before you click the Print Topic button or you may print the table of contents instead of the topic.

The following conventions are used in the Help system.

Convention

What it Means and How it Works

bold text

Indicates the name of an item on the tools site, such as a link, button, or text box.

blue link

Indicates a link to a new topic. Clicking the blue text opens a new topic.

green link

Indicates a definition of the word highlighted in green. Clicking the green text opens a pop-up window that contains the definition of the word. To close the pop-up window, click in the topic or select the Close button in the pop-up window.

Indicates a note. The text following the icon is supplemental information that is not necessary to complete the task.

Indicates a tip. The text following the tip provides alternate ways of completing tasks or provides information that may help you use the database upload tools more successfully.

Indicates an alert. The text following the alert is essential to the task. You should pay special attention to alerts.
